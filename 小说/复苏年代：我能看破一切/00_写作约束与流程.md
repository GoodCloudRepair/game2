# 《复苏年代：我能看破一切》写作约束与流程

## 目标

防止长篇连载中后期战力通胀、官方形象翻车、金手指变成面板词条、主线空转。

## 一、硬约束

### 1. 主线优先级

1. 表层：活下去，走上武道路，攒资源突破
2. 中层：武道高考前列 / 顶尖武道大学，进入官方视野
3. 深层：成为特勤/国之战力，守护身边人与城市
4. 终极：探明灵气复苏根源，稳住全球超凡格局

无关四层的剧情不写或压缩。

### 2. 世界观来源统一

所有超常现象只能回到：

- 灵气复苏（公开10年，实际异常监测30年以上，秩序未崩）
- 妖兽畸变与秘境诞生
- 武道修炼体系（凡人→灵台…）
- 超凡管理局主导的公开维稳

不另开无关宇宙法则；金手指「万物洞察」禁止写成游戏词条UI。

### 3. 主角写法约束

陆衡强在：

- 冷静务实，先自保再护人
- 用洞察打信息差，不靠嘴炮装逼
- 恩怨分明，不圣母

禁止：长期装弱卖惨、公然对抗国家机构、灰产倒卖当主线。

### 4. 能力写法约束

万物洞察必须写清：触发 / 效果 / 代价 / 当前上限。  
禁止临时救场新开挂；禁止生物词条面板。

### 5. 人物写法约束

重要角色至少承担：推主线 / 对照 / 冲突 / 情感锚点 / 揭示局势之一。

### 6. 信息投放约束

每章最多扩展1个关键新信息（一境、一功法、一组织规则、一秘境规则）。

### 7. 章节功能约束

每章至少完成：推进剧情 / 关系 / 信息 / 伏笔 中的一项，最好两项。  
战斗/探索必须有可见收益（灵石、材料、功法、修为、身份）。

### 8. 章节字数约束

硬下限2300字；软目标2800–3200字；必要时可突破3200字。  
开篇第001章按四段结构，字数落在同一区间。

### 9. 文风与审核红线

- 平台：起点男频 · 都市高武 / 灵气复苏
- 官方、军警、特勤**永远正面**，不写军队无力、高层腐朽
- 禁止大规模血腥屠城；袭击以自保、击退为主
- 境外反派=散修/敌对超凡组织，不抹黑他国整体
- 不写非法猎妖倒卖灰产当主线
- 与《全球迷雾》区隔：不用「词条」「校正」话术

### 10. 节奏约束

- 每5章至少一个小型冲突
- 每20章一个阶段性高潮
- 每战有回报，禁止空打流水账

## 二、写作流程（六步）

```
看纲 → 看台账 → 填章节记录 → 写正文 → 勾连续性检查 → 回填台账
```

详见 skill：`novel-serial-writing`。
