# 严苛审稿门槛

## 总原则

- 默认以最严苛口径对待正文：宁可少写、重写，也不保留逻辑弱、人物假、收益虚的段落。
- 不达标优先重构章纲或重写段落，不用润色掩盖结构问题。
- 每章必须让读者明确看到：冲突从哪里来，主角做了什么，代价是什么，换到了什么，下一章为什么还要看。

## 一票否决

出现以下任意一项，本章不算通过：

- 主线空转：删掉本章后，资格、资源、关系、能力、任务权限没有变化。
- 冲突虚假：冲突只靠误会、路人嘴贱、作者强行安排，而不是来自人物欲望、规则压力或资源差距。
- 人物失真：角色说话不像本人，去掉角色名分不出谁在说话。
- 官方降智：为了抬主角，把老师、特勤、管理局写得无能、迟钝或甩锅。
- 能力偷懒：洞察变成万能答案，没写触发、判断链、代价和当前上限。
- 收益虚浮：只写“被认可、开始、机会”，但没有分数、记录、资格、资源、任务编号、贡献点等可见落点。
- 设定倾倒：一段台词或旁白连续解释规则、费用、地点、流程，像公告或合同。
- 重复推进：同类名单、同类测试、同类说明连续出现，但没有更高层级的新结果。
- 章末模板化：用“不是结果，是开始”“有门”“往里走”等抽象总结代替具体钩子。
- 平台错位：写成悬疑、规则怪谈、异常调查文，压过都市高武的训练、资源和任务线。

## 写前六问

写正文前必须答清：

1. 本章推进哪一个具体结果：分数、资格、资源、排名、训练效果、任务权限中的哪一项？
2. 本章核心冲突是否来自人物欲望、制度门槛或资源差距？
3. 主角这一章的主动行动是什么，不是被动等通知吗？
4. 主要配角本章各自想要什么，是否只是工具人？
5. 本章和前后三章是否重复，重复的话新增层级在哪里？
6. 章末钩子是否是具体动作或具体信息，而不是作者总结？

任意一问答不出，先改章纲，不写正文。

## 写后四层审稿

### 1. 读者层

- 前300字是否有明确压力或可追问题？
- 每800字是否至少有一次信息变化、冲突推进或结果反馈？
- 本章是否给了一个可复述的爽点或现实压力点？

### 2. 情节层

- 因果是否闭合：上一章钩子是否被承接，本章结果是否能自然推到下一章？
- 关键转折是否靠证据、流程、训练结果或人物选择，而不是巧合？
- 低层级事件是否服务卷目标，不额外开无关支线？

### 3. 人物层

- 陆衡是否冷静务实，先算成本、看风险、留证据？
- 顾绍南是否资源型竞争者，不写成低级反派？
- 周冕、沈岚、王成是否各有身份口吻，不互相替代？
- 陆秀兰是否先关心钱和安全，不被写成纯卖惨工具人？
- 陈屿是否嘴碎但有真情绪，不连续抢主线？

### 4. 语言层

- 台词是否拆短，一句只承担一个主要信息？
- 说明是否被动作、表格、追问、反应分摊？
- 是否有悬疑氛围词、模板章末、公告腔、解释腔残留？
- 是否保留都市日常颗粒：高三、通勤、饭钱、家长签字、流程回执？

## 评分门槛

| 分数 | 处理 |
| --- | --- |
| 9.0以上 | 可进入正文，但仍清理口吻和冗余 |
| 8.0-8.9 | 只允许在明确问题修完后入正文 |
| 7.0-7.9 | 不入正文，重修章纲或重写关键段 |
| 7.0以下 | 推倒重写，不做局部润色 |

## 修正文原则

- 结构问题优先改章纲，不在正文里硬补。
- 人物动机问题优先加选择和代价，不加旁白解释。
- 口吻问题优先重写台词，不只替换语气词。
- 节奏问题优先删无结果段落，不靠加字数填满。
- 收益问题优先落到可见凭证：记录、分数、资格卡、贡献点、补贴、训练数据。
