# 设定补充：修炼 · 经济 · 装备

## 修炼路径（简）

1. 强身功法（凡人）→ 气血境（淬体）
2. 通脉（打通经脉，灵气细流）
3. 凝灵（灵力外放）
4. 聚丹 / 灵台（卷后）

突破通脉起，明显依赖灵石与秘境药；气血期猎低阶材料可辅助。

## 功法品级（本书）

| 品级 | 说明 | 获取 |
| --- | --- | --- |
| 凡级 | 强身、入门桩 | 书店/培训班几百到几千 |
| 黄级 | 正统武道入门 | 培训班进阶、任务兑 |
| 玄级 | 市内顶尖苗子常用 | 悬赏、大学、拍卖 |
| 地级及以上 | 稀缺 | 官方、大赛、深层秘境 |

陆衡开局：凡级《铁骨桩》。

## 货币与资源

| 资源 | 用途 |
| --- | --- |
| 人民币 | 日常、凡级功法、培训费 |
| 低阶灵石 | 修炼、换材料、任务点 |
| 妖兽材料 | 卖店/兑功法/炼药 |
| 任务点/学分 | 官方与学校体系 |

## 副业（第10章后开）

- 靠洞察辨药材年份与品相，帮衬炼低阶疗伤药或倒腾差价（合法店铺/培训班渠道）
- 不写黑市主线

## 装备线（中期）

- 合金战衣、热武器附魔、开灵短时强化
- 开篇不堆；作为技能学习，不单开金手指
