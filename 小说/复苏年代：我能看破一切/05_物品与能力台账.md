# 物品与能力台账

## 核心规则

| 规则 | 说明 |
| --- | --- |
| 灵气复苏 | 公开10年，实际异常监测30年以上；城市可居住；野外与废弃区风险升高 |
| 武道公开化 | 高考含武道科；功法可合法购学；顶尖仍稀缺 |
| 管理局优先 | 超凡事件由军警+管理局处理；民间猎杀受资质约束 |
| 洞察非面板 | 只描写裂痕/前摇/药性年份/功法别扭处，禁止词条UI |
| 气血值口径 | 只作为凡人、准武者、气血境早期体测指标；通脉境后改看经脉贯通数、灵力控制、实战评级和任务权限 |
| 异常目击备案 | 学生在官方事故中有目击、提前预警或受伤记录，可进入排查、备案、复核和校内专项说明流程；备案不等于奖励或补助 |
| 感知/反应复核 | 官方备案学生可进行专项复核；结果能进入校内补测、旁听观察和减免候选材料，但不直接等于补助或录取 |

## 金手指：万物洞察

| 项 | 内容 |
| --- | --- |
| 触发 | 专注注视；危机时被动唤醒（第001章） |
| 效果 | 见生物薄弱处与发力前摇；见药材大致年份与药性偏向；见功法运转别扭处与自身瓶颈方向 |
| 不能 | 读心；看穿完全伪装的高境杀意细节；弹出数值血条/技能名 |
| 代价 | 短时间连续洞察超过约3次→头痛欲裂；高两境目标只见模糊裂痕；情绪失控时失准 |
| 明面说法 | 「感知类天赋」；不对任何人说能量化看破 |
| 当前阶段 | 第013章后进入可控萌芽：可在训练、设备、图片筛查中短时主动注视风险轨迹和发力错路；连续使用仍眼眶发酸、太阳穴发紧，身体执行速度不足 |
