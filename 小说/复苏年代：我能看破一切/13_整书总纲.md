# 《复苏年代：我能看破一切》整书总纲

## 一句话梗概

灵气复苏满三年，武道已进高考。澄江三中高三生陆衡夜遇畸变野犬濒死，觉醒「万物洞察」——看破弱点、功法缺陷与修炼瓶颈，从普通人一路杀进武道大学与超凡管理局，探寻复苏真相。

## 题材与定位

- 平台向：起点男频
- 类型：都市高武 · 灵气复苏（慢复苏，秩序完整）
- 主情绪：绝境求生 → 偷练翻盘 → 高考争锋 → 国之战力
- 金手指：万物洞察（非系统、非词条面板）
- 禁止：游戏UI词条、官方废物化、灰产主线、对抗国家机构

## 主角四层目标

1. 表层：活下去；买功法、报培训班、踏入气血境
2. 中层：武道测试崭露头角；高考/保送顶尖武道大学
3. 深层：特勤实习与国境危机中成为新锐强者，护住身边人
4. 终极：探明灵气复苏根源，稳住全球超凡格局

## 世界观底层（超常现象统一回这里）

- **灵气复苏**：全球灵气回潮刚满3年；城市秩序仍在，军警与超凡管理局主导维稳
- **妖兽与畸变**：动植物受灵气刺激畸变；一阶起可伤人；新闻有通报但多数人当猎奇
- **秘境**：野外、深山、废弃工地/医院等诞生小型秘境，产灵石、材料、灵药
- **武道体系**：凡人→气血→通脉→凝灵→聚丹→灵台…；武道高中/大学；高考文化+修为双考
- **认知分层**：多数人只听新闻；少数人练强身；顶尖武者/异能者小众
- **万物洞察**：主角私有；肉身直觉见裂痕/前摇/缺陷，禁止弹出词条框
