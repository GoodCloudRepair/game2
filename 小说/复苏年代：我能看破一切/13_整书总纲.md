# 《复苏年代：我能看破一切》整书总纲

## 一句话梗概

灵气复苏公开十年，实际异常监测已持续三十年以上，武道早已进入高考体系。澄江三中高三生陆衡在武道体测中卡在补助线外，又被卷入学校附近封控事故，觉醒「万物洞察」——看破危险轨迹、功法缺陷与修炼瓶颈，从普通人一路杀进武道大学与超凡管理局，最终参与破界危机，探寻复苏真相。

## 题材与定位

- 平台向：起点男频
- 类型：都市高武 · 灵气复苏（慢复苏，秩序完整）
- 主情绪：资源卡线 → 体测翻身 → 高考争锋 → 特勤新锐 → 破界稳界
- 金手指：万物洞察（非系统、非词条面板）
- 禁止：游戏UI词条、官方废物化、灰产主线、对抗国家机构

## 主角四层目标

1. 表层：活下去；拿补测/备案资格，走上武道路，踏入气血境
2. 中层：武道测试崭露头角；高考/保送顶尖武道大学
3. 深层：特勤实习与国境危机中成为新锐强者，护住身边人
4. 终极：探明灵气复苏根源，参与破界危机，稳住全球超凡格局

## 世界观底层（超常现象统一回这里）

- **灵气复苏**：公众正式知晓已10年；实际异常监测30年以上；城市秩序仍在，军警与超凡管理局主导维稳
- **妖兽与畸变**：动植物受灵气刺激畸变；一阶起可伤人；新闻有通报但多数人当猎奇
- **秘境**：野外、深山、废弃工地/医院等诞生小型秘境，产灵石、材料、灵药；长线本质与世界边界变薄有关，前期只以「封控区/界隙痕迹」埋线
- **武道体系**：凡人→气血→通脉→凝灵→聚丹→灵台…；武道高中/大学；高考文化+修为双考
- **认知分层**：多数人只听新闻；少数人练强身；顶尖武者/异能者小众
- **万物洞察**：主角私有；肉身直觉见裂痕/前摇/缺陷，禁止弹出词条框

## 背景时间线

| 时间 | 状态 |
| --- | --- |
| 三十多年前 | 边境、深山、废弃矿区和无人海域出现无法解释的异常事件，归入特殊灾害档案 |
| 二十年前 | 灵气浓度变化被确认，科研系统、军方和特殊部门开始内部武道研究 |
| 十年前 | 灵气复苏正式公开，超凡管理局挂牌，公众开始接受妖兽、秘境、武道 |
| 六年前 | 武道教育进入重点城市试点，高校建立武道院系 |
| 三年前 | 武道高考全国铺开，普通高中开始常态化气血测试、补助线和备案培训 |
| 开书当年 | 澄江这类普通地级市制度已成型，资源差距开始明显拉开普通学生与武道优等生 |
