# 武考制度与资源经济

## 武道高考结构

| 项 | 规则 |
| --- | --- |
| 总结构 | 文化课60% + 武道科40% |
| 武道科 | 气血值、体能、基础技、实战模拟、感知/反应测试 |
| 普通本科线 | 文化课为主，武道只加分 |
| 武道大学线 | 文化课过线 + 武道科达标 |
| 顶尖武道大学 | 文化课不拖后腿，武道科排名进省前列 |
| 补助评定 | 气血值、进步率、家庭收入、校内推荐综合评定 |
| 补测资格 | 官方备案异常事件、校内推荐、训练进步异常，可触发一次 |

## 气血值口径（低境界体测）

| 气血值 | 含义 |
| --- | --- |
| 7以下 | 身体偏弱，不建议走武道 |
| 7-9 | 普通学生 |
| 10-11 | 普通学生中等偏上 |
| 12 | 校级补助观察线 |
| 13-15 | 武道班优等生 |
| 16-18 | 市级重点苗子 |
| 20 | 气血境入门门槛 |

> 气血值只作为凡人、准武者、气血境早期的体测指标。通脉境以后不再用单纯气血值衡量实力，改看经脉贯通数、灵力控制、实战评级和任务权限，避免后期数值爆炸。

## 正规任务与灰产对比

| 路径 | 收益 | 风险/边界 |
| --- | --- | --- |
| 灰产采样 | 常以日结八百等快钱诱导学生 | 无备案、无保险、无路线、风险自负；无证靠近管控区或私采材料违法 |
| 正规外围辅助 | 300-800/次，另有贡献点，可辅助培训减免和初级证材料 | 需学校推荐、家长知情、管理局/培训中心审核、带队人、保险、回执；学生只做辅助 |

## 开篇收益原则

- 第001章不给现金大奖，先给资格：官方备案、学校补测、周冕关注。
- 第002-003章再兑现基础班减免、旁听资格或备案功法折扣。
- 每次收益都要写清：省了多少钱、换到什么资格、距离武道门槛近了多少。
