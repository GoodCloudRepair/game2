# 异常门域规则库（秘境规则）

> 本书以「秘境」为主，不做规则怪谈门域。表头保留便于对照。

## 秘境总则

| 项 | 规则 |
| --- | --- |
| 诞生地 | 野外、深山、废弃工地、废弃医院等灵气淤积点 |
| 评级 | 外围 / 浅层 / 中层 / 深层；由秘境勘察部评定 |
| 开放 | 评级后有限开放；无证擅入违法 |
| 产出 | 灵石、妖兽材料、天材地宝 |
| 危险 | 妖兽+环境异变；禁止写成无解绝望屠城 |

## 界隙 / 破界长线

| 阶段 | 呈现方式 | 写法边界 |
| --- | --- | --- |
| 001–200 | 封控区、秘境逸散、异常动植物、空间感轻微错位 | 只当城市事故和秘境风险写，不直接解释破界 |
| 201–600 | 大型秘境出现边界层、重复地形、灵气潮汐异常 | 官方研究线推进，陆衡以洞察看见局部裂痕 |
| 601–800 | 界隙扩大，跨区秘境互相牵连，畸变领主前兆明显 | 破界线浮出，但仍以官方协同维稳为主 |
| 801–1000 | 异界裂缝与稳界任务成为主轴 | 不写飞升换地图；核心是稳住现实世界边界 |
