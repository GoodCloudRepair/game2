# 势力与组织台账

## 超凡管理局（正面·核心）

| 项 | 内容 |
| --- | --- |
| 定位 | 国家级超凡事务主管机构；与军警协同 |
| 下属 | 特勤队（应急清剿）· 武道总队（考核/编制）· 秘境勘察部（秘境评级与开放） |
| 形象 | 专业、可靠；永不写成废物或整体腐朽 |
| 澄江站 | 地级市派出机构；沈岚所属 |
| 对民间 | 鼓励正规武道培训与报考；打击非法猎妖团伙 |

## 权限与资质（简）

- 无证猎杀城内妖兽：违法，由管理局处理
- 学生外围志愿辅助：仅限官方划定外围，需学校推荐、家长知情、带队人、保险、路线和回执
- 临时外围学徒辅助资格：三十天有效，仅限观察记录、设备编号、撤离辅助；不得入红线、不得接触材料、不得单独行动
- 初级武者证：可接外围任务、进入部分浅层秘境外围
- 特勤实习：大学阶段后

## 封控事故响应流程（卷01开篇口径）

| 环节 | 执行方 | 正常表现 | 可给主角的剧情收益 |
| --- | --- | --- | --- |
| 1. 风险提示 | 学校广播/班主任 | 校园安全播报，要求学生留在指定区域或按班撤离 | 让事故从现实秩序切入，不像随机刷怪 |
| 2. 学生疏散 | 班主任/安保/民警 | 按班级、校门、道路分流，不允许围观拍摄 | 陆衡和陈屿被卷入边缘险情，但不是主动作死 |
| 3. 交通封控 | 民警/交警 | 拉警戒线、控路口、清空人流 | 体现官方反应快；普通人只能配合 |
| 4. 异常处置 | 特勤队 | 特勤接管核心危险，普通警力负责外围秩序 | 官方不降智；主角只在缝隙里发挥感知价值 |
| 5. 证人登记 | 管理局澄江站 | 记录目击者、异常表现、受伤情况 | 陆衡的洞察表现被写入备案，而不是私下瞒天过海 |
| 6. 医疗检查 | 校医/医院 | 排查污染、外伤、精神应激 | 给陆母线和费用压力切口 |
| 7. 学校上报 | 学校/教育口 | 提交涉事学生名单，申请补测或特殊说明 | 合理导向武考补测资格 |
| 8. 后续复核 | 管理局/武道培训中心 | 对异常感知、反应能力做非公开复核 | 周冕登场有制度理由，不是路边收徒 |

原则：封控事故可以混乱，但不能写成官方无能；主角能赢的是「信息差中的一秒判断」，不是替官方清场。
