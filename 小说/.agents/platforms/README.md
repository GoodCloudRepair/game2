# 平台抓力审稿插件

主笔**底审**（章节记录第10节 + 文风抽检）与平台无关，每章必做。

**平台分**由本书 `18_简介封面与运营包装.md` 声明的插件 ID 决定：

```markdown
## 平台插件

| 项 | 值 |
| --- | --- |
| 主投放平台 | 番茄 / 起点 / … |
| 启用插件 | `tomato` / `webnovel-generic` / （空=只底审） |
| 评分落盘 | 可选：本书 `13_…评分.md` 或仅交付报告 |
```

| 插件 ID | 文件 | 适用 |
|---------|------|------|
| `tomato` | [tomato.md](tomato.md) | 番茄小说读者抓力 |
| `webnovel-generic` | [webnovel-generic.md](webnovel-generic.md) | 通用网文抓力（未挂专用插件时的默认） |

规则：

1. `启用插件` 为空 → 交付只做底审，不打平台分。
2. 平台分**只预警、不擅自按分重写**；底审硬伤必须改到过关。
3. 新增平台：在本目录加 `{id}.md`，书内 `18_` 改插件 ID 即可。
