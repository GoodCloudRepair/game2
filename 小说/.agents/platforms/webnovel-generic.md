# 平台插件：webnovel-generic（通用网文）

评分口径：不绑定单一 App 的**连载抓力**审稿。  
适用：主投放已定（如起点）但尚无专用插件，或跨平台试水时的默认抓力审。

## 维度

| 维度 | 看什么 |
|------|--------|
| 吸引度 (0–10) | 开篇是否尽快入戏、章内是否有可见进展/小兑现、情绪或冲突是否抬升、章末是否想点下一章 |
| 衔接度 (0–10) | 是否接住上章未收之钩、目标/场景切换是否清楚、有无「读完不知下一章要干嘛」 |
| 负担 (低/中/高) | 单章新设定、人名、规则是否过载（高负担需在判断里说明） |
| 一句话判断 | 本章追读理由 / 掉点风险 |

不强制：千人留存、订阅转化等平台后台指标。

## 交付时输出

```markdown
### 平台审（webnovel-generic）
- 吸引度：X.X
- 衔接度：X.X
- 负担：低/中/高
- 判断：…
- 预警：（偏低时 1–3 条建议；不自动改文）
```

## 落盘

默认**只写交付报告**。若本书自行建立评分表，可按表头追加一行。
