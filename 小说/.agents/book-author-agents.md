# 单书「作者 Agent」说明

每本书根目录可放一份 **`作者Agent.md`**：写该书时，通用主笔先读它进入**平台 + 题材人设**，再跑台账流程。

## 为何分开

| 层 | 文件 | 管什么 |
|----|------|--------|
| 通用主笔 | `小说/.agents/novel-writer.md` | 看纲、章节记录、底审、回填、口令路由 |
| 平台插件 | `小说/.agents/platforms/*.md` | 抓力打分维度 |
| **单书作者** | `【书名】/作者Agent.md` | 「你是谁、擅什么、本书指纹、不要写成别的书」 |
| 硬约束台账 | 书内 `00` / `07` 等 | 主线四层、文风细则（作者 Agent 指向它们，不替代） |

## 模板（开新书可复制）

```markdown
# 《书名》作者 Agent

## 人设（写章时进入此角色）

你是一名资深【平台】作者，擅长【类型】……

```
