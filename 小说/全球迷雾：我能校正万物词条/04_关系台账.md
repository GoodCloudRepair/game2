# 关系台账

关系不是静止设定，而是会随着章节推进不断变化。  
这份台账只记“已经发生变化的关系”和“未来准备推进的方向”。

| 关系双方 | 当前关系 | 核心纽带/矛盾 | 最近变化 | 后续方向 |
| --- | --- | --- | --- | --- |
| 沈辞 / 沈知微 | 兄妹，强保护关系，已绑定封门结构 | 沈辞想救她，知微却正在压住城市门缝 | 第108-112章知微继续稳住17格，帮沈辞压住外线回执和真门名倒写反噬；沈辞在撤城与替换方案前明确拒绝把她当作一次性锚点 | 第113章必须用规则证明`共鸣项不可替换`，不能只靠情绪反驳顾焰 |

| 沈辞 / 门域核心 | 已进入真门名倒写与封门夜战段 | 核心只认未写圆活项，并不断切断门外关系 | 第108-112章核心以外线回执稳内名，排除文昌回头线后开始倒写真门名，同时反读沈辞自身词条；第一栏失守后撤城和替换方案都暴露定层被接管风险 | 后续继续读倒写笔画，但必须防止活项被门反读和共鸣项被工具化 |

| 沈辞 / 财团联盟 | 外缘记录与工具化利用关系 | 财团代表旧秩序和效率优先逻辑 | 第110-112章顾焰黑箱连续记录门反读、文昌动线与共鸣项替换可行性，财团线从记录沈辞价值推进到尝试替换知微源位 | 从信息差和试探，走向“能不能把人当参数”的正面冲突 |

## 卷01收束关系补记

- 沈辞 / 沈知微：第113-120章从“不可替换”推进到“互相托命后幸存”。知微以半称前痕主动轻应，沈辞用她争来的三息读门；封门后她仍有源位灰痕，完整姓名与完整亲称暂不可碰，兄妹线完成阶段性回收但未完全恢复日常。
- 沈辞 / 林照：第116章林照用只报物不报人名的方式帮沈辞抵住门反读，第119章明确支持留缝保未归项，第120章继续替知微挡系统编号；关系从内外协作推进到现实层互相托底。
- 沈辞 / 杜衡：第118-120章杜衡接受沈辞最终旧边名并执行留缝封门，随后递出临时证；关系从封门互用转为官方吸纳/监管借用。
- 沈辞 / 顾焰：第113章替换方案被规则否决，第117章黑箱记录近似改名不可复现，第119-120章复核标记未撤；关系从工具化冲突转为财团持续观察与后续解释权争夺。
- 沈辞 / 城防局：卷尾正式从临时协查升级为`监管借用`，沈辞被纳入体系但不是自由入职，也不是完全信任。

## 卷02开篇关系补记

- 沈辞 / 城防局：第121章临时证正面入档，背面自补`不得离开监测半径`；关系从“监管借用”进一步制度化为“可调度但受监测半径限制”。
- 沈辞 / 城防局：第122章反读评估判出`调度污染：三格`与`不稳定变量`，关系从单纯监管推进为“可用但不能直连系统”的隔离使用状态。
- 沈辞 / 城防局：第125-127章进入安置区后，城防必须依赖沈辞处理编制表、签名栏和床位号错名，但每次都限制他只稳局部、不得越过监测半径。
- 沈辞 / 杜衡：杜衡明确不解释更深背注，只执行记录与评估传票；两人关系进入更冷的制度化互用。
- 沈辞 / 杜衡：第122章杜衡确认“初评通过不代表安全”，继续只给执行边界，不替沈辞争取信任背书。
- 沈辞 / 杜衡：第123-127章杜衡一边承认城防印盖不上候选字，一边把沈辞投入安置区现场；关系变成“高价值工具但必须限位使用”的现实互用。
- 沈辞 / 林照：林照继续替知微挡系统编号，并把`紧急联系人`栏压空；关系保持现实边线互保。
- 沈辞 / 林照：第124-127章林照用医疗旧识别片、胶膜、红水杯旧痕和白床线压制配合沈辞，关系从医疗托底推进到共同处理现实表格失名。
- 沈辞 / 沈知微：知微仍只对两下旧触产生轻应，没有恢复完整称名；兄妹线继续克制，不进入团圆状态。
- 沈辞 / 沈知微：第124章知微灰痕对HL-02轻应，第127章十七白床线被安置床号短暂借走；兄妹线从后遗症转入第二次失名敏感点。
- 沈辞 / 顾焰：顾焰的联席蓝点只能外缘可见，无法写入`编目师候选`来源；解释权争夺仍在，但权限边界再次显露。
- 沈辞 / 顾焰：第123章顾焰以半页旧实验附注重新抬财团解释权，但仍只能给半页、半句和同类手法，不能给完整来源。
- 沈辞 / 顾焰：第128章从现场对峙转为信息口争夺，顾焰不进现场也能通过复核通道抢半页；沈辞拒绝让他替城防放大旧标注，继续防止财团添字或定义来源。
- 沈辞 / 沈知微：第131-133章亲属登记室连续逼近`沈`字边、代领人待核和`两下按腕 / 待核兄称`，沈辞始终不接完整亲属关系，只把行为痕迹压成不可入亲属栏。
- 沈辞 / 林照：第133章紧急联系人栏试图抓取林照医疗处置痕迹，两人共同把它压回医护项；关系继续保持“医疗托底，不替亲属”的边界。
- 沈辞 / 杜衡：第129-133章杜衡继续依赖沈辞核验文修手法和亲属登记旧痕，但只给任务调度与封存权限，不放开完整信任。
- 沈辞 / 林照：第134-138章林照从“医疗托底”进一步被规则明确成“不替家属签”的医护边界；沈辞保护她不被亲属栏吞掉，她保护知微白床线不被亲属栏借走。
- 沈辞 / 杜衡：第134-138章杜衡承认沈辞在救援编号、岗位名和临时调度上有用，但只开边格、不放总调度；关系继续是高压限权互用。
- 沈辞 / 沈知微：第138章白床线被亲属栏短抓，沈辞仍不接完整兄称，只把她推回医疗监测；兄妹线继续是推迟团圆的保护关系。
- 沈辞 / 杜衡：第139章总表诱导整表清零，杜衡需要现场止损但仍没让沈辞直连总表；两人关系推进为“必须听他判断，但不能放他接管”的清零前对峙。
- 沈辞 / 顾焰：第139章顾焰要求保留全表样本，沈辞默认样本有价值但不让财团接管；关系从截半页报告延伸为全表样本解释权争夺。
- 沈辞 / 林照：第139章林照用“清零会抹掉医疗底痕”继续守住知微和普通幸存者的现实痕迹；她仍只给医疗边，不给亲属口。
- 沈辞 / 杜衡：第140章杜衡暂按沈辞的残痕分栏核验执行并承担责任，但不是完全信任，而是现场止损下的继续限权互用。
- 沈辞 / 顾焰：第140章顾焰从“留样”升级到正式申请调取完整样本，并触发城防调度权待复核；沈辞与顾焰的分歧进入权限争夺层。
- 沈辞 / 林照：第140章林照用十七白床监测底痕证明活人残痕不能抹，关系继续是医疗现实边托底，不进入亲属认回。
- 沈辞 / 杜衡：第141-145章杜衡连续借沈辞判断保住原表、退自动派单、开短指令和残痕边界，但始终用现场封条和窄令限制他；关系进入“调度权争夺下的高压限权协作”。
- 沈辞 / 顾焰：第141-145章顾焰从完整样本调取转为旁读、任务口、短指令和样本确定性施压；沈辞能借他的半个办法，但不能接他的完整路径。
- 沈辞 / 林照：第141-145章林照继续在外线守白床线和医护底痕，提醒别连续接、半径不能跟任务走，并在第145章把三名待核者压成在场生命确认，不接医疗流程。
- 沈辞 / 沈知微：第141-145章知微未直接出场，但白床线多次被任务和半径重算短拉；沈辞保护她的方式仍是把白床线压回医疗监测，而不是认亲或接任务。
- 沈辞 / 杜衡：第146-150章杜衡继续依赖沈辞判断压住转送、回执、单位和仲裁，但每一步都要求记录、限项和退出边线半步；关系仍是限权协作，不是信任放权。
- 沈辞 / 顾焰：第146-150章顾焰在样本确定性与现场留人之间持续施压，只让出延后对应和影号用途；沈辞继续只借最窄有效部分，不接财团路径。
- 沈辞 / 林照：第146-150章林照保持医疗边界，只给止血建议和生命确认，拒绝签收、接诊和调度单位；她与沈辞的协作继续围绕“不让医疗口变亲属/任务口”。
- 沈辞 / 三名待核本人：第145-150章从被同填`周岷`的待核对象，推进到学会守住袖章、布包和鞋带旧痕自证；他们不是队友，但已能按沈辞规则保持沉默和守物。
- 沈辞 / 杜衡：第151-155章杜衡继续把沈辞当可用但危险的损痕判断口，只执行背注复核、传唤改写、封袋旁读和旧章转盖，不替沈辞担保来源，也不让他升成完整调度对象。
- 沈辞 / 顾焰：第151-155章顾焰从调度权争夺转向来源解释权争夺，能拿出半门样本和旧文修路径一角，但每一步都被沈辞压成旁读、半开和回流路径，不能写完整候选来源。
- 沈辞 / 林照：第151-155章林照仍只守医疗现实边，在旧章和封袋压迫时提醒身体代价，不替背注来源、取件、盖章或三名待核者作医疗旁证。
- 沈辞 / 沈知微：第151-155章知微未直接出场，白床线没有被二次背注、文修回流件或旧红章借走；兄妹关系继续通过“不让她被写进来源旁证”被保护，而不是通过认亲推进。
- 沈辞 / 文修旧流程：第151-155章关系从“材料手法可识别”推进到“旧文修口回流路径贴近本人职业起点”，但沈辞只承认证物路径，不承认候选来源和原修复员责任。
- 沈辞 / 杜衡：第156-160章杜衡继续只做限权执行，把民用修复单、旧工号、夜借、半门批号和更高复核席都压进记录边界；两人关系仍是“必须用他判断，但不能替他担保”。
- 沈辞 / 顾焰：第156-160章顾焰用旧工号、夜借、批号和更高复核席持续逼近父线，但最终只承认财团握住一角；沈辞继续借他露出的半层信息，拒绝接完整来源。
- 沈辞 / 林照：第156-160章林照仍不进来源线，只在外线提醒别签、别旁证，守住沈辞身体和知微医疗边；关系继续是现实托底而非规则代答。
- 沈辞 / 沈知微：第156-160章知微白床线未被文修旧流程牵走，兄妹线继续保持保护性延迟，不通过父线、旧工号或更高复核席推进认亲。
- 沈辞 / 文修旧流程：第156-160章文修旧流程从回流路径推进到民用修复单、旧工号、夜借未销、半门批号和更高复核席；沈辞确认这些只证明缺失原因和经手痕，不证明候选来源。

- 沈辞 / 顾焰：第161-165章顾焰以半门价值、父线残角、完整归档和训练化方案施压，甚至用“降低现场依赖”诱惑沈辞；沈辞继续拒绝让财团把人和职业线变成样本。
- 沈辞 / 林照：第161-165章林照继续只守医疗监测和身体线，不替半门旁签、安全记录或训练样本作证；她的功能仍是把白床线和沈辞身体代价压在现实边。

## 使用规则

1. 关系变化必须写清楚“因为什么变”。
2. 如果关系变了，但没有事件支撑，那就是写飘了。
3. 感情线推进必须和共同经历绑定，不能凭空升温。
