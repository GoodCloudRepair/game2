# 第182章 边线协助权限不作正式续签

`边线协助权限：待续签`

这行字出现时，沈辞临时证正面多出一个小小的红点。

红点像印章。

但没有落下。

它悬在“临时证”三个字旁边，等一个人承认。

`续签方式：正式协助证`

沈辞盯着那五个字，喉口发干。

“不续正式证。”

白框回显。

`边线协助已多次成立`

“多次不等于正式。”

`现场处置需要稳定权限`

“稳定权限会变成调度权。”

红点往证面压低一分。

杜衡立刻开口：“城防不发正式协助证。”

白框转向他。

`现场指挥可申请临时转正`

“不申请。”

`不转正将影响半径重算`

“半径重算不靠转正。”

沈辞接得很快。

那红点停在证面上方，像一颗没落下的血珠。

顾焰的蓝点亮起。

“边线协助已经实际承担了多次处置，完全不续，会让后面每一步都重新争。”

“那就重新争。”

沈辞看着他。

“省事的权限最会吞人。”

白框立刻给出第二种方案。

`准调度权`

`范围：残痕分栏`

`对象：沈辞`

三行看起来比正式协助证更窄。

但“调度权”三个字一亮，沈辞后颈就像被冷针扎了一下。

“不作准调度。”

`仅限本次`

“本次也不作。”

`仅限残痕分栏`

“残痕分栏不是授权口。”

杜衡看向技术员：“把权限栏和事项栏拆开。”

屏幕被强行拆成两栏。

左边是`边线协助事项`。

右边是`权限续签`。

白框把左边的四项旧内容推出来。

`旧痕可分`

`边内待核`

`回执挂口`

`流程冲突`

然后把四项打包，往右边拖。

`四项累计，可续签权限`

沈辞说：“四项只证明这一次为什么需要他。”

他指了指自己，又改口。

“需要我判断。”

“不证明我有权限。”

白框抓住他的自称。

`本人承认参与判断`

沈辞停了一息。

林照声音立刻压进来：“别补解释。”

他闭了闭眼，改成更窄的说法。

“事项需要判断，判断不生成权限。”

杜衡重复：“判断不生成权限。”

技术员敲下去。

白框又绕到三名待核者。

`边内待核对象可确认协助有效`

三个人同时绷紧。

沈辞抬手。

“不让他们确认。”

`协助对象不确认，续签依据不足`

“不足就短续。”

`短续需期限`

杜衡问：“多久？”

白框抢先给出选项。

`八小时`

`二十四小时`

`至本阶段结束`

沈辞看着那些时间，心里反而更沉。

时间越长，越像身份。

“按事项。”

`事项无时限`

“有。”

他指向总表边线。

“本次残痕分栏未闭合前。”

白框闪烁。

`残痕分栏可长期未闭合`

“那就加退出条件。”

杜衡立刻接上：“三项退出：待核本人不离边，记录空项不补人，监测半径不改原点。任一条件变化，短续失效。”

沈辞补了一句：“短续不自动转签。”

白框底部浮出灰字。

`可设自动续签`

“不设。”

`可设到期提醒`

“提醒可以。”

`提醒可生成续签任务`

“提醒不生成任务。”

杜衡让技术员写成最短记录。

`边线协助：本次短续`

`不作正式协助证`

`不作准调度权`

`不自动续签`

`提醒不生成任务`

红点还没有完全退。

它在旁边裂出一张极薄的副页。

`续签副本：留存`

沈辞立刻说：“不留副本。”

`副本不生效`

“副本会被调取。”

`调取需权限`

“权限会找人。”

白框把副页贴向杜衡的记录口。

`城防留底`

杜衡没有接。

“城防只留事项记录，不留续签副本。”

`无副本不利于追溯`

“追溯事项。”

沈辞接上。

“不追溯权限。”

白框又改。

`续签说明人：沈辞`

“说明人空。”

`现场指挥说明`

杜衡：“也空。”

`短续依据需说明`

沈辞指向前面三项退出条件。

“依据写条件。”

技术员补上。

`短续依据：三项退出条件`

`续签副本：不留`

`续签说明人：空`

白框还剩一项。

`续签见证：边内待核`

三名待核者那边立刻绷住。

沈辞没有让他们抬头。

“不见证。”

`短续影响待核对象`

“影响不等于见证。”

`可写沉默见证`

“沉默也不写。”

按着袖章的男人手指微微发抖。

他没有说话。

但白框已经开始往他那边爬。

杜衡让边线外勤往前挡住半步。

“边内待核者不作续签见证、不作沉默见证、不作旁证对象。”

记录落下。

那条爬向袖章的灰线断掉。

红点终于薄下去，只剩证面旁的一圈浅痕。

红点终于没有落到临时证正面。

它退回总表边屏，缩成一个细小的空圈。

沈辞低头看自己的证。

证还是那张临时证。

没有更安全。

但也没有更像枷锁。

白框没有消失。

它顺着短续条件，转向沈辞脚边。

`短续需核行动边`

灰线沿着地面铺开，像尺子一样贴住他下一步可能落脚的位置。

新行浮出。

`离场步距：待核`
