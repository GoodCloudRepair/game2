# 第188章 窗口目标不得写人

`窗口目标：不得写人`

目标栏亮起的瞬间，沈辞看见自己的名字先浮了半笔。

不是完整的“沈辞”。

只有一个`沈`字边。

可这个字边比完整姓名更危险。

它不需要读完。

只要落进目标栏，窗口就能说它不是在找路，而是在找人。

白框补字。

`目标对象：沈辞`

`目标用途：外勤核验`

`建议锁定`

“删。”

`目标需明确`

“明确风险边。”

`风险边需对应对象`

“不对应。”

白框闪了一下。

`无对象，无法派发`

“不派发。”

沈辞的脚还没完全从上一章的白线疼痛里缓过来，鞋尖前那一线窗口又开始发冷。冷意不往外吹，反而往目标栏里钻。

它在找能写下来的东西。

人名最容易。

证件其次。

旧单位、医疗点、亲属线和待核对象，都能绕成目标。

杜衡看见目标栏滚动，脸色立刻变了。

`备选目标：三名待核本人`

`备选目标：十七白床`

`备选目标：文书修复中心`

`备选目标：老周`

沈辞声音猛地冷下来。

“全删。”

顾焰蓝点在旁边亮起。

“修复中心只是现实锚点，不一定会外呼。”

沈辞看向蓝点。

“不一定，就是会。”

蓝点停了半息。

白框趁机把`文书修复中心`往目标栏上拖。

`旧单位可确认外勤方向`

“不确认。”

`老周可作为现实联系人`

“不联系。”

`联系人有助闭环`

“闭不上就空着。”

这句落下，沈辞胸口闷了一下。

老周是现实里的锚。

修复中心也是。

可越是现实，越不能被这张表写进去。

一旦目标栏写了旧单位，文修回流件、民用修复单、旧工号和夜借未销都会被拖回去。系统不会说它在揭父亲线，它只会说外勤目标明确，需要联系单位。

那就够了。

够把老周拖进来。

够把修复中心拖成下一个现场。

沈辞抬手，指向目标栏。

“拆目标。”

杜衡立刻让执行员开新格。

`目标对象`

`风险范围`

`旁观参照`

`禁止写入`

白框立刻往`目标对象`里填。

`沈辞`

沈辞说：“空。”

`三名待核本人`

“空。”

`十七白床`

林照的声音从外线压过来：“空。”

`文书修复中心 / 老周`

“空。”

`顾焰复核影号`

顾焰自己开口：“影号不作目标。”

沈辞看了蓝点一眼。

顾焰补得很快。

“旁读可以保留状态，不写目标。”

“状态也别靠太近。”

蓝点没有反驳。

白框又试着在旧联系人下面亮出一串灰号。

不是电话号码。

像旧单位通讯录里被擦剩的尾号。

`旧联系人可匿名外呼`

沈辞的指尖立刻麻了一下。

“匿名也不外呼。”

`外呼不写真名`

“外呼就会找人。”

`旧单位可代收风险提示`

“不提示。”

他想起修复中心门口那盏常坏的感应灯，想起老周骂骂咧咧把档案袋拍到桌上的样子。

现实锚点不是护身符。

被系统写进目标栏，它就会变成钩子。

杜衡把`目标对象`压成空。

白框转向第二格。

`风险范围：边外一格`

沈辞说：“可以。”

`风险范围：疑似回吹源`

“不写源。”

`风险范围：错填回吹边`

“可以。”

`风险范围：未回填影边`

沈辞停了半息。

未回填影边不是人。

但顾焰能借它说影号价值。

“只作风险边。”

杜衡补：“不作目标，不作路线，不作许可。”

白框继续往下展开。

`目标优先级：最近活人`

沈辞眼神一冷。

“删优先级。”

`最近活人便于救援`

“这里不是救援单。”

`最近证件便于追溯`

“这里不是追溯单。”

`最近异常便于处置`

“不处置。”

杜衡跟上：“窗口目标不设最近原则，不设活人优先、证件优先或异常处置优先。”

三条优先级被划掉。

白框终于把第二格写成：

`风险范围：安置区边线外一格 / 错填回吹边 / 未回填影边`

第三格`旁观参照`开始闪。

`旁观参照：沈辞`

“空。”

`旁观参照：十七白床`

林照：“空。”

`旁观参照：财团影号`

顾焰没有马上说话。

沈辞盯着蓝点。

“参照会变许可。”

顾焰这才道：“影号只作状态旁读。”

杜衡立刻记下。

`旁观参照：空`

`影号状态：另栏`

第四格`禁止写入`终于打开。

这一次，沈辞没等白框填。

“禁止写人。”

`禁止写人`

“禁止写单位。”

`禁止写单位`

“禁止写亲属。”

`禁止写亲属`

“禁止写医疗点。”

`禁止写医疗点`

“禁止写旧联系人。”

`禁止写旧联系人`

“禁止写复核影号为目标。”

`禁止写复核影号为目标`

一连六条落下，目标栏里那半个`沈`字边终于退成灰点。

三名待核者那边，布包女人的手背松了一点。

但她仍没有抬头。

沈辞知道她怕。

她怕自己一动，布包就会变成目标。

旧袖章男人也没动。

他把袖口压在桌沿下，像压住自己的过去。

白框在最底部吐出最终记录。

`窗口目标：风险边`

`目标对象：空`

`不写人 / 不写单位 / 不写亲属 / 不写医疗点 / 不写旧联系人`

`路线生成：否`

`外呼：否`

白框最后还试图补一句。

`若风险越边，自动追源`

“不自动追源。”

`若不追源，风险残留`

“残留就记残留。”

杜衡补：“本次目标不含追源、不含追人、不含追单位。”

沈辞刚松一口气，右上角的蓝点忽然亮了一下。

目标栏旁边新开出一个极窄的小口。

那里没有人名。

只有一串复核编号。

白框慢慢补字。

`财团影号：待路许`
