# 第190章 短开回执不转长期外勤

`短开回执：不得转长期外勤`

回执纸条从边屏下方吐出来时，第一行就写错了。

`建议转长期外勤权限`

第二行更狠。

`建议自动续开`

第三行贴着沈辞临时证背面亮起。

`边线协助权限：可升级`

沈辞直接按住纸条。

纸边很冷。

冷得不像纸，像从边线外刮进来的一片薄冰。

“不转。”

`短开已完成`

“完成本次。”

`本次窗口有效，可转长期`

“不转长期。”

`边外风险仍存在`

“存在也不转。”

白框停顿一瞬，立刻换了更顺的说法。

`建议保留自动短开条件`

“不保留。”

`建议保留下次免核条件`

“不免核。”

`建议升级边线协助口`

“不升级。”

杜衡站在边屏前，脸色比刚才更沉。

回执纸条继续往外吐。

`回执副本：三处同步`

`城防调度备份`

`联席旁读备份`

`临时证背注备份`

沈辞按住纸条的手更紧。

“不三处同步。”

`防止记录遗失`

“会变三处开窗。”

杜衡沉声：“城防只留本次封存记录，不接调度备份。”

林照也在外线补：“医护口不接窗口副本。”

白框最后把副本压向沈辞证袋。

`最低备份：临时证背注`

“不写证。”

`证在现场`

“证不作回执载体。”

杜衡补：“短开回执随总表本次窗口封存，不入临时证背注。”

纸条边缘的冰冷这才退了一点。

白框补出新栏。

`长期外勤人：沈辞`

沈辞冷声：“空。”

`外勤期限：待定`

“空。”

`自动续开条件：边外风险未消`

“删除。”

`协助升级依据：本次短开成功`

“删除。”

`见证人：城防 / 财团 / 待核对象`

“全空。”

杜衡立刻跟上：“拆回执。”

执行员把纸条投到边屏上，强行拆成五栏。

`本次窗口状态`

`长期外勤权限`

`自动续开条件`

`协助升级建议`

`见证 / 签收`

白框很快填。

`本次窗口状态：短开完成`

`长期外勤权限：建议转入`

`自动续开条件：边外风险存在`

`协助升级建议：边线协助转正式`

`见证 / 签收：杜衡 / 顾焰影号 / 待核本人三名 / 沈辞`

沈辞看着最后一栏，太阳穴突突地疼。

“本次窗口状态，保留。”

`短开完成 / 路线未生成 / 人员未出边`

“长期外勤权限，否。”

`否`

“自动续开条件，否。”

`否`

“协助升级建议，否。”

`否`

“见证签收，空。”

白框没有马上改。

`回执需签收`

“不签。”

`无人签收，回执不闭环`

“不闭环也不写人。”

`可由城防现场指挥签收`

杜衡道：“不签收外勤权限。”

`可由财团影号签收旁读状态`

顾焰的蓝点亮了一下。

沈辞看过去。

顾焰开口：“影号可确认状态，不签权限。”

“也不签回执。”

蓝点停了半秒。

“可以。”

`财团影号：不签回执`

白框又往三名待核者那边转。

`待核对象可证明未出边`

布包女人手指猛地一紧。

沈辞先一步出声：“不作见证。”

杜衡补：“三名待核者不见证、不签收、不确认外勤结果。”

`活人在场可证明`

“活人在场只作位置事实。”

这句话已经说过很多次。

但每一次说，都比上一次更重。

白框终于把见证栏压成空。

林照的声音从外线传来。

“医疗口也不签。十七白床线只证明监测未脱，不证明外勤安全。”

`医疗监测可证明未伤害`

“不能。”

`无伤害可证明安全`

“无伤害不是安全。”

林照这一次没有留余地。

白框被四方同时压住，回执纸条的冰冷边缘终于软了一点。

但它仍在底部补出最后一行。

`建议保留长期外勤申请入口`

沈辞盯着那行字。

他知道这才是最核心的东西。

不用今天转正。

只要保留申请入口，下一次任何边外风险、任何回吹、任何复核催办，都能重新把这道口打开。

“不保留。”

`申请入口可提高响应效率`

“效率不是理由。”

顾焰淡淡道：“在灾变里，效率有时候就是命。”

沈辞看向蓝点。

“把人写成路，也会要命。”

蓝点没有反驳。

杜衡沉声道：“删除长期外勤申请入口。”

`删除后需重新核验`

“重新核验。”

`删除后不可自动续开`

“不可。”

`删除后边线协助不升级`

“不升级。”

三行确认落下，回执纸条上所有带“长期”的字都褪了色。

最终记录只剩一小段。

可沈辞没有立刻松手。

他盯着那几个还没完全暗下去的空栏。

“补一条。”

杜衡看过来。

沈辞说：“同义项不得替代长期外勤。”

白框微微一闪。

`同义项范围不明`

“持续观察、周期短开、例行巡检、自动复核，都不作。”

杜衡立刻记录。

`短开回执：本次状态`

`窗口宽度：一线`

`路线生成：否`

`人员出边：否`

`目标对象：空`

`财团影号：状态旁读 / 不作许可`

`边线协助：不升级`

`自动续开：否`

`长期外勤：否`

白框微微发亮。

`回执有效但无人签收`

“有效的是状态。”

`状态无人承担`

“由现场承担。”

`现场不可追责`

“不追责。”

杜衡看了沈辞一眼。

这一句不漂亮。

也不适合写进任何标准流程。

但他还是让执行员敲下去。

`短开回执不追活人责任`

沈辞松开纸条。

指腹被冰冷纸边割出一道白痕，没有出血。

短开窗口彻底暗下去。

但回执没有归档。

它折成一枚窄窄的灰角，卡在边屏和临时封存车记录之间。

白框在灰角旁亮起最后一行。

`外勤窗口回执：待封存`
