# 第199章 外沿灰栏只读不巡

`外沿灰栏：只读不巡`

读口打开得很慢。

没有门缝。

没有风。

也没有边外实景。

总表边屏底部只是多了一片更淡的灰。

灰得像纸背上的水印。

沈辞盯着那片灰。

他看不见边外。

只看见本次人工旁观留下的状态。

`外沿灰栏：当前`

`风险回吹：残留`

`自动化：未生成`

`班次：未生成`

四行很干净。

干净到像在诱导人多读一眼。

顾焰的蓝点先亮。

“只读状态至少要标出灰点位置。否则这个旁观结果没有复核价值。”

沈辞说：“不标位置。”

`建议标记灰点`

灰栏顺着顾焰的话往外铺出七个小点。

小点排在屏底，像城郊地图上被按亮的灯。

沈辞的指节发白。

他没有靠近。

“灰点不命名。”

`未命名灰点不可复核`

“不可复核。”

`可编号：灰点一至七`

“编号也是命名。”

`编号不涉及地名`

“会变巡检点。”

杜衡立刻道：“删除灰点编号。”

执行员切掉编号预案。

七个小点仍在。

它们不再写编号，却开始轻轻扩大。

每个点边缘都冒出细线。

细线连向安置区总表边屏的旧槽、封存车记录槽和固定驻点位置栏。

像一张没有完全画出来的巡检路线。

沈辞说：“不连线。”

`只读需对照边界`

“对照不连线。”

`不连线无法判断巡看顺序`

“不巡看。”

`只读不等于不看周边`

“只读就是不看周边。”

灰栏微微一停。

它把七个点推成一条半弧。

半弧外侧浮出几个很淡的字。

`建议路线预览`

杜衡低声骂了一句。

“路线预览删除。”

`预览不执行`

“预览会生成路线。”

`路线未授权`

“未授权路线也是路线。”

沈辞看着半弧。

那条弧没有延伸到安置区外。

但它已经在总表里留下了“可到达”的想象。

只要想象成立，下一次就会有人问为什么不到。

“外沿灰栏只读当前状态，不生成可达性。”

半弧消失了一半。

剩下的一半还压在边线外勤窗口旧槽旁。

顾焰开口：“可达性可以不写，但灰点风险级别必须留。否则你们连哪个方向更危险都不知道。”

沈辞说：“不分级。”

“不分级，城防无法优先处理。”

“优先处理会变巡检顺序。”

“不是巡检，是应急。”

“应急也要触发。现在没有触发。”

蓝点安静了片刻。

灰栏趁这段沉默，又推出一个更像记录的选项。

`建议截取只读快照`

沈辞看见“快照”两个字，立刻想起上一轮只读样本影。

快照不是路线。

也不是巡检。

可快照会有边框。

有边框就会有坐标，有时间，有读取人。

下一次对比快照，巡检就会以“差异分析”的名义回来。

“不截图。”

`快照可证明未巡`

“证明未巡，也会留下巡前和巡后。”

`可去除时间`

“去除时间也有顺序。”

`可去除读取人`

“仍有读取动作。”

杜衡接上：“只读状态不生成截图、快照、对比底图或差异分析。”

执行屏删掉快照项。

总表边缘却短暂亮出一个小小的放大镜标识。

沈辞说：“放大镜也删。”

`仅为查看工具`

“工具会改变读取方式。”

放大镜灭掉。

灰栏底下还浮出一条小得几乎看不清的灰线。

`可标注：未巡原因`

沈辞说：“不写原因。”

`原因可证明限制合理`

“原因会变例外条件。”

`例外条件可防误用`

“例外条件会被复用。”

杜衡补：“只写未巡结果，不写未巡原因和例外条件。”

执行员删得很慢，像怕删掉最后一点能交差的解释。

灰栏却继续。

`建议标注：低 / 中 / 高`

七个小点里有两个开始发黑。

黑色很淡，却比其他点更像在叫人看。

沈辞的眼前又黑了一下。

这不是普通标注。

这是让危险自己排队。

排出轻重，就会有先后。

有先后，就会有路。

“不标等级。”

`高风险不标，可能延误`

“本次旁观不处置。”

`旁观发现风险，应触发处置`

“旁观不发现新风险。”

`灰点已显示`

“灰点只作灰栏状态，不作风险点。”

杜衡补：“灰点不命名、不编号、不分级、不触发处置。”

执行员敲下去。

两个发黑的点缓慢褪回灰色。

林照外线传来一声很轻的吸气。

“沈辞，你在看它放大。”

沈辞立刻移开半寸视线。

不是不看屏。

是不盯小点。

小点在他的视线边缘像活物一样抖。

`建议二次放大`

`仅放大灰栏，不进实景`

他低声说：“不放大。”

`二次放大可确认是否误报`

“不确认。”

`误报会影响后续判断`

“后续判断不继承。”

`本次判断仍需准确`

“本次只读，不校正灰点。”

这句话让灰栏明显一暗。

沈辞感觉耳后那道热线往下流了一点。

他没抬手去擦。

林照也没再催。

她知道这时候任何“处理伤口”的动作，都可能被灰栏抓成医护旁观介入。

七个灰点终于收回纸背水印里。

屏底只剩四行。

`外沿灰栏：只读`

`巡看：否`

`点名 / 编号 / 分级 / 路线预览：否`

`二次放大：否`

沈辞看完，才慢慢呼出一口气。

空气里有旧纸和金属发热后的味道。

封存车没有动。

边线旧槽也没有响。

这说明读口没有转成巡检。

三名待核者也没有被位置线牵动。

布包女人的手依旧按在布包口。

鞋带男人脚尖没跨线。

年轻男人低着头，眼睛没有看灰点的位置。

他们都在学会不替系统补动作。

杜衡看向执行屏。

“记录只读状态。”

执行员按下去。

总表边屏底部吐出一张很薄的灰页。

灰页没有编号，只有标题。

`人工旁观回执`

下一行随即亮起。

`签收人：待指定`

沈辞按住金属架的手背浮出青筋。

灰页又补出一行更小的字。

`人工旁观回执：不得找人签`
