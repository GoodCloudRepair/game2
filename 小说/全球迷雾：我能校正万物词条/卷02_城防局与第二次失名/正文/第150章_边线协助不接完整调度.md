# 第150章 边线协助不接完整调度

`候选人可否接入`

这一次，临时证没有立刻发烫。

它先静了一秒。

然后从折痕下面渗出一圈灰。

沈辞知道，这比发烫更糟。

发烫是外面在写。

渗灰是里面在应。

`边线协助权限：待确认`

`候选人可否接入`

总表把问题问得很小。

不是完整指令。

不是最终调度单位。

只是“可否接入”。

可一旦他点头，前面所有被退掉的东西都会换个名字回来。

接入之后，任务派发口能找他。

转送回执能找他。

冲突仲裁能找他。

最终调度单位虽然暂缺，但会默认经过他。

沈辞把临时证按在掌心，没有看证背。

杜衡盯着屏幕。

“接入范围能不能限死？”

技术员答得很快。

“可以填限制项，但系统要求候选人确认。”

顾焰说：“不确认，边线协助口可能关闭。”

边线外勤窗口只剩几十秒。

灰框边缘开始发薄。

三名待核者都站在边里。

袖章、布包、鞋带三条旧痕暂时压住他们。

可窗口一闭，如果没有新的协助口，复核名单会重新要求转送。

沈辞看着灰框。

他不能接完整调度。

也不能让边线现在闭上。

这就是它真正等的地方。

让他在“救眼前三个人”和“保住自己不成入口”之间选。

少年脚下鞋带断口轻轻抖了一下。

年轻女人把布包放低，露出底缝。

中年男人把袖章烧边往外翻，血还在渗，却没盖住针脚。

他们都没说话。

只把物留在能被看见的位置。

沈辞忽然明白，这一章不是他一个人在撑。

只要他们不把自己交给名字，旧痕还能替他们多站一会儿。

他开口。

“接边线协助。”

总表立刻亮。

杜衡脸色一沉。

沈辞下一句压上去。

“不接完整调度。”

屏幕停顿。

`限制冲突`

冲突两个字刚亮，临时证背面就试着补解释。

`边线协助属于任务调度子项`

`子项默认继承主项`

主项。

沈辞看见那两个字，耳后旧伤像被针穿过。

主项就是完整调度。

如果默认继承，前面所有限制都会变成子项说明。

说明不是禁止。

只是备注。

备注可以被上级覆盖。

他把指甲压进证背折痕。

“不继承主项。”

`子项不可独立`

“那就不叫子项。”

杜衡立刻说：“改名？”

沈辞摇头。

“不改名。”

改名更危险。

一旦给它新名，系统会要求定义。

定义会扩权。

他只能用旧位置。

“挂残痕分栏下。”

总表静了一下。

残痕分栏本来不是任务。

它只是第140章保住的窄核验方案。

挂在它下面，边线协助就不能继承完整调度，只能继承残痕限制。

屏幕回：

`残痕分栏非调度项`

沈辞说：“所以不接完整调度。”

沈辞没有退。

“协助只限四项。”

他伸出手指，一项一项压。

“旧痕可分。”

`袖章 / 布包 / 鞋带`

“边内待核。”

`对象不离边`

“回执挂口。”

`不找活人签`

“流程冲突。”

`不裁本人`

四项落下，临时证灰圈往外散。

总表试图在第五项补：

`候选人确认`

沈辞直接压掉。

“不确认人。”

`确认权限`

“确认限制。”

他把第五项空着。

空项旁边很快渗出一条灰线，想往他名字上搭。

他没有让灰线碰到证面。

只把证背折痕露出来。

折痕是不完整的。

也是前面拒绝完整指令留下的损痕。

灰线碰到折痕，被迫停住。

`候选人确认：损痕留边`

这行字不好看。

但它没有写沈辞全名。

也没有写执行人。

杜衡看了一眼，没有问。

他知道这是沈辞用自己的损证边，替所有限制挡了一下。

但这不是信任。

他仍旧下令：“记录不写姓名，只写临时证损痕。”

技术员照录。

证背那行`完整指令待写入`突然亮了一瞬。

像被唤醒。

沈辞用指甲按住折痕。

旧伤疼得他眼前白了一下。

“不生成完整指令。”

总表回：

`边线协助需调度编号`

杜衡立刻说：“用窗口编号。”

`窗口编号即将失效`

林照外线道：“用残痕分栏编号。”

`分栏编号非任务编号`

顾焰终于开口。

“用旁读序列。”

杜衡看向蓝点。

“你又想把它接回复核口？”

顾焰说：“旁读序列已经存在，不需要新任务编号。”

沈辞看着屏幕。

顾焰给的办法仍然带钩。

旁读序列来自复核口。

用它，就等于承认财团旁读参与了边线协助。

不用，完整指令会重新生成。

他只能再压一层。

“旁读序列只作影号，不作调度号。”

顾焰沉默。

杜衡补上。

“影号不离场，不归档，不复制原表。”

林照说：“不接医疗名。”

三边同时压下。

总表上四项限制被挤成一行很窄的灰字。

`边线协助：临时成立`

`范围：旧痕可分 / 边内待核 / 回执挂口 / 流程冲突`

`不接完整调度`

沈辞胸前的临时证终于冷下来。

不是恢复。

是被暂时封住。

复核名单上的三行`周岷`没有消失。

但旁边的三条旧痕亮得更稳。

袖章旧痕。

布包旧痕。

鞋带旧痕。

底部的无名转送倒计时退成灰色。

`转送暂缓`

`待核者可留场`

复核名单还剩最后一口气。

它把三行`周岷`往中间挤，试图合成一个总名。

沈辞没有再动用临时证。

他看向三名待核者。

中年男人先把袖章烧边压住。

年轻女人把布包底缝翻开。

少年把断鞋带踩在脚边，不让它被推床黄灰盖住。

三条旧痕同时发亮。

`周岷`三个字被迫停在姓名栏里，没有再往旁边漫。

这一次，是他们自己守住了物。

沈辞没有替他们说一句多余的话。

亲属登记室里有人很轻地吐出一口气。

杜衡没有让任何人放松。

“分栏留场，三人隔位，继续闭名。沈辞退出边线半步。”

沈辞照做。

他刚退半步，临时证背面忽然又冷了一下。

这次没有任务。

没有回执。

也没有名单。

只有一行比前面都细的字，从`编目师候选`下面慢慢渗出来。

`二次背注：待复核`
