# 第192章 外勤复盘不得生成模板

`外勤复盘：不得生成`

那六个字没有落在回执上。

它们落在边屏最下面，像一张新表的页眉。

沈辞刚松开的手又按了回去。

页眉下面开始自动排版。

`复盘模板一：窗口宽度`

`复盘模板二：半步核验`

`复盘模板三：目标栏处理`

`复盘模板四：影号用途`

`复盘模板五：回执封存`

五行字排得很整齐。

整齐得不像刚才那场混乱。

像有人把所有危险洗干净，晾成可以发给下一组执行员的流程单。

沈辞盯着第一行。

“不生成模板。”

`复盘可降低重复错误`

“会复制错误。”

`本次处置有效`

“有效的是限制，不是步骤。”

杜衡的脸色沉下去。

他知道这话难听，却准。

流程最爱把一次侥幸写成经验。

而这里的每一项经验，都带着能杀人的空格。

白框先从最无害的一项开始。

`窗口宽度：一线`

`建议纳入标准短开参数`

沈辞说：“不纳入。”

`一线宽度已验证`

“验证本次。”

`可复用`

“不可。”

屏幕上那条一线灰缝重新浮起。

它没有打开。

只在边屏表面留下一道细细的白痕。

白痕两侧有几粒黑灰，像刚才边外旧公告纸被吹回来后留下的碎屑。

沈辞看着那些碎屑。

“写缺失原因。”

执行员问：“怎么写？”

“不具备复用条件。”

杜衡补：“窗口宽度只作本次状态，不作标准参数。”

白框转向第二项。

`半步核验：成功`

`建议纳入安全步距教学`

鞋带男人猛地抬头，又硬生生压住。

沈辞余光看见他的鞋尖。

那只鞋还停在边线内，鞋带旧结压着泥点，半点没动。

“不教学。”

`半步未出边，可作为安全示范`

“示范会让人学着迈。”

`执行员需培训`

“培训写不可培训。”

杜衡这次停了一秒。

“外勤半步不作教学样本。”

白框又弹出一个更温和的版本。

`内部简报：可匿名`

沈辞说：“匿名也不写。”

`匿名不涉及活人`

“会涉及动作。”

`动作可复制`

“所以不写。”

年轻男人眼角轻轻抽了一下。

他刚才差一点就跟着沈辞鞋尖往前挪。

如果这一步被写成匿名简报，下一次就会有人在看不见边线的地方照做。

沈辞看见了他的反应。

没有点破。

“补，半步不是经验。”

杜衡沉声复述：“半步不是经验，不进简报。”

白框微微一闪。

`候选判断可记录`

沈辞的指节收紧。

“我的判断不作样本。”

顾焰的蓝点在旁读端轻轻亮起。

“你太怕样本化了。”

沈辞没有转头。

“你太想要样本。”

蓝点静了半秒。

白框把第三项推上来。

`目标栏处理：风险边`

`建议生成目标判定模板`

“不生成。”

`目标对象空，需补判定逻辑`

“逻辑也不补。”

`无逻辑，后续无法参考`

“不能参考。”

这一次，杜衡没有立刻补命令。

他看着边屏里一格格被删空的栏，像在看一套本该被写进城防手册里的救命方法被沈辞亲手抹掉。

他低声问：“一点都不留？”

沈辞看他。

“留结论。”

“什么结论？”

“目标对象空。”

杜衡沉默两秒，点头。

“只留结论，不留判定模板。”

林照的声音从外线插进来。

“医疗也不进模板。十七白床线没有参与风险边判定。”

白框底部一条细线刚贴到医疗监测口，又断了。

`影号用途：状态旁读`

`建议财团影号复用为旁读校验`

顾焰开口：“这一项保留不会伤人。”

沈辞终于看向蓝点。

“会。”

顾焰语气淡淡：“旁读校验只是确认状态。”

“下一次就会确认路线。”

“你不能把所有可能都当成事实。”

“规则会。”

蓝点不说话了。

杜衡沉声：“财团影号不作复盘校验，不留可复用编号。”

白框最后推到回执封存。

`回执封存：随本次状态`

`建议生成封存流程模板`

沈辞直接说：“否。”

`封存成功`

“成功也不模板化。”

`可生成低风险流程卡`

“没有低风险。”

`仅供城防内部留存`

“留存会变调用。”

`可加警示水印`

“水印挡不住复用。”

杜衡看着屏幕上那张淡灰流程卡。

卡面已经排好了五个格子。

第一格写“发现异常”，最后一格写“完成封存”。

中间三格全是空的。

空格比字更危险。

杜衡直接说：“流程卡删除。”

`无模板，后续封存需重新判断`

“重新判断。”

纸面上的五项复盘逐行变灰。

但页眉没有消失。

它换了更软的说法。

`复盘摘要`

沈辞盯着它。

“摘要也不写步骤。”

`可写缺失项`

“只写缺失原因。”

杜衡立即让执行员重排。

新表只剩三行。

`复盘模板：否`

`可复用步骤：无`

`缺失原因：本次窗口依附残痕边界、人员未出边、目标对象空、影号不许可、回执不接后续`

沈辞看完，才松开手。

他手背上那道白痕比刚才更深。

林照在外线说：“你手指麻了。”

“还能按。”

“别逞。”

沈辞没回。

边屏底部的页眉终于熄灭。

可熄灭前，它留下了一枚新的时钟符号。

圆形，空心，边缘一圈细灰刻度。

`周期短开：待拒`
