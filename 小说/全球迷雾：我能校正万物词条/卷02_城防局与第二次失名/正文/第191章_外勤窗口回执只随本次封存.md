# 第191章 外勤窗口回执只随本次封存

`外勤窗口回执：待封存`

灰角卡住的时候，边屏没有立刻归档。

它像一小片没被剪干净的指甲，压在总表边屏和临时封存车记录之间。

每闪一次，沈辞临时证背面的监测线就轻轻一紧。

白框先吐出最稳妥的字。

`封存人：待填`

沈辞没有抬头。

“空。”

`封存人空，记录不可追溯`

“追溯本次状态。”

`建议封存单位：城防现场组`

杜衡站在边屏侧面，手里的记录笔停了一下。

他比沈辞更清楚这几个字的分量。

一旦写了单位，下一次窗口短开就会有回找路径。

回找路径会变成复盘路径。

复盘路径再往下，就是标准流程。

杜衡沉声道：“封存单位不写城防现场组。”

白框换得很快。

`封存单位：临时封存车`

沈辞看向灰角。

封存车侧面那道旧漆裂缝微微发亮，像有人用指尖从里侧顶了一下。

“车也不作单位。”

`载体在场`

“载体不是单位。”

执行员咽了一下。

三名待核者站在边线内侧，没人说话。

布包女人把布带往掌心里卷了一圈，动作很慢，像怕多动一毫米也会被边屏写进去。

白框停了一秒。

`封存说明：窗口短开完成，现场人员确认`

“删现场人员。”

`现场人员可证明未离边`

“前一章已经写过。”

`可补封存责任：现场共同承担`

沈辞抬眼。

“不共同承担。”

`共同承担可避免单人追责`

“会把所有在场人写进回执。”

白框把“所有在场人”四个字吞掉，改成更小的一行。

`责任范围：现场范围内`

“范围不是责任。”

杜衡这一次接得很快：“现场范围只作物理位置，不作责任范围。”

执行员把这句话敲下去时，三名待核者明显松了一口气。

那口气很轻。

但边屏听见了。

`在场反应可作确认`

沈辞声音压低：“呼吸也不作确认。”

布包女人的手停住，连布带都不敢再卷。

沈辞按住灰角。

纸角冷。

冷意顺着指腹往上爬，爬到腕骨处时，临时证轻轻震了一下。

`建议关联：短开回执 / 外勤窗口 / 现场见证`

“只关联本次窗口状态。”

`状态需要封存说明`

“说明写状态，不写人。”

白框像是不满意这个答案，灰角边缘往外翘起半分。

一行更细的字贴着翘起处浮出。

`封存依据：沈辞按压确认`

沈辞眼神一冷。

“按压不是确认。”

`按压动作已记录`

“动作只证明阻止扩写。”

杜衡立刻接上：“记录为阻止扩写，不作封存确认。”

顾焰的蓝点在财团旁读端亮了一下。

“如果没人确认，封存价值会降低。”

沈辞看着蓝点。

“降低就降低。”

顾焰轻笑了一声。

“你现在宁愿让记录不漂亮，也不让它有用。”

“不是所有有用的东西都该留下。”

蓝点安静了半拍。

林照的外线声音插进来，短而硬。

“医疗口不作封存安全说明。十七白床线没有参与窗口，不在回执里。”

白框原本正往医疗口旁边拉一条细线，听见这句后，那条线断在半空。

断口像被烟头烫过，灰了一圈。

`安全说明：无医疗异常`

林照说：“删。”

`未见伤害`

“不等于安全。”

沈辞没看她，却把手指又压低了一点。

这句话他说过。

林照也说过。

但这一次，白框没有继续用安全绕路。

它转向三名待核者。

`见证对象：边内待核三名`

三人同时僵住。

鞋带男人的脚尖往后缩，差点碰到身后灰线。

沈辞先一步开口。

“不见证。”

`其在场位置可证`

“位置不作见证。”

杜衡补：“三名待核者只作边内待核位置事实，不参与回执封存。”

白框终于把见证对象压空。

灰角却没有暗下去。

它在封存车记录边缘蹭出一条细白痕。

`建议生成封存闭环`

“不闭环。”

`未闭环记录易被重提`

“可重提状态，不可重提人。”

`可补闭环方式：系统代签`

沈辞指尖一僵。

白框立刻把那四个字贴到灰角背面，像替所有空栏找到了最干净的答案。

`系统代签无需活人参与`

“代签也不行。”

`无活人责任`

“会有系统责任口。”

杜衡看了一眼执行员。

执行员额头已经出汗。

杜衡道：“系统不代签，流程不自签，回执不闭环。”

`建议留复盘口`

沈辞指尖一顿。

这才是它真正想要的东西。

封存人可以换。

封存单位可以换。

安全说明可以换。

只要留下复盘口，回执就能在下一次被重新展开。

“复盘口，否。”

白框变白了一层。

`无复盘，不利于下次处置`

“没有下次。”

`边外风险仍未消`

“风险未消，不等于窗口可继承。”

杜衡这一次没有等沈辞再提醒。

他直接命令执行员：“拆封存栏。”

边屏被切成四格。

`本次窗口状态`

`封存人员 / 单位`

`安全说明 / 见证`

`后续复盘口`

执行员的指尖发抖，却还是一项一项敲下去。

`本次窗口状态：短开完成 / 路线未生成 / 人员未出边`

`封存人员 / 单位：空`

`安全说明 / 见证：空`

`后续复盘口：否`

灰角缩回去一半。

沈辞没有松手。

“补一句。”

杜衡看他。

沈辞说：“只随本次状态封存。”

白框停住。

这一次，它没有立刻反驳。

封存车侧面的旧漆裂缝也暗了下去。

记录边上，最终落下一行小字。

`外勤窗口回执：随本次状态封存`

灰角被压平。

可边屏底部，另一个空白框慢慢浮出来。

`外勤复盘：不得生成`
