# 第144章 监测半径不随任务走

`请选择任务原点`

圆心在屏幕上晃了三次。

第一次落到沈辞胸前的临时证。

证角立刻发白。

第二次落到安置区总表。

残痕分栏同时抖动。

第三次落向医疗点外线。

林照那边的仪器声猛地一尖。

“白床线被扫到。”

沈辞没动。

他知道动一步，圆心就会跟一步。

任务要把监测半径改成能调度他的绳。

绳一旦套到证上，他走到哪，任务就能派到哪。

套到总表上，他就会被总表拖住。

套到医疗点，知微会被连进任务原点。

哪一种都不对。

杜衡问：“原点能不能选人？”

“不能。”

沈辞声音发哑。

“也不能选表。”

技术员急得额头冒汗。

“不选，重算会超时。”

总表已经开始倒计时。

`重算剩余：十`

`九`

每跳一次，沈辞胸口就像被绳勒紧一点。

顾焰的蓝点在旁边亮起。

“选短指令边界。”

杜衡看向他。

“什么意思？”

“不用人当原点，用边界当原点。”

顾焰停了一下。

“理论上能绕过任务接管。”

“理论上？”

“失败的话，半径会直接挂到复核口。”

杜衡骂了一个很短的字。

沈辞没有骂。

他看见了另一条更窄的路。

第140章留下的残痕分栏，不是人，也不是表。

它是边。

腕条旧痕在老人身上。

物资袋旧痕在孩子身边。

岗位旧痕在隔离岗胸牌上。

医疗底痕在白床线里。

这些点不能合成一个人。

但能合成一圈不能越过的边界。

`八`

`七`

沈辞抬手。

杜衡立刻挡住技术员。

“让他来。”

不是信任。

是没有别的手。

沈辞把手指按到残痕分栏四角。

不按中间。

中间会变原点。

四角只定边。

他先读腕条旧血点。

老人站在原地，腕条没有离身。

再读物资袋齿痕。

孩子没有报名字，袋子仍在女人手里。

再读胸牌裂痕。

隔离岗的人还站在隔离岗。

最后读医疗底痕。

白床线在医疗点，没有进亲属栏，也没有进任务单。

四个点被读成四个角。

圆心没有落下。

倒计时停在`三`。

总表开始改字。

`任务原点未选`

`边界可用`

沈辞眼前一黑。

他没有听见后半句。

只看见屏幕上的圆被拉成了一个很窄的灰框。

灰框刚成形，蓝边就伸出一根细针。

`边界需责任中心`

针尖先指沈辞。

临时证边角又白了一层。

沈辞把手收回袖口。

针尖转向杜衡。

杜衡胸前的城防执勤牌跳出一行：

`现场指挥可代为中心`

杜衡没有答。

他如果答，刚才没交出去的调度权会从另一扇门回到他身上。

城防指挥一旦成中心，财团旁读就能说现场边界由城防签发，再要求复核城防签发过程。

针尖最后滑向医疗点外线。

林照那边仪器声又尖了一下。

“别让它落我这里。”

医疗点一旦成责任中心，白床线就不再是底痕。

会变成任务保障对象。

保障对象后面，最容易接亲属。

沈辞咬住舌尖。

疼意让他重新听见声音。

“边界不设中心。”

总表回：

`无中心，边界不稳定`

沈辞看着四角。

“残痕自己就是中心。”

这句话说出来，他耳后旧伤猛地一跳。

不是因为它真对。

而是因为它很危险。

旧痕如果被承认为中心，就可能被系统抽走当核心样本。

所以他又补了一句。

“只在本次短指令内有效。”

灰框边缘终于停住。

框套住安置区总表边屏。

也贴着医疗点外线。

但没有套住沈辞本人。

更没有套住沈知微。

林照那边的仪器声慢慢降回低频。

“白床线退回监测口。”

杜衡立刻下令。

“记录：监测半径按残痕边界，不按任务原点。”

技术员照录。

总表却弹出红字。

`任务执行范围不足`

顾焰说：“它要外勤窗口。”

杜衡冷笑。

“刚稳住半径，又要把人派出去？”

顾焰没有避。

“总表里的错名不是只在这间屋里。你们不出窗口，后面的人进不来。”

这句话又是对的。

安置区外还有排队登记的人。

还有转运口。

还有救援编号没核完的临时床区。

残痕分栏能阻止清零。

不能替所有人完成核验。

外面正好传来一阵压低的骚动。

隔离帘外有人问能不能进。

没人敢报名字。

只敢说“腕条快掉了”“床号被换了”“孩子袋子找不到格”。

每一句都像被总表咬住半边。

屏幕边缘同步跳出待办数。

`未入场旧痕：十七`

`转送等待：六`

`无监护候补：三`

数字没有大到像灾难。

却够把刚稳住的边挤裂。

顾焰说：“十分钟也许不够。”

沈辞说：“够开口。”

“不够核完。”

“没说核完。”

他现在最清楚一件事。

任何“核完”都会变成完整流程。

完整流程就是陷阱。

他们只能让外面的人先进边，保住旧痕不被当场清掉。

后面再一格一格验。

慢。

但活。

沈辞看向灰框边缘。

如果出窗口，任务就会继续想把他写成执行人。

如果不出，外面的人会被干净表重新吞掉。

他把舌尖抵住上颚。

血味还在。

“窗口可以开。”

杜衡皱眉。

沈辞接着说：“不以我为原点。”

总表边缘亮起。

像等的就是这句。

沈辞没有给它机会。

“以残痕边界为原点，只开十分钟。”

杜衡补令。

“外勤窗口十分钟，只核物痕，不收姓名，不接完整任务。”

技术员敲下最后一项。

屏幕灰框外侧裂开一条很窄的口。

`边线外勤窗口：十分钟`

`核验范围：旧痕 / 位置 / 物`

沈辞胸前临时证没有再发热。

这说明它暂时没能把窗口写成他的私人任务。

但总表很快送来下一份东西。

不是任务单。

是一张复核名单。

名单上没有姓名。

只有一排排被擦到发白的格。

`复核名单先到场`

第一格下面浮出：

`待核本人：三名`

第二格更淡。

`其中一人已被空表预填`

安置区外传来推床轮子压过地面的声音。

杜衡抬头。

“人到了。”
