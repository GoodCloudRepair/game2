# 第149章 冲突仲裁不裁本人

`请裁定本人`

仲裁框一出现，三名待核者身边的旧痕同时发白。

袖章烧边被拉直。

布包药粉底痕被抹平。

断鞋带黑灰被压成一条细线。

仲裁要的不是流程。

是结果。

它想绕过前面所有“可分”，直接问谁是谁。

中年男人咬着牙，没有说话。

年轻女人抱着布包，眼神往沈辞这边看。

少年低头盯着自己的鞋带，肩膀一点点绷紧。

他们都知道不能报名字。

但被一个框问“本人”时，沉默也像否认。

仲裁框马上亮：

`本人拒答`

`拒答对象可转无名`

杜衡冷声道：“闭名不是拒答。”

总表回：

`无名状态持续`

顾焰说：“仲裁需要一个判定。”

沈辞看向蓝点。

“裁什么？”

顾焰没有立刻答。

沈辞替他说出来。

“裁活人，还是裁流程？”

仲裁框像被这句话刺了一下。

`裁定范围：本人 / 任务 / 单位 / 回执`

四个选项并排。

第一个最亮。

本人。

沈辞伸手，按住后面三个。

任务。

单位。

回执。

不碰本人。

`本人项未选`

屏幕马上红。

沈辞说：“冲突仲裁不裁本人。”

`无本人裁定，仲裁不足`

“本人不属于你们的冲突。”

仲裁框周围的蓝白线同时收紧。

城防封条和复核蓝边都被勒出细纹。

上方弹出：

`现场与复核均要求本人结果`

杜衡冷声道：“城防要求的是留场核验，不是本人结果。”

蓝点那边，顾焰停了两息。

“复核要求的是样本一致性，不是本人裁定。”

仲裁框没有放过他。

`样本一致性需对应本人`

顾焰说：“对应可以延后。”

这句话让复核蓝边暗了一下。

沈辞看见顾焰那边的黑箱投影像被压低半寸。

他不是突然善心。

他也在让出一点权限，换旁读还能继续留在场。

这才符合顾焰。

沈辞没有谢他。

谢意也会被表格写成关系。

他只说：“仲裁流程。”

`请提交流程冲突项`

这才是口。

不是裁人。

是裁哪条流程不该吞掉人。

这句话落下，亲属登记室里安静得只剩仪器低响。

城防和财团可以争任务。

争单位。

争回执。

但三名活人不是他们争出来的答案。

仲裁框一时没能弹回。

沈辞趁这一息，把三条旧痕重新压回各自边上。

袖章旧痕旁边，忽然多出一小截旧红线。

沈辞眯了一下眼。

那不是第145章出现的线。

是更早的名单线。

安置区岗位名表里，有一栏曾被擦到只剩半个“巡”。

中年男人的袖章烧边，正好能接住那半个字的旧针脚。

不是名字。

是位置。

`旧巡逻袖章 / 岗位旧痕反向对应`

总表弹出这行时，中年男人猛地抬头。

他差点开口。

沈辞先说：“别报。”

男人硬生生把话咽回去。

仲裁框立刻想把这条对应拖进本人项。

`岗位对应可确认本人`

沈辞压住“本人”。

“只能确认他不是另外两个人。”

`否定确认不足`

“够留场。”

仲裁框继续逼近。

`排除不构成本人权利`

“但构成转送风险。”

沈辞把袖章烧边压到转送章上方。

旧针脚抵住章角。

`转送章适用对象不明`

这行字一出来，中年男人的肩膀明显松了一点。

适用对象不明，就不能立刻转送。

不能确认他是谁。

至少能确认这张转送章不能随便贴到他身上。

杜衡跟上。

“记录：岗位旧痕反向对应，仅作排除，不作本人裁定。”

技术员敲下去。

仲裁框震了一下。

第一行`本人`暗了半格。

年轻女人的布包底痕也开始起变化。

药粉白痕下，有一小块旧蓝纸屑。

林照在外线忽然说：“那种蓝纸是安置区早发药袋内衬。”

顾焰问：“你确定？”

“我拆过。”

林照声音冷。

“但我不签医疗项。”

沈辞点住蓝纸屑。

`药袋内衬 / 物资旧痕反向对应`

女人眼眶红了。

她仍没有说给谁拿药。

这次她学会了。

只说：“纸屑在包底。”

仲裁框第二行也暗了半格。

物资旧痕刚稳，亲属栏就又想伸手。

`取药行为可关联亲属`

沈辞没有让它接上。

“物资对应，不问给谁。”

林照跟着补：“不作亲属用途。”

布包底缝里的蓝纸屑被压回物资栏。

亲属栏那条细线没能搭上。

女人这次没有低头。

她盯着自己的布包底缝，像终于知道该看哪里。

少年那边最慢。

鞋带黑灰没有对应安置区任何名单。

推床黄灰却和转运等待栏边的一道泥印重合。

危险在这里。

如果读推床，他会被写成转运对象。

沈辞不读推床。

他只读鞋带断口上的黑灰。

黑灰里有极细的红砖粉。

安置区入口外墙是灰水泥。

红砖粉来自旧体育馆后门临时坡道。

那条坡道只给临时补入的人走。

`后门坡道旧灰 / 入场旧痕反向对应`

少年终于抬眼。

他没有说谢谢。

只是把脚往鞋带旁边挪了一点。

仲裁框把三条反向对应重新排了一遍。

岗位旧痕。

物资旧痕。

入场旧痕。

三条都不是姓名。

但三条都能证明一件事。

这三个人不是同一个`周岷`。

也不是可以合批转送的无名对象。

总表还想补：

`可转后续确认`

沈辞按住“转”字。

“后续确认可以，后续转送不行。”

杜衡跟上：“留场后续，不出边线。”

顾焰补得很冷。

“旁读继续，不调原物。”

三边再次把转送口压窄。

少年脚边的推床轮子轻轻晃了一下，没有再转。

中年男人的袖章还在臂上。

年轻女人的布包还在掌心。

少年的鞋带还在脚边。

三件物都没有被送出边。

边线也还没完全闭死。

三条反向对应全部成立。

仲裁框却没有消失。

它把三条记录并排，底部补出：

`流程冲突成立`

`本人裁定暂缓`

沈辞松开手时，指尖发冷。

这不是赢。

只是把仲裁对象从人身上挪回流程。

屏幕最下方又浮出一行：

`边线协助权限：待确认`

`候选人可否接入`
