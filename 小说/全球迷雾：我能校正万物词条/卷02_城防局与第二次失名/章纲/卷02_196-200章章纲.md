# 卷02_196-200章章纲

## 使用说明

- 对应卷：卷02 城防局与第二次失名
- 对应阶段：阶段03 / 任务调度与改名边缘
- 当前目标：承接第195章`总表外沿巡检：不得自动化 / 待人工旁观`，允许总表外沿只作本次人工旁观，但不让它变成自动巡检、固定班次、周期短开同义项、路线预览或新外勤入口。
- 章内约束：不脱离监测半径；不转新大型场景；不让沈辞成为人工旁观人、巡检责任人或固定班次对象；不让三名待核者作旁观见证；不让顾焰拿到巡检主导权。

## 段落总钩

- 起点：封存号不接调度后，总表边屏底部生成外沿巡检灰栏，标注`不得自动化 / 待人工旁观`。
- 预定落点：第200章前后确认人工旁观只可本次只读，不接路线、不设班次、不写责任人，也不能把边外灰栏转成城市巡检口。
- 后续方向：可转入卷02后段更强的城市级管理名压力，例如`城市外沿灰栏：待上报`或`人工旁观回执：不得找人签`。

- 状态：待写。

- 地点：安置区总表边屏 / 总表外沿巡检灰栏 / 边线外勤窗口旧槽
- 必须接钩：`总表外沿巡检：不得自动化 / 待人工旁观`亮起，系统试图把人工旁观写成巡检事项和旁观人。

- 建议章末钩：`人工旁观人：不得写沈辞`。
- 禁触：不让沈辞成为巡检员、旁观人或责任人。
