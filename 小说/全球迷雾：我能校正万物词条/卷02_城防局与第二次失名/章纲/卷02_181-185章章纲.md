# 卷02_181-185章章纲

## 使用说明

- 对应卷：卷02 城防局与第二次失名
- 对应阶段：阶段03 / 任务调度与改名边缘
- 当前目标：已承接第180章`监测半径：待重算`，把监测半径、边线协助续签、离场步距、十七白床锚点和固定驻点五层压窄，防止沈辞被固定成任务原点或正式调度单位。
- 章内约束：不让沈辞脱离监管半径；不让沈辞成为固定任务原点；不让知微白床线作半径锚点；不让边线协助升级为正式调度权；不新增大型场景或外勤副本。

## 段落总钩

- 起点：复核名单二次回填被压住后，安置区总表边屏改从“名单归属”转向“监测半径重算”。
- 已落正文：第181-185章已确认沈辞仍可短时边线协助，但半径不跟任务走、不借白床线、不写固定驻点。
- 后续方向：已生成`边线外勤窗口：待短开`，让主角有短距离行动空间但不获得自由。

## 第181章 预定方向：监测半径重算不改原点

- 状态：已写。
- 时间：D2 17:50-18:00
- 地点：安置区总表边屏 / 沈辞脚下监测边界 / 临时证半径线
- 必须接钩：`监测半径：待重算`绕着沈辞脚下边界收紧，试图把半径原点从安置区边线改成沈辞本人。
- 核心冲突：若原点改成沈辞，后续任务、名单和复核都会围着他转，他会变成固定调度原点。
- 建议关键新信息：监测半径可重算边界，但不改原点；半径原点仍归安置区边线和临时监管，不归沈辞本人。
- 建议章末钩：`边线协助权限：待续签`。
- 禁触：不让沈辞脱离监管，也不让他成为原点。
- 正文落点：监测半径重算不改原点，原点归安置区临时监管边，不归沈辞本人，任务随动不设。

## 第182章 预定方向：边线协助权限不作正式续签

- 状态：已写。
- 核心冲突：边线协助权限到时限后，系统试图把续签写成正式协助证或准调度权。
- 建议关键新信息：边线协助只能按本次残痕分栏短续，不作正式续签、不作常驻权限。
- 建议章末钩：`离场步距：待核`。
- 禁触：不让沈辞升级成正式城防协助员。
- 正文落点：边线协助只作本次短续，不作正式协助证、准调度权、自动续签、副本或待核者见证。

## 第183章 预定方向：离场步距不归任务路线

- 状态：已写。
- 核心冲突：离场步距被任务路线读取，试图把沈辞移动的每一步变成执行轨迹。
- 建议关键新信息：步距只作监测安全边，不作任务路线、不作执行轨迹、不作带队依据。
- 建议章末钩：`十七白床锚点：待拒`。
- 禁触：不转新地图，不让沈辞直接离开安置区主场。
- 正文落点：离场步距只作监测安全边，不作任务路线、执行轨迹、脚印、放行依据或下一步建议。

## 第184章 预定方向：十七白床不作半径锚点

- 状态：已写。
- 核心冲突：系统尝试用十七白床线稳定沈辞半径，等于把知微拖成半径锚点。
- 建议关键新信息：白床线只作医疗监测，不作半径锚点、不作稳定参照、不作调度安全证明。
- 建议章末钩：`固定驻点：待清栏`。
- 禁触：不让知微清醒，不让兄妹关系进入亲属口或任务口。
- 正文落点：十七白床只作医疗监测，不作半径锚点、稳定参照、误差校验、调度安全证明、家属提示或待核关系。

## 第185章 预定方向：固定驻点不写沈辞

- 状态：已写。
- 核心冲突：半径重算后，固定驻点栏试图写入沈辞，方便后续调度和复核回找。
- 建议关键新信息：固定驻点只能写安置区边线或封存车位置，不写沈辞本人；沈辞不是驻点、单位或任务原点。
- 建议章末钩：可生成`外勤半步：待核`或`边线外勤窗口：待短开`。
- 禁触：不让沈辞获得自由外勤，也不让他被钉死成驻点。
- 正文落点：固定驻点只写安置区边线和临时封存车位置，不写沈辞、临时证、协助口、待核对象或回访联系人；章末生成`边线外勤窗口：待短开`。
