# 卷02_121-130章章纲

## 使用说明

- 对应卷：卷02 城防局与第二次失名
- 对应阶段：阶段01 / 入档与编制失名
- 当前目标：承接第120章临时证和`编目师候选`，把HL-02落到现实管理体系。
- 章内约束：单章最多1个关键新信息；前五章不稳定改名、不揭编目庭、不让知微康复。

## 第121章 临时证不准过夜

- 章节功能：卷02开篇，正式把沈辞从封门幸存者写成被监管借用对象。
- 核心冲突：临时证可以证明他有用，却也会把他锁进更深系统。
- 主要剧情：城防入档口夜间复核临时证，正面能读，背面`编目师候选`让设备卡死；证背不是被盖章，而是自行渗出`不得离开监测半径`。
- 关键信息：限制来自更深系统，城防局只是执行层。
- 章末钩子：评估传票落下：`封门余留活项 / 反读风险评估 / 立即入档`。

## 第122章 评估不测能力只测反读

- 章节功能：把城防局对沈辞的态度写成“需要但害怕”。
- 核心冲突：沈辞以为是能力测试，实际被测试的是会不会污染调度系统和被门反读。
- 主要剧情：评估室不让他校正任何东西，只让他靠近空白档、封门残线和旧名样本；设备反复测试他的自名缺口。
- 关键信息：沈辞不是普通工具，而是不稳定变量。
- 章末钩子：评估设备吐出残段：`HL-02`。

## 第123章 候选字不归城防印

- 章节功能：身份线抬层，财团线重新进场。
- 核心冲突：杜衡要把候选痕迹压在城防内部，顾焰却能用黑箱看见一部分但无法写入。
- 主要剧情：杜衡、顾焰、沈辞短对峙；城防印盖不上候选字，黑箱能扫到灰痕却无法复刻；林照在外线挡住无关登记。
- 关键信息：`编目师候选`不属于城防，也不属于财团。
- 章末钩子：顾焰丢出半页旧实验附注，提示文修回流件存在同类手法。

## 第124章 灰痕先认第二新号

- 章节功能：把知微后遗症接入HL-02。
- 核心冲突：沈辞想留在医疗点，任务调度却要求他入档；知微灰痕对第二新号产生远距轻应。
- 主要剧情：知微不回应完整称呼，只在灰痕发热时按沈辞手腕两下；监测针同步抖动，显示HL-02不是只在监测缝里。
- 关键信息：知微可能成为第二轮失名的共鸣敏感点。
- 章末钩子：城防安置区名单出现第二新号反应。

## 第125章 同名替换先改编制表

- 章节功能：卷02前段第一次小高潮。
- 核心冲突：第二次失名不是怪潮乱冲，而是改编制名、床位名和亲属登记名。
- 主要剧情：调度屏岗位名漂移，有人应了错名被拖进缝影；沈辞用识名只稳住一格，耳后出血，自名缺口再疼。
- 关键信息：第二次失名攻击被管理后的名字。
- 章末钩子：错名表出现`文修前声`同类旧红章。

## 第126章 安置名单不收本人签

- 章节功能：进入安置区同名替换段。
- 核心冲突：本人签名无法证明本人，反而会把错误编制固定。
- 主要剧情：沈辞随杜衡进入安置区，发现幸存者签名栏不断自动补全相同尾字。
- 关键信息：HL-02会利用安置手续固化错名。
- 章末钩子：一名幸存者的亲属栏写出已经死亡的人。

## 第127章 床位号先替亲属答

- 章节功能：放大第二次失名对生活秩序的侵入。
- 核心冲突：亲属关系被床位号先行替答，活人无法纠正。
- 主要剧情：沈辞识名发现床位牌比人更早回应，林照用医疗记录压住一处错答。
- 关键信息：床位名成为第二次失名载体。
- 章末钩子：知微白床监测线短暂与安置区床号同步。

## 第128章 复核通道截走半页报告

### 定位

- 章节功能：财团线继续插入，把“复核通道已截取”从红字变成现场动作。
- POV：沈辞近景限知。
- 时间：D2夜间，紧接第127章床位号被压回旧器材牌之后。
- 地点：海陵城防安置区三排十七床附近 / 临时归档桌 / 复核通道终端。
- 目标字数：2000-2200。

### 六问

- 推进哪一层主线：财团线与第二次失名线交叉，确认财团对HL-02有独立信息口。
- 核心冲突：城防局还没来得及把床位号异常归档，财团复核通道已经先截走半页报告。
- 结束后局面变化：沈辞的识名结果不再只被城防使用，财团能抢先拿到错名格残页。
- 读者新认知（对谁/什么）：顾焰不必进现场，也能通过复核通道更快接触第二次失名材料。
- 唯一关键新信息：财团复核通道比城防归档更快。
- 章末钩子：被截回的半页残影上出现父亲式旧标注，但仍不是完整签码。

### 节拍（直接可扩写）

- 开场状态：接第127章末，床头牌背面红章渗出`复核通道已截取`。
- 事件一：杜衡命令封存床位牌和识名记录，归档纸刚压住，报告右半页被红线从夹层里抽走。
- 事件二：城防终端追不上截取路径，只能显示联席复核占用；顾焰不在现场，却通过黑箱投来半页错名格影子。
- 事件三：沈辞尝试读回被截走的边缘，发现截走的不是完整报告，而是自己识名后留下的“错名格空白”。
- 结尾结果：半页残影短暂回贴到床位牌背面，露出父亲式旧标注的笔压和半句警告。

### 代价 / 规则触点

- 本章动用的能力或手续：沈辞识名追读报告边缘；城防归档封存；财团联席复核截取。
- 可见代价：耳后旧伤再渗血，识名越贴近被截半页，自名缺口越疼。
- 规则边界（不能写成什么）：不能写成沈辞夺回报告或破解财团通道；只能确认通道更快、材料被抢先看见。

### 禁触（3–5条）

1. 不让沈辞稳定改名或补全自名。
2. 不揭编目庭本体。
3. 不让知微清醒并主动参战。
4. 不露出父亲完整签名、完整签码或完整立场。
5. 不把财团写成全知，只写“更快截取半页”。

### 衔接

- 必须接住的上章钩：床头牌背面红章旁的`复核通道已截取`。
- 必须留给下章的钩：半页残影出现父亲式旧标注，转入第129章文修前声同源手法。

### 情绪 / 节奏一句

- 冷压手续感里插入“纸被抽走”的刺感，少解释，多写终端、夹层、红线、半页残影。

### 回填标记

- 04关系：有→沈辞/顾焰：从现场对峙转为信息口争夺。
- 05物品能力：有→复核通道半页报告 / 错名格空白。
- 06伏笔：触动→父亲式旧标注只露笔压和半句，不给完整签码。
- 14-20：有→14财团复核通道权限更快；16安置区三排十七床成为首个被截报告点。

## 第129章 文修前声不只在旧带里

### 定位

- 章节功能：父亲线与文修线再抬，把第128章半页旧标注落到可观察的材料手法。
- POV：沈辞近景限知。
- 时间：D2 09:10-09:20。
- 地点：海陵城防安置区三排尽头 / 文修核验口。
- 目标字数：2000-2200。

### 六问

- 推进哪一层主线：文修线、父亲旧研究线、财团解释权线。
- 核心冲突：文修核验口要核材料手法，城防想封存，财团想先定义来源，沈辞只能用职业经验抢在两边之前读清同源痕迹。
- 结束后局面变化：`文修前声`从单个旧带线索升级成一类旧流程材料。
- 读者新认知（对谁/什么）：沈辞的文修职业经验在官方体系里仍不可替代。
- 唯一关键新信息：`文修前声`不是单个旧带现象，而是一类旧流程材料。
- 章末钩子：半个旧签码被人为刮掉。

### 节拍（直接可扩写）

- 开场状态：承接第128章末，物资暂存门褪成`文修核验口`。
- 事件一：杜衡要封门，门口却只收“材料手法”，不收城防封条和财团投影。
- 事件二：沈辞用旧纸压、封口胶、磁带壳压纹辨出旧红章、半页残影和文修前声封套同源。
- 事件三：顾焰远端试图给出财团编号，核验口退回，沈辞只读材料层不读签名层，避免被父亲线拖深。
- 结尾结果：核验口吐出半格旧签码，右半被人为刮掉。

- 可见代价：沈辞耳后血未止，越靠近旧签码越出现自名缺口刺痛。
- 规则边界（不能写成什么）：不能写成父亲签名确认；不能让财团或城防直接给答案。

2. 不让顾焰解释完整真相。

4. 不让文修核验口变成大型新门域。
5. 不让知微清醒参战。

## 第130章 旧签码不露完整名

- 章节功能：完成121-130小段收束，给父亲线硬一点的证据，但不揭底。
- POV：沈辞近景限知。
- 时间：D2 09:20-09:30。
- 地点：文修核验口 / 安置区临时归档桌。
- 目标字数：2000-2200。

- 推进哪一层主线：父亲旧线、财团/城防封存争夺、安置区同名替换段收束。
- 核心冲突：沈辞想追完整签码，城防要封存，财团要复刻，旧签码本身却拒绝露完整名。

- 读者新认知（对谁/什么）：沈父旧痕可能曾接触这类文修流程，但仍不能断言立场。
