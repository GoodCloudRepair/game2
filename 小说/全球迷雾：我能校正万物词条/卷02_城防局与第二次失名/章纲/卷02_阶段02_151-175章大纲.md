# 卷02_阶段02_151-175章大纲

## 一、阶段定位

- 阶段名：文修回流与财团半门
- 章节范围：151-175
- 时间：D2中午后，承接第150章`二次背注：待复核`
- 场景重心：安置区总表边线、临时证复核口、文修回流件封存车、财团旁读半门样本、城防局临时复核席

本阶段从“谁能调度沈辞”转向“谁能解释沈辞”。二次背注不是奖励，也不是正式升权，而是临时证在多方流程中留下的复核疤痕。城防要把它纳入可管流程，财团要借它证明候选来源，沈辞则必须继续只认材料手法，不认完整来源。

## 二、阶段起点状态

- 第150章边线协助临时成立，但只限旧痕可分、边内待核、回执挂口和流程冲突。
- 三名待核者可留场，无名转送暂缓。
- 沈辞临时证出现`二次背注：待复核`。
- 财团仍只有旁读序列影号，未拿到完整样本。
- 文修核验口、半格旧签码、半页旧实验附注仍未给完整来源。

## 三、阶段终点状态

- `二次背注`被压成来源待核，不能写本人、城防或财团完整来源。
- 文修回流件重新入场，但只许封袋旁读，不许离袋、拆封或补签。
- 财团证明自己掌握父亲线一角，却不能证明目的、立场或完整签码。
- 沈辞进一步确认自己和文修旧流程有关，但仍不能把猜测补成父亲真相。

## 四、三段推进

### 1. 151-160：二次背注与回流件入场

先处理临时证背注来源复核，再把背注牵向文修回流件。重点是：背注不写本人，传唤不叫候选人，回流件不离封袋，半门样本不全开，旧红章不盖临时证。

- 当前完成：第151-175章已落正文；阶段已从二次背注、文修回流、半门试开推进到父线旧页方法边界，并在第175章落成`方法边界：待调度复核`。

### 2. 161-170：财团半门试开

财团以半门样本和旁读协议施压，试图证明候选痕可复制、可归档、可训练。沈辞只能从材料手法、封袋磨损和旧章刮痕中判断“这条路有人提前堵过”，不能给完整解释。

- 当前完成：第161-175章已落正文；半门试开和父线旧页已被压成`旁签人空 / 试开宽度一线半 / 父线立场未证 / 记录随袋 / 候选痕不可训练 / 失败项不可完成 / 缺项原袋退回 / 旧项目首字不补 / 旧页脚不读完整旧页 / 页码不索引 / 裁切不补人 / 缺项格式不作签码位 / 方法边界不转任务口`。

### 3. 171-175：父亲线一角与阶段转向

阶段后段给出更硬但不完整的父亲旧研究痕迹，顾焰证明财团握着一角旧流程；但沈辞只拿到方法边界，不拿到立场答案。阶段末转向下一段任务调度与改名边缘。

- 当前完成：第171-175章已落正文；父线旧页脚只读裁切边和缺项格式，页脚页码只作残化痕，裁切边只证明旧页提前拆离，旧页缺项格式只作防补源同类，父线方法边界只作证物旁读结论。下一阶段承接`方法边界：待调度复核`，防完整指令重新借边界派单或调度沈辞。

## 五、阶段必须兑现的信息

- 二次背注是临时证被多方流程借用后的复核疤痕，不是系统奖励。
- 文修回流件和候选痕同类，但不能直接证明来源。
- 财团能拿出半门样本，说明其掌握父亲线一角；但它仍不能补完整签码。
- 沈辞的职业经验继续有用，但越接近旧流程，临时证损痕和自名缺口越危险。

## 六、阶段禁止

- 禁止揭开编目庭本体。
- 禁止让沈辞稳定掌握改名。
- 禁止让父亲完整签码或完整立场出现。
- 禁止把财团半门写成大型新门域副本。
- 禁止让顾焰成为可靠队友或解释型救场角色。
- 禁止让知微完全康复或进入并肩作战状态。

## 七、阶段爆点

第155-160章左右，财团半门样本可证明“候选痕不是城防产物，也不是财团现造”，但样本封袋内侧有一条人为刮断的旧红章压痕。它像父亲手法，却故意不留完整签名。沈辞越想读，临时证二次背注越试图把他写成`原修复员`。
