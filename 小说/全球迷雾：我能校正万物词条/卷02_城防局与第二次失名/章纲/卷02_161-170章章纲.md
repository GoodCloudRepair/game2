# 卷02_161-170章章纲

## 使用说明

- 对应卷：卷02 城防局与第二次失名
- 对应阶段：阶段02 / 文修回流与财团半门
- 当前目标：承接第160章`半门试开席：待旁签`，把半门试开写成责任、宽度、立场、归档、训练化的连续争夺，而不是新门域副本。
- 章内约束：不全开半门；不让沈辞签完整责任；不让顾焰拿走完整影像、训练样本或父亲完整真相；不让知微康复或参战；不让三名待核者成为训练标签。

## 段落总钩

- 起点：更高复核席将完整来源核验改写为缺失原因待验后，生成`半门试开席：待旁签`。
- 当前落点：第170章已将旧项目名压成缺首字题头，章末生成`父线旧页脚：待旁读`。
- 后续方向：第171-175章处理父线旧页脚、页码、目录索引和更硬但不完整的父亲线证明，为阶段转向下一段任务调度与改名边缘做准备。

## 第166章 试开失败项不作全开入口

- 状态：已落正文（2075字）。
- 时间：D2 15:20-15:30
- 地点：半门试开席 / 试开摘要随袋栏
- 必须接钩：`试开失败项：待复核`贴住试开摘要或沈辞临时证，要求说明失败原因。
- 核心冲突：财团会把失败解释为样本不足、操作不足或宽度不足，从而要求全开、二线扩读、补签责任或训练样本。

- 禁触：不让失败项打开半门内层，不新增副本。
