# 第197章：人工旁观人不得写沈辞

## 1. 本章定位

- 所属卷：卷02 城防局与第二次失名
- 所属阶段：阶段03 / 任务调度与改名边缘
- 章节功能：防止“人工旁观”通过旁观人、观察员、判断人、说明人、口令或原话留痕写回沈辞
- POV人物：沈辞
- POV类型：第三人称贴近沈辞
- 时间：D2 20:30-20:40
- 地点：安置区总表边屏 / 人工旁观人栏 / 沈辞临时证边
- 实际字数：2061

## 2. 本章目标

- 核心冲突：系统用“最懂风险的人”试图把沈辞写成人工旁观人、临时观察员、限制判断人、说明人或候选提示口令
- 局面变化：旁观人栏空，沈辞只提供限制判断内容，不作观察身份、岗位、口令或原话样本
- 本章新增规则：`人工旁观人：不得写沈辞`
- 章末钩子：`巡检班次：不得生成`

## 3. 本章剧情梳理

- 灰栏先以“旁观人必要项”要求填写沈辞，被沈辞拆成姓名索引陷阱。
- 系统改写为临时观察员、限制判断人和总表外沿说明人，均被拒绝。
- 临时证、候选身份、医疗身体状态和现场口令被依次试探，最终只留判断内容，不留提出者或原话。
- 结尾人员栏清空后，系统转向时间框，引出班次生成风险。
