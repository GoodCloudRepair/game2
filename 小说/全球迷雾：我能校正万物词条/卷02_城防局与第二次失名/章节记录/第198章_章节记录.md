# 第198章：巡检班次不得生成

## 1. 本章定位

- 所属卷：卷02 城防局与第二次失名
- 所属阶段：阶段03 / 任务调度与改名边缘
- 章节功能：旁观人为空后，防止系统把人工旁观转成城防、医疗、待核者或财团的轮值班次
- POV人物：沈辞
- POV类型：第三人称贴近沈辞
- 时间：D2 20:40-20:50
- 地点：安置区总表边屏 / 巡检班次时间框 / 城防值守表边缘
- 实际字数：2002

## 2. 本章目标

- 核心冲突：系统试图用“连续人工旁观”生成临时巡检班次、值守并入、医疗旁观、待核者轮看、财团复核班和未读队列
- 局面变化：巡检班次不生成，不排城防、不排医疗、不排待核者、不排财团，不留待查看或提醒队列
- 本章新增规则：`巡检班次：不得生成`
- 章末钩子：`外沿灰栏：只读不巡`

## 3. 本章剧情梳理

- 总表边屏把现场人物压成灰点并配上时段，试图生成临时巡检班次。
- 杜衡拒绝城防值守表并入巡检，林照拒绝医疗外线和十七白床参照。
- 三名待核者被试探成轮看者，布包女人差点口述，被沈辞及时压回。
- 财团复核班、未读队列、待查看和自动消失提示均被删除，结尾转入只读不巡。
