# 第121章：临时证不准过夜

## 1. 本章定位

- 所属卷：卷02 城防局与第二次失名
- 所属阶段：阶段01 / 入档与编制失名
- 章节功能：卷02开篇，把沈辞从封门幸存者正式写入城防局监管借用流程
- POV人物：沈辞

## 2. 本章目标

- 本章要推进的主线层级：卷02官方体系线、主角身份线
- 本章核心冲突：临时证能证明沈辞有用，却把他锁进更深监管半径
- 这章结束后局面具体变化：沈辞的临时证通过入档口，但证背自行补出`不得离开监测半径`
- 这章会让读者对谁产生新的认知：城防局不是单纯奖励沈辞，而是先把他登记、复核、限制后再使用
- 本章必须交代的信息：限制不是杜衡盖章，也不是顾焰黑箱投影，而是临时证背面自行渗出
- 本章新增物品/能力/规则/伏笔：临时证背面新增`不得离开监测半径`；评估传票：`封门余留活项 / 反读风险评估 / 立即入档`
- 本章结尾钩子：反读风险评估传票落下
- 本章风格目标：冷、被管、制度化，不写庆功
- 本章对应卷目标：建立“胜利后的不自由”，为HL-02体制内失名做前置

## 3. 本章剧情梳理

- 开场状态：沈辞手中临时证背面`编目师候选`仍在，城防执行员要求临时证先过入档口
- 事件一：入档口正面读取成功，`监管借用`、`异常能力评估未完成`等项被写入城防系统
- 事件二：背面扫描卡死，城防终端、财团复核点和监测缝终端都无法读出完整来源
- 事件三：证背自行补出`不得离开监测半径`，杜衡只按执行层处理，不解释来源
- 结尾结果：评估传票落下，沈辞被要求立即入档并接受反读风险评估
