# 《全球迷雾：我能校正万物词条》卷01_091-100章章纲

## 使用说明

- 当前状态：091-100正文已完成，101已转入下一组章纲执行
- 回填原则：若后续正文细节调整，以正文与章节记录为准，再反写本章纲
- 对应阶段：卷01 阶段04 / 门域核心前段
- 本组目标：
  - 把“到核心外窗”推进成“正式入核”
  - 让知微从被追索对象推进成内圈共鸣锚点
  - 把沈辞的特殊性从“规则适配者”推进成“未写圆活项”
  - 继续给父亲线留实物、留视线、留半句，不一次讲透

## 第091章 入核口不收完整人

- 章节功能：把核心外窗后的第一道真正入核规则写死，明确沈辞只能以“未写圆活线”身份继续前送
- 核心冲突：沈辞必须主动退掉门外挂识，压住完整姓名、完整关系和完整来处，才能不被入核口退回
- 主要剧情：入核白片要求沈辞先退门外挂识，不许带着完整旧称和完整关系进门，最终只认下他`活 / 在`
- 信息点：`入核口 / 不收完整人`；`完整旧称，退回外窗`；`先退门外挂识`；`活 / 在`
- 章末钩子：侧缝亮出`先校物 / 后校人`

## 第092章 先校物后校人

- 章节功能：让核心真正把“人”和“人身上带着的外层东西”拆开处理，并把沈辞主动接半步
- 核心冲突：沈辞必须接受自己一路带来的挂识、外写、残物被逐项归类，才能保住继续入内的资格
- 主要剧情：白托盘先收走门外挂识与外写，把`样残边`判成`可续链`，最终给出`活项前移 / 后校本人`
- 信息点：白托盘四格`挂识 / 外写 / 残物 / 后身`；`样残边 / 可续链`；`止于窗 / 止于口 / 可续 / 可入`
- 章末钩子：真正轮到“后校本人”

## 第093章 后校先看后身影

- 章节功能：把核心“后校本人”的第一步写成先认人留在物上的后身影，而不是先认脸和名字
- 核心冲突：沈辞必须在不被白道逼乱节奏的前提下，看懂后校口到底把“人”当成什么来读
- 主要剧情：沈辞进入后校白道，发现核心先不看正面、不问姓名，而是从他碰过、留过、被记过的物痕里抽出后身影
- 信息点：`后校先看后身影`；`正面不作本人凭`；后身影由物痕、停留痕、被记录痕共同构成
- 章末钩子：白道终点翻出`后身格不认单人影`

## 第094章 后身格不认单人影

- 章节功能：揭出沈辞能继续往里走，不是因为他是单独的人，而是因为他的后身早被17线、样残边和更旧的链痕并挂
- 核心冲突：沈辞必须接受“自己已经不是单人影”的事实，同时找出哪一层影不是自己的
- 主要剧情：后身格把沈辞拆成数层并挂影线，单人影被直接判空，真正被留下的是和17、样带、旧流程纠在一起的并链后身
- 信息点：`单影不作准`；`并链影留格`；沈辞的后身不是独立人的后影，而是一条复合链影
- 章末钩子：最深那条旧影前端翻出`前来句不留完整称`

## 第095章 前来句不留完整称

- 章节功能：把父亲线第一次推进到“他到过这里”，但仍只给半句，不给整句答案
- 核心冲突：沈辞必须在碎句里判断哪一条真和父亲有关，同时不能被“我终于快知道了”的冲动拖快
- 主要剧情：白道后的留句格只存半句前来提醒，完整自称和完整亲称都会被剪断，沈辞在其中认出父亲记号风格留下的`先看定层……`
- 信息点：`前来句不留完整称`；`满句退白`；父亲到过这一层，但留下的也只能是半句提醒
- 章末钩子：内侧去向栏正式亮出`共鸣项先入定层`

## 第096章 共鸣项先入定层

- 章节功能：明确知微不是被核心随手暂存，而是被当作稳定内圈秩序的共鸣锚点使用
- 核心冲突：沈辞必须压住立刻去碰知微的本能，先看清她在内圈到底承担什么位置
- 主要剧情：沈辞隔着定层环看见共鸣项先入位，知微所在那一格每次起伏都会带动周围白环回稳，核心第一次露出“用她定层”的真意
- 信息点：`共鸣项先入定层`；知微不是单纯被收，而是在替某一层维持稳定；共鸣项越稳，内圈越不乱
- 章末钩子：近层警示牌翻出`活项不得直认共鸣源`

## 第097章 活项不得直认共鸣源

- 章节功能：把兄妹重逢再往后压半步，写明沈辞不能直接认、直接叫、直接扑向知微，否则两边都会被打回
- 核心冲突：知微明明就在更近处，沈辞却必须在最难的一刻继续不把“哥哥”和“知微”说满
- 主要剧情：沈辞一旦把视线和心念锁死在知微那一格，定层白环就立刻紧收，核心明示活项不得直认共鸣源，逼他改走更冷的一条“回认”路径
- 信息点：`活项不得直认共鸣源`；直接认亲会让活项与共鸣项同时失稳；核心只允许绕物、绕旧痕、绕回认去重新接近
- 章末钩子：侧牌推出下一条合法路径`回认台先收旧触物`

## 第098章 回认台先收旧触物

- 章节功能：把“不能直接认人”改成“先送她认得的物”，让兄妹线重新借物搭桥
- 核心冲突：沈辞必须从自己手上与知微有关的残物里，找出真正能被回认台接受的一件
- 主要剧情：回认台不收现认、不收整件，只收知微先碰过且长期留下身体习惯的小物痕；沈辞最终用月票夹内侧常触磨白边搭出半步回认白桥
- 信息点：`不收现认`、`不收整件`、`收她先碰过的`、`链上残边 / 非她旧触`、`外壳太多 / 先取常触边`、`常触边入台 / 旧壳有应`、`静名槽不放完整亲称`
- 章末钩子：旧触认回后，下一步转入`静名槽`，但完整亲称不能放进去

## 第099章 静名槽不放完整亲称

- 章节功能：让“回认”真正落到称呼来处层，但仍不能把完整亲称说出口
- 核心冲突：沈辞已经搭出旧触白桥，却不能把完整的`哥`、`知微`或任何门外亲称直接送进静名槽
- 主要剧情：静名槽要求先交“静下来的称”，沈辞压下完整亲称，改用“小微”之前那点称前停顿与未说满半称，让知微先认出旧称习惯而不是被称呼直接扣回门外关系
- 信息点：`静名槽`、`完整亲称，退`、`意满退`、`门外名，退`、`称前静处 / 旧触承接`、`半称，可收`
- 章末钩子：静名槽只接住一半称呼，内圈给出下一步：`认回的人不从正环出`

## 第100章 认回的人不从正环出

- 章节功能：让本组以一次高压回认完成阶段性情感回收，同时把更深内圈路线真正打开
- 核心冲突：知微被半步认回后不能按沈辞期待从定层正环离开，正环一动就会牵连二环稳定
- 主要剧情：沈辞本能想从17格正前方带离知微，正环立刻收紧并牵动二环；他改用月票夹压住旧触白桥转向偏缝，知微也把回应推向偏环
- 信息点：`正环不出`、`回认非带离`、`源位不脱层`、`偏环待接`、`外锁已接 / 待内名`
- 章末钩子：外层封门装置接入内环，但只能锁外名，后续必须转向内名读口
