## 第041章 缺考的人先补名字

- 章节功能：正式把17号线从广播楼推进到西教学楼，并立起“先认位置、再认名字”的补名秩序
- 核心冲突：沈辞必须在不惊动补名桌的前提下，尽快读懂西教学楼这一层到底先认什么，才能决定下一步怎么继续跟进去
- 主要剧情：17号线从广播楼正式推进到西教学楼补名口，沈辞看见这栋楼先认位置、列次和铃声，再决定要不要把名字补回去
- 信息点：`缺考的人先补名字 / 未落座前不问来处 / 借名者不得入列`；补名桌名录没有姓名栏，只认楼层、室号、列、号；黑板写`一铃入座 / 二铃封题 / 缺考补名，不得回头`；这里比广播楼更先认位置；林照摸到`考务后勤`蓝布外套和`一号补名室列签`
- 章末钩子：第一铃后，17号被带向`一号补名室`，更深处真正的课桌秩序已经露出轮廓

## 第042章 一号补名室不收本名

- 章节功能：让沈辞与林照借`考务后勤`和列签潜入一号补名室，并正式立住“补名只照座签抄”的规则
- 核心冲突：沈辞必须在不暴露潜入身份的前提下，看懂一号补名室怎样把人和旧座位残名绑在一起，同时继续跟住知微
- 主要剧情：沈辞与林照借`考务后勤`和列签潜入一号补名室，确认这里补的不是对象自己的名字，而是桌角旧座签愿意借出的残名缺口
- 信息点：`一号补名室不收本名 / 补名只照座签抄 / 写满者先入空卷`；黑板写`先坐后抄 / 未点到列不落第一字 / 空卷不改名`；补名室用的是旧年级旧班列裁下来的座签残名；写自己完整本名会被判入`空卷`；中右列当前比左列`空卷待收`更安全；17号桌签下先露出的是`沈`
- 章末钩子：知微补名桌角的黑纸掀起后，17号座签下露出沈家的姓

## 第043章 先写姓不补后字

- 章节功能：让17号正式把`沈`姓写进一号补名室流程，并把补名室第一铃阶段的规则从“照签抄”推进到“先收姓、后点尾”
- 核心冲突：沈辞必须在不能出手、不能喊停的前提下，看着知微把`沈`姓先写进补名室，同时快速拆出这一层对前字、后字和回描的限制
- 主要剧情：17号在一号补名室里只被允许先写下`沈`姓，抢落后字、回描前字或提前护尾都会被当场压回流程
- 信息点：`先写姓，不补后字`；17号白夹变为`待尾`；`后字抢落`会记`迟到补列`；`前字入列，不得回描`；`铃前不护后字`；补名桌存在`前字簿`与后字灰纸套；黑板写`前字入列 / 后字待点 / 二铃后不改姓`；活动牌翻出`二铃后，待尾点字`
- 章末钩子：第二铃后才会真正点尾，而17号已经被先压进`沈`姓之下

## 第044章 二铃之前只点姓

- 章节功能：把一号补名室从“先写姓”推进到“二铃前只点姓”，并正式抬出同姓未齐的锁与`沈`姓下的`知 / 辞`尾卡
- 核心冲突：沈辞必须在不暴露自己也被`沈`姓卷入的前提下，看懂同姓与后字之间的关系，并确认知微后面那一半名字究竟将如何被点
- 主要剧情：补名室在第二铃前只按姓记账，`周姓未齐`坐实同姓锁，沈辞则在后柜抽屉里看见`沈`下压着`知`与`辞`两张尾卡
- 信息点：二铃前女老师只点姓，不点全名；`周姓未齐，只点前，不点尾`；`只认点到的姓，不替自己挑后字`；前字簿按姓记账；后柜抽屉按姓存放尾卡；`沈`字签下同时压着`知`和`辞`；窗边白卷题头是`夜间缺考列`，且带有`未齐`红框
- 章末钩子：夜间缺考列的白卷与`沈`下并排的`知 / 辞`同时露出，说明名字层的危险还要继续往上走

## 第045章 不要替十七点尾字

- 章节功能：让沈辞与林照第一次在补名室内部做一次极小却关键的规则对抗，把知微从本层点尾里拖开，改送`上楼并尾`
- 核心冲突：第二铃后就要点尾，沈辞必须在不暴露身份、不直接碰知微的前提下，用最小动作改歪一号补名室对`沈`姓的计算
- 主要剧情：第二铃前后，沈辞与林照借`同姓未齐`规则和另一张旧`沈`姓空桌，硬把17号从本层点尾里拖成`上楼并尾`
- 信息点：黑板新增`同姓未齐，不并后字 / 代点作代认`；同姓未齐会让本层无法并尾；沈辞借另一张旧`沈`姓空桌和灰夹制造新的未齐对象；女老师因此判定`沈姓未齐，先不上尾`；17号被发`上楼并尾`黑边卡；门外已有承接`待尾上楼`的旧教工雨衣；白卷露出`二层待尾`
- 章末钩子：知微没在一号补名室被点死后字，却被继续送向`二层待尾`

## 第046章 上楼只带姓牌

- 章节功能：把17号从一号补名室正式送上二层待尾区，并立住“上楼只带姓牌”的转层规则
- 核心冲突：沈辞必须在不暴露继续跟线意图的前提下，看懂楼层交接时到底遮掉了什么、留下了什么
- 主要剧情：17号被换上黑边送层夹，从一号补名室正式送往二层待尾，教学楼第一次露出按层拆名、按批次交接的内部转运链
- 信息点：`上楼只带姓牌 / 待尾不露第二格 / 送层不报本班`；黑边卡存在分批编号；楼梯是`待尾上行`而非普通二层；送层对象连班项都要一起遮掉；平台登记纸写`待尾对象交接`
- 章末钩子：17号被送到`二层待尾区`，门口规则写着`不收并肩人`

## 第047章 待尾区不收并肩人

- 章节功能：立住二层待尾区按姓隔位、按未齐记账的规则，并把对面的空`沈`位真正写成威胁
- 核心冲突：沈辞必须在不能惊动待尾区执行层的前提下，尽快读懂这里到底在等什么齐
- 主要剧情：二层待尾区把17号记成`沈`姓`到位一`，并用隔位、隔线和`未齐不过册`明确这一层真正等的是另一个`沈`补齐
- 信息点：`待尾区不收并肩人 / 同姓先隔位 / 未叫到不看对面`；同姓不并肩也不照面；黑板记`到位 / 未齐`；`未齐不过册`；对面的`沈`牌下压着`待补位`
- 章末钩子：老女人抱出`旧位册`，直接点到`沈姓，先核旧位`

## 第048章 沈姓先核旧位

- 章节功能：把二层流程推进到`旧位核对`，并正式揭出`知`是旧生尾卡、`辞`是广播楼留下的`东口外应`
- 核心冲突：沈辞必须在不被学校这条线反向盯住的前提下，确认`辞`那张尾卡到底是不是冲着自己来的
- 主要剧情：二层流程推进到`旧位核对`，`知`被坐实为旧生尾卡，`辞`则被翻成广播楼留下的`东口外应`补位
- 信息点：`同姓先核旧位 / 旧位不认今序 / 位不定，不翻尾卡`；何姓示范证明碰木牌即算认位；`知`尾卡来自旧学生旧座位；`辞`尾卡来自`外应补位`册，旁注`东口外应`；`广播`残布和`近场相关，沈……辞`复写条坐实学校记录了沈辞
- 章末钩子：老女人宣布`沈姓，一位旧生，一位外应，先排旧位`

## 第049章 不要替十七排旧位

- 章节功能：在`知 / 辞`双来源压桌之后，把17号从`旧位成`的临界线拖成`旧位不清`
- 核心冲突：沈辞必须在不能出声、不能靠近、不能代指的前提下，阻止知微在二层把旧位先认下来
- 主要剧情：在`知 / 辞`双来源压桌后，沈辞与林照靠`门位调换，以后册为准`制造合法模糊，把17号从`旧位成`硬拖成`旧位不清`
- 信息点：`排旧位不许代指 / 靠窗靠门都算认位 / 旧位不清，送旧名室`；线外不代位；旁声也会被当作`旁声作证`；林照扶正压片后露出`门位调换，以后册为准`与斜改线，使靠门与中列变得无法明确判断；最终17被判`旧位不清`
- 章末钩子：17号被记去向`旧名室`

## 第050章 旧名室先收半个人

- 章节功能：打开`旧名室`这一层，并把二层的尾字、旧位问题继续压进`旧称`与`来源未并`的新危险里
- 核心冲突：沈辞必须继续贴着门缝看清旧名室的处理逻辑，同时压住自己在`旧称`层面被点出来的本能
- 主要剧情：旧名室被正式打开，17号被加挂`位失`灰扣压成“半个人”，而门内直接把知微与门外的沈辞一起点成`沈姓两位，先交旧称`
- 信息点：17号被加挂`位失`灰扣；`旧名室先收半个人 / 未核旧位，不翻全名 / 先认缺口，再认人`；门边写`尾未定者，先交旧称`；旧名室内收集乳名、小字、旧喊法、门外称；`沈`下被拆成`旧生 / 外应`两类来源；`来源未并，不翻全名`
- 章末钩子：旧名室不仅在等知微补完后字，也在等沈辞把自己的“旧称”一起交进去
