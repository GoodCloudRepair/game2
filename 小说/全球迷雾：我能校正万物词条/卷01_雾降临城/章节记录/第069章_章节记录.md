# 第069章：井栏后只认单向镜

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学收束
- 章节功能：正式让沈辞在`17`号井位前确认知微，并坐实听井对象位通过单向镜被观察
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 02:41
- 本章结束时间：D2 02:48
- 时间跨度：7分钟
- 与上一章间隔：0分钟
- 地点：西楼听井 / `17`号井栏观察位、平行检修缝
- 目标字数：2000-2100字

## 2. 本章目标

- 本章要推进的主线层级：从“看到对象位人影”推进到“知微确实被固定在`17`号单向镜后的观察格里”
- 本章核心冲突：沈辞已经看到知微，却不能贴镜、不能认接、不能用正常相认方式确认她
- 这章结束后局面具体变化：知微的状态从抽象的观察对象，推进成被单向镜、听筒和井栏规则固定的真实观察格对象
- 这章会让读者对谁产生新的认知：对知微在观察格里的克制、对学校深层观察方式、对沈辞压住冲动的方式
- 本章必须交代的信息：`井栏后只认单向镜 / 镜前不贴对象脸 / 敲栏不过三`；`观位不作交接位`
- 本章新增物品/能力/规则/伏笔：单向镜观察格、窄台白纸、`复听后`提示、`转交白栏`白缝
- 本章结尾钩子：白缝后亮出`复听后 / 转交白栏`
- 本章风格目标：把重逢感压到最低，用“看见但不能接”的冷规则把情感吊住
- 本章对应卷目标：学校线继续收束，并把知微推进到“即将被交给更外层系统”的边缘

## 3. 本章剧情梳理

- 开场状态：第068章末，`17`号灯格里的人影出现
- 事件一：井栏规则翻全，沈辞确认`17`号格是单向镜观察格而非交接位
- 事件二：知微在窄台纸上写下`复听后`，说明她知道自己仍在等下一段流程
- 事件三：观察格左侧白缝亮起，露出`转交白栏`
- 结尾结果：知微即将从井下被转去白栏交接层
