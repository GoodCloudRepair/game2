# 第098章：回认台先收旧触物

## 1. 本章定位

- 所属卷：卷01 雾降临城
- 所属阶段：阶段04 / 门域核心前段
- 章节功能：把“不能直认”转成可执行的旧触物回认
- POV人物：沈辞
- POV类型：近景限知
- 时间：D2 清晨
- 本章开始时间：D2 05:54
- 本章结束时间：D2 06:01
- 时间跨度：约7分钟

- 目标字数：2000-2200字（可上浮至2300字，不能低于2000字）

## 2. 本章目标

- 本章要推进的主线层级：保护知微 / 门域核心线 / 兄妹线
- 本章核心冲突：沈辞必须找到核心承认的旧触物，不能把整件物、整段关系或完整称呼直接送进去
- 这章结束后局面具体变化：回认台收下月票夹内侧常触磨白边，知微对这一点旧触产生稳定反应，回认路径前移到静名槽
- 这章会让读者对谁产生新的认知：对核心“认物不认情绪”的严格顺序，对知微仍能被日常小习惯唤回一点的状态
- 本章必须交代的信息：旧触物必须是她先碰过、长期碰过的实体小痕迹，不收现认、不收整件
- 本章新增物品/能力/规则/伏笔：`不收现认`、`不收整件`、`收她先碰过的`、`链上残边 / 非她旧触`、`外壳太多 / 先取常触边`、`常触边入台 / 旧壳有应`、`旧触认回 / 静名前移`、`静名槽不放完整亲称`
- 本章结尾钩子：下一步必须进入静名槽，但完整亲称不能直接放进去
- 本章风格目标：让情感从具体旧物和身体习惯里长出来，避免口头煽情
- 本章对应卷目标：把兄妹线从“可见不可认”推进到“可借旧物认回半步”

## 3. 本章剧情梳理

- 开场状态：第097章末回认台弹开，沈辞被迫从直认知微改走旧触物路径
- 事件一：回认台亮出`不收现认 / 不收整件 / 收她先碰过的`，先把`样残边`判为链上残边而非知微旧触
- 事件二：沈辞取出沈知微月票夹，没有递整件，也没有递便签文字，而是找到透明夹层内侧被她长期顶出的常触磨白边
- 事件三：回认台只收磨白边上的白痕，知微在17位产生极轻反应，二环随之稳住半分
- 结尾结果：旧触认回只搭出半步白桥，回认流程转入`静名槽不放完整亲称`
