- 章节功能：翻出旧借册，把`门前校时样带 / 外转文修 / 缺一位`这条老路钉实，并让父亲线抬头

## 2. 本章目标

- 本章要推进的主线层级：父亲线抬头 / 保护知微 / 主角异常线继续被更深流程卷住
- 本章核心冲突：旧借册揭出一条更早的“外转文修”老路，而沈辞又在侧注笔习里看见父亲痕迹，必须克制不当场说破
- 这章结束后局面具体变化：回流件被送进民修口不再只是“有人故意挂单”，而是沿用过一条更老的旧借转文修路径
- 这章会让读者对谁产生新的认知：对父亲在这条线上的可能参与、对旧档借册的历史层、对杜衡的观察敏感度
- 本章必须交代的信息：`门前校时样带 / 外转文修 / 缺一位`是真实存在过的旧借条目；侧注笔习与父亲留下的旧痕高度相似
- 本章新增物品/能力/规则/伏笔：旧借册、侧注`先罩壳，再走民口 / 别进正档`、`现场单裁失效`
- 本章结尾钩子：旧借册触发联席复核，白门后另一套权限要进场
- 本章风格目标：用旧卡、旧注和笔习勾父亲线，不直接人物揭秘
- 本章对应卷目标：让父亲线从回声和材料链推进到明确旧档证据层

## 3. 本章剧情梳理

- 开场状态：17柜与回流件刚被改送旧借册
- 事件一：旧借册被唤醒，亮出`门前校时样带 / 外转文修 / 缺一位`
- 事件二：更细的旧注露出“先罩壳，再走民口 / 别进正档”，沈辞从笔习里认出父亲痕迹
- 事件三：沈辞强压住不说破父亲，只承认像旧标注手；系统随即判`旧借未销 / 转联席复核`
- 结尾结果：白门后压来红授权条，另一套更高权限即将接手
