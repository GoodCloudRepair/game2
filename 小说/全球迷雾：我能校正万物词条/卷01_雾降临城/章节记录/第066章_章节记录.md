# 第066章：观察册不记正常缺席

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学收束
- 章节功能：把`观察册`这一层正式立住，并确认知微在学校里已被从普通旧生流程中分离出来，进入长期异常观察记录
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 02:21
- 本章结束时间：D2 02:28
- 时间跨度：7分钟
- 与上一章间隔：0分钟
- 地点：回听格下层 / 观察册夹袋前、外窗平行观察线
- 目标字数：2000-2100字
- 实际字数：2000字

## 2. 本章目标

- 本章要推进的主线层级：从“知微会对`哥`起稳定回听”推进到“她在学校里根本不按正常缺席学生处理，而是被单独记录观察”
- 本章核心冲突：沈辞必须压住对观察记录内容的本能厌恶，继续从冷表格里看清学校究竟怎样长期看待知微
- 这章结束后局面具体变化：学校对知微的处理不再是旧生、待补、待尾这些表层名目，而是进入真正的`观察册`体系
- 这章会让读者对谁产生新的认知：对知微在学校内部被观察的时长和深度、对学校深层执行方式、对兄称为何会被记成`稳`
- 本章必须交代的信息：`观察册不记正常缺席 / 异常回听另页 / 旧生册只作封皮`；记录项变成`常规点名 / 外称回听 / 停步方向 / 复核结果`
- 本章新增物品/能力/规则/伏笔：墨绿色观察册、`不归缺席`小章、`观察对象`夹袋签
- 本章结尾钩子：夹袋里露出暗红色分隔卡，标明`观察对象`
- 本章风格目标：冷到像真正的观察记录，但要让读者从格子错位里感到恶意
- 本章对应卷目标：继续收束学校线，把知微从“被追索的人”推进到“被系统长期观察的对象”

## 3. 本章剧情梳理

- 开场状态：第065章刚翻出`观察册`
- 事件一：观察册封面亮出`不记正常缺席 / 异常回听另页 / 旧生册只作封皮`
- 事件二：沈辞翻开记录页，发现学校记录的不再是班级缺席，而是`常规点名 / 外称回听 / 停步方向 / 复核结果`
- 事件三：他看到`不归缺席`与`转观察页`等标记，坐实知微早已被从普通学生处理链里剥离
- 结尾结果：夹袋中露出暗红色分隔卡与`观察对象`短签，下一章将进入更深层对象分类
