# 第029章：回执不认带队

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：旧城追索
- 章节功能：把北库17号线从“知道去向”推进到“拿到改签后的实际外送路线”
- POV人物：沈辞
- POV类型：主POV
- 时间：D1
- 本章开始时间：D1 21:20
- 本章结束时间：D1 21:34
- 时间跨度：14分钟
- 与上一章间隔：0分钟
- 地点：殡仪街社区文化馆北库签转区、十七号位、黑门外短驳巷口
- 目标字数：2000-2200字（可上浮至2300字）
- 实际字数：2122字

## 2. 本章目标

- 本章要推进的主线层级：从“17号被转往澜中临设”推进到“确认它必须经由西短坡与四号短驳继续外送”
- 本章核心冲突：沈辞和林照不能在北库里直接抢人，也不能让自己被补签流程反写进去，只能在极短时间里偷到下一段路线
- 本章必须交代的信息：`补签回执须本人复领 / 回执不认带队 / 空位先核后退`；17号改签后走`四号短驳`；签入地为`澜中临设西短坡`；`23:40前签入`；签入后不回旧城线
- 本章结尾钩子：沈辞放弃在北库里截人，决定`跟线`
- 本章风格目标：把“领物补签”写成比医院更冷的去人化物流恐怖
- 本章对应卷目标：把旧城后车线真正变成一条可追、可延伸、可继续变形的流程链

## 3. 本章剧情梳理

- 开场状态：白防护七组进入北库补签回执，沈辞与林照贴在黑箱阴影里观察
- 事件一：沈辞从新增规则条里判断出，17号空位必须被重新写进回执，窗口会因此生成新的改签路线
- 事件二：七组领头人去17号位复核空床，窗口打出新的`澜中临设西短坡`路条和`四号短驳`信息
- 事件三：沈辞与林照制造铁牌声响，趁七组回窗口时偷走蓝色存根
- 结尾结果：沈辞从存根上确认`先过西坡闸，再交临设外勤 / 签入后不回旧城线`，决定不在北库里硬抢，而是咬住后续车线
