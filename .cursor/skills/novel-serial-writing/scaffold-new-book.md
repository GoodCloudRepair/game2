# 开新书脚手架清单

在新目录 `【书名】/` 下执行。可复制 `全球迷雾：我能校正万物词条/` 的结构后清空内容，或从零创建。

## 初始化步骤（约30分钟）

### 1. 创建根目录与13个控制文件

- [ ] `00_写作约束与流程.md` — 从已有书复制后改主线四层、世界观底层、主角约束
- [ ] `01_章节记录模板.md` — 可直接复用，不改
- [ ] `02_章节总表.md` — 只留表头
- [ ] `03_人物台账.md` — 填主角 + 核心配角骨架
- [ ] `04_关系台账.md` — 只留表头 + 核心关系
- [ ] `05_物品与能力台账.md` — 填核心规则表 + 能力表
- [ ] `06_伏笔台账.md` — 只留表头 + 总纲级长线伏笔
- [ ] `07_写作风格约束.md` — 填本书文风（视角、句式、禁忌）
- [ ] `08_时间线约束与时间轴.md` — 填 D0/D1 规则 + 空时间轴表
- [ ] `09_POV视角约束.md` — 默认POV + 切视角记录表
- [ ] `10_能力升级约束.md` — 能力阶段表 + 升级记录表
- [ ] `11_卷级规划总表.md` — 各卷任务与卷尾落点
- [ ] `12_思维笔记.md` — 空模板 + 待定决策区
- [ ] `13_整书总纲.md` — 一句话梗概、题材、核心设定、八卷方向（可先只写第一卷）

### 2. 创建第一卷目录

```
卷01_【卷名】/
├── 章纲/
│   ├── 卷01_卷纲.md
│   ├── 卷01_阶段01_001-030章大纲.md
│   ├── 卷01_001-010章章纲.md
│   └── （其余阶段/章纲按需补）
├── 正文/
└── 章节记录/
```

### 3. 第一卷最小可写配置

开写前至少齐备：

- [ ] `13_整书总纲.md` — 一句话 + 第一卷任务
- [ ] `卷01_卷纲.md` — 卷尾落点明确
- [ ] `卷01_阶段01_001-030章大纲.md` — 第一段推进清楚
- [ ] `卷01_001-010章章纲.md` — 前10章逐章功能
- [ ] `03_人物台账.md` — 主角 + 第一卷核心人物
- [ ] `08_时间线` — D1 起点锚点

**不要**一开始铺满120章章纲；按阶段推进，写完一段再拆下一段。

### 4. 第一卷开写前必填占位

在 `13_整书总纲.md` 填：

```markdown
## 一句话梗概
【】

## 主角四层目标
1. 表层：
2. 中层：
3. 深层：
4. 终极：

## 世界观底层（所有超常现象必须能回到这里）
- 
- 
- 

## 第一卷卷尾落点
【】
```

在 `00_写作约束与流程.md` 改：

- 主线四层目标（本书专属）
- 世界观来源统一（本书专属）
- 主角写法约束（本书专属）
- 单章字数（默认2000-2100，可改）

### 5. 写第一章的执行顺序

```
看 13 → 卷纲 → 阶段01 → 001-010章纲
→ 看时间轴/人物/伏笔台账
→ 复制 01_章节记录模板 → 第001章_章节记录.md
→ 填六问 + 1-9节
→ 写 正文/第001章_标题.md
→ 勾第10节
→ 回填 02/08/03/04/05/06/12
```

## 从旧书复制时的清空规则

复制 `全球迷雾：我能校正万物词条/` 时：

| 文件 | 处理 |
|------|------|
| `01_章节记录模板.md` | 保留 |
| `00_写作约束与流程.md` | 改本书专属段落，保留通用流程 |
| `02_章节总表.md` | 清空数据行 |
| `03-06` 台账 | 清空章节状态，保留表头 |
| `08_时间线` | 清空 T 编号事件，保留规则 |
| `12_思维笔记` | 清空决策记录 |
| `卷01_*` | 新建卷目录，不复制旧正文 |

## 验收：新书可以开写

- [ ] 文档层级齐全（至少到前10章章纲）
- [ ] 主角与核心关系已入台账
- [ ] 第一卷卷尾落点已写
- [ ] 第001章章节记录可填、六问答得出
- [ ] Agent 可按 `novel-serial-writing` Skill 执行六步
