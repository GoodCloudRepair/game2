# 末日割草+放置挂机

> Cocos Creator 3.8 · 末日废土 · 抖音小游戏竖屏 · 40天开发

## 游戏简介

俯视角2D割草射击 + 放置挂机混合玩法。玩家操控末日幸存者，在变异怪物潮中割草升级、选择肉鸽技能、击败Boss通关，离线挂机积累资源永久养成角色。

## 核心玩法

- **割草战斗** — 3-5分钟/局，自动射击+摇杆闪避，升级选技能(肉鸽3选1)
- **放置挂机** — 离线产出金币/材料，上限8小时，广告双倍
- **永久养成** — 角色等级/星级 + 武器强化 + 装备收集 + 天赋树
