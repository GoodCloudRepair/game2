# 《三国：从黄巾开始抢天下》写作约束与流程

## 目标

防止长篇连载中后期主线散掉、人物跑偏、地图与归属混乱、伏笔不回收。

台账用于控进度，不服务于悬疑压迫文风。

## 一、硬约束

### 1. 主线优先级

1. 表层：活下去，抢兵权与地盘
2. 中层：据州自立，班底成型
3. 深层：改写名场面，扩大争霸优势
4. 终极：一统或开创新朝

无关四层的剧情不写或压缩。

### 2. 世界观来源统一

所有超常点只能回到：

- 争霸日历
- 名将缘
- 节点预支

不另开无关神秘体系。

### 3. 主角写法约束

顾川强在：

- 卡节点
- 敢下场
- 会用人
- 会把预支变成实打实的胜仗

可以打，可以谋，但不能长期装弱不打，也不能开局无脑全知横推。

### 4. 能力写法约束

争霸日历 / 名将缘 / 节点预支必须写清：

- 触发条件
- 效果
- 代价
- 当前上限

禁止临时救场新开挂。

### 5. 人物写法约束

重要角色至少承担：推主线 / 对照 / 冲突 / 情感锚点 / 揭示局势之一。  
名将不要一章收齐。

### 6. 信息投放约束

每章最多扩展1个关键新信息（一城、一将、一节点、一条制度）。

### 7. 章节功能约束

每章至少完成：推进剧情 / 关系 / 信息 / 伏笔 中的一项，最好两项。

### 8. 章节字数约束

硬下限2000字；软目标2000-2200字；可上浮至2300字。

### 9. 文风约束

快、爽、外放。  
战争3-5章内分胜负。  
禁止悬疑长扣、冷压迫停顿、大段考据。

## 二、写作流程（六步）

```
看纲 → 看台账 → 填章节记录 → 写正文 → 勾连续性检查 → 回填台账
```

层级：

```
13_整书总纲 → 卷纲 → 阶段大纲 → 当前10章章纲
```

## 三、单章检查（写完必勾）

- 字数达标
- 有兑现（打脸/收地/收人/坐实预支之一）
- 有章末钩子
- 地图/归属/粮草若变，已记台账
- 没有新开无关体系
