# PRD: 末日割草+放置挂机 抖音小游戏

> 创建时间: 2026-05-18
> 状态: 已确认

## 1. 需求背景

开发一款面向抖音小游戏平台的割草+放置挂机混合玩法游戏。利用短视频导流获取用户，通过 IAA+轻度 IAP 混合变现。

## 2. 目标

- 35-40 天内完成可上架版本
- 抖音 WebGL 小游戏，竖屏 9:16
- 次留 ≥ 40%，7 留 ≥ 15%
- 单用户日均广告观看 ≥ 5 次

## 3. 方案摘要

**题材**: 末日废土（丧尸/变异怪物）
**视角**: 俯视角 2D
**核心循环**: 割草战斗(3-5分钟/局) + 肉鸽选技能 + 放置挂机离线收益 + 角色永久养成
**引擎**: Cocos Creator 3.8+（TypeScript）
**平台**: 抖音小游戏（原生导出，首包 < 4MB）
**变现**: IAA 为主 + 轻度 IAP（月卡/去广告/限定皮肤）
**联网**: 弱联网（登录+云存档+排行榜）

## 4. 功能范围

### 包含
- 割草战斗系统（自动射击+摇杆闪避+怪物波次+Boss）
- 肉鸽技能系统（升级弹3选1，局内成长）
- 多角色+多武器系统
- 放置挂机系统（离线产出金币/经验/材料）
- 角色永久养成（等级/星级/装备/天赋）
- 关卡章节系统（不同场景+难度递增）
- 广告变现（复活/双倍奖励/宝箱/每日免费抽）
- 商城 IAP（月卡/去广告/限定礼包）
- 排行榜（通关层数）
- 成就/图鉴

### 不包含
- 实时 PVP
- 联盟/公会社交
- 剧情/对话系统
- 直播互动玩法

## 5. 关键设计决策

| 决策项 | 结论 | 理由 |
|--------|------|------|
| 题材 | 末日废土 | 短视频素材冲击力强，怪物设计自由度高 |
| 视角 | 俯视角 2D | WebGL 性能友好，开发快，美术成本低 |
| 屏幕 | 竖屏 9:16 | 抖音原生体验，单手操作 |
| 单局时长 | 3-5 分钟 | 碎片化，适合短视频用户 |
| 联网 | 弱联网 | 不需要实时对战，降低服务端成本 |
| 同屏怪物上限 | 200 | WebGL 性能约束 |
| 引擎 | Cocos Creator 3.8+ | 原生支持抖音导出，包体<10MB，2D性能优 |

## 6. 核心玩法设计

### 6.1 割草战斗

```
进入关卡 → 怪物波次刷新（30秒/波，共5-8波）
         → 角色自动射击最近敌人
         → 玩家摇杆控制移动闪避
         → 击杀掉落经验宝石
         → 经验满升级 → 弹出3选1技能
         → 最终波Boss → 通关奖励
```

**操作方式**：
- 左手虚拟摇杆控制移动
- 攻击全自动（朝最近敌人）
- 主动技能按钮（大招，CD制）

### 6.2 肉鸽技能（局内）

每次升级从技能池随机抽3个，玩家选1个：
- 攻击类：子弹+1、射速提升、范围伤害、穿透
- 防御类：护盾、回血、减伤、闪避
- 辅助类：移速、拾取范围、经验加成、金币加成

技能可叠加升级（同技能选多次变强）。

### 6.3 放置挂机

- 离线自动产出金币+材料（按拥有角色战力计算产出速率）
- 上限 8 小时收益，超过不再累积
- 登录领取，可看广告双倍
- 挂机界面显示角色自动战斗动画（纯表现）

### 6.4 角色养成（局外永久）

| 养成维度 | 说明 | 消耗 |
|----------|------|------|
| 等级 | 提升基础属性 | 金币 |
| 星级 | 大幅提升+解锁被动 | 角色碎片 |
| 装备 | 武器/护甲/饰品 | 材料 |
| 天赋 | 通用被动加成树 | 天赋点(通关获得) |

### 6.5 变现设计

**IAA 广告点位**：
| 场景 | 触发 | 奖励 |
|------|------|------|
| 战斗死亡 | 复活1次 | 续命30秒 |
| 通关结算 | 双倍奖励 | 金币×2 |
| 离线收益 | 领取时 | 收益×2 |
| 每日宝箱 | 免费开启 | 随机材料/碎片 |
| 幸运转盘 | 每日3次 | 随机奖励 |
| 技能重选 | 不满意当前3选1 | 刷新选项 |

**IAP 商品**：
| 商品 | 价格 | 内容 |
|------|------|------|
| 月卡 | 30元 | 每日领元宝+去广告30天 |
| 周卡 | 12元 | 每日领元宝7天 |
| 去广告 | 68元 | 永久去广告（保留主动看广告奖励） |
| 限定皮肤 | 18-30元 | 角色外观 |
| 首充礼包 | 6元 | 角色+资源大礼包 |

## 9. 验收标准

- [ ] 首包 < 4MB，总包体 < 10MB
- [ ] 战斗帧率 ≥ 30fps（200怪同屏）
- [ ] DrawCall < 80
- [ ] 首屏加载 < 3s
- [ ] 核心循环可玩（割草→升级→通关→养成→再战斗）
- [ ] 放置离线收益正确计算
- [ ] 广告全点位正常播放和回调
- [ ] IAP 充值全链路通过
- [ ] 抖音审核规则适配
- [ ] 竖屏全机型适配（刘海屏/折叠屏）

## 10. 风险与应对

| 风险 | 应对 |
|------|------|
| WebGL 同屏性能不足 | 对象池+合批+控制怪物上限200+LOD简化远处怪物 |
| 包体超限 | 图集压缩+按需加载+音频压缩+代码裁剪 |
| 割草手感不够爽 | 优先打磨打击感（屏幕震动+击退+数字飘字+慢动作） |
| 数值膨胀太快 | 设计软顶+赛季重置机制 |
| 广告 eCPM 低 | 优化广告点位时机（死亡瞬间/大奖励前） |
