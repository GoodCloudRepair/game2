# 物品与能力台账

## 核心规则

| 规则 | 说明 |
| --- | --- |
| 灵气复苏 | 满3年；城市可居住；野外与废弃区风险升高 |
| 武道公开化 | 高考含武道科；功法可合法购学；顶尖仍稀缺 |
| 管理局优先 | 超凡事件由军警+管理局处理；民间猎杀受资质约束 |
| 洞察非面板 | 只描写裂痕/前摇/药性年份/功法别扭处，禁止词条UI |

## 金手指：万物洞察

| 项 | 内容 |
| --- | --- |
| 触发 | 专注注视；危机时被动唤醒（第001章） |
| 效果 | 见生物薄弱处与发力前摇；见药材大致年份与药性偏向；见功法运转别扭处与自身瓶颈方向 |
| 不能 | 读心；看穿完全伪装的高境杀意细节；弹出数值血条/技能名 |
| 代价 | 短时间连续洞察超过约3次→头痛欲裂；高两境目标只见模糊裂痕；情绪失控时失准 |
| 明面说法 | 「感知类天赋」；不对任何人说能量化看破 |
| 当前阶段 | 第001章已觉醒雏形（仅战斗向弱点/前摇；用后头痛） |
