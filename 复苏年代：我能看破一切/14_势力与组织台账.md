# 势力与组织台账

## 超凡管理局（正面·核心）

| 项 | 内容 |
| --- | --- |
| 定位 | 国家级超凡事务主管机构；与军警协同 |
| 下属 | 特勤队（应急清剿）· 武道总队（考核/编制）· 秘境勘察部（秘境评级与开放） |
| 形象 | 专业、可靠；永不写成废物或整体腐朽 |
| 澄江站 | 地级市派出机构；沈岚所属 |
| 对民间 | 鼓励正规武道培训与报考；打击非法猎妖团伙 |

## 教育体系

| 组织 | 说明 |
| --- | --- |
| 普通高中（澄江三中） | 陆衡所在；有体育/强身选修，非专业武道校 |
| 澄江武道培训中心 | 民办正规机构；可考初级武者证；周冕任教 |
| 武道高中 / 武道大学 | 全国序列；高考双考；卷二后重点 |

## 其他势力

| 势力 | 立场 | 说明 |
| --- | --- | --- |
| 军警体系 | 正面 | 城防、封控、协同特勤 |
| 非法猎妖团伙 | 敌对 | 盗猎材料；卷02可交锋；非主角灰产线 |
| 境外散修/敌对超凡组织 | 敌对 | 不上升到抹黑他国整体 |

## 权限与资质（简）

- 无证猎杀城内妖兽：违法，由管理局处理
- 初级武者证：可接外围任务、进入部分浅层秘境外围
- 特勤实习：大学阶段后
