# 《全球迷雾：我能校正万物词条》POV视角约束

## 一、核心原则

本书默认采用：

- 第三人称
- 限定视角
- 贴近沈辞感知

也就是说，正文大多数时候应该让读者和沈辞知道差不多多的信息，而不是站在上帝视角俯瞰全局。

## 二、默认POV规则

### 1. 主POV

主POV固定为 `沈辞`。

他的视角承担以下功能：

- 观察异常
- 推演规则
- 做出选择
- 承担危险
- 逼近真相

### 2. 默认比例

全书至少 `80%-90%` 的正文，应当保持在沈辞视角附近。

### 3. 单章规则

原则上：

- 一章只保留一个核心POV
- 不在同一自然段或同一小场景里来回跳头

如果确实需要切视角，必须满足：

- 场景已经切换
- 信息目标不同
- 视角变化能够带来必要价值

## 三、前期视角限制

为了保证开篇抓力和悬疑稳定性，建议：

- 前10章只使用沈辞视角
- 前30章原则上不写完整的非沈辞POV章节
- 如果前30章真的要切，只允许非常短的补充场景，并且不能透支关键真相

## 四、允许切换POV的场景

### 1. 林照POV

适合承担：

- 城防局视角
- 医疗与秩序压力
- 沈辞看不到的救援现场

不适合承担：

- 直接替作者解释世界观
- 过早暴露林照全部情感

### 2. 顾焰POV

适合承担：

- 财团视角
- 镜像主角对照
- 强者逻辑和阶层压迫感

不适合承担：

- 过早揭穿大反派底牌
- 替读者总结剧情

### 3. 沈知微POV

适合承担：

- 失联后的异常体验
- 共鸣、失名、门域带来的陌生感
- 情感冲击

不适合承担：

- 前期大量独立展开
- 过早揭开她和失名体系的真实联系

### 4. 反派或组织POV

适合少量使用在：

- 卷尾压迫感增强
- 真相误导
- 展示更大的威胁

不适合频繁使用。  
每卷控制在 `1-2次` 以内更稳。

## 五、禁止越界的视角错误

以下情况尽量避免：

1. 沈辞不知道的事，正文却直接写成确定事实
2. 一段对话里连续写三四个人的内心
3. 为了解释设定，突然切到作者全知口吻
4. 为了制造悬疑，故意让POV知道却不说人话

## 六、切POV前必问三件事

1. 不切这个视角，这章真的写不成吗
2. 这个视角能提供什么沈辞视角给不了的价值
3. 切完以后，会不会破坏悬疑和压迫感

如果三问里有两问答不清，就不要切。

## 八、执行建议

1. 前10章把沈辞视角写稳，再考虑扩展。
2. 非沈辞POV最好带明确任务，不要只是“换个角度看看”。
3. 如果某段内容既能用沈辞写，也能用别人写，优先用沈辞。
