# 第038章 旧播控不放全句

器材间后侧还有一条更窄的楼梯。

铁扶手生了锈，台阶边全是被雨鞋踩出来的黑印，像这条线平时不走学生，也不走前厅待听的人，只走负责把声音送进各个口子的边线人员。

沈辞和林照顺着那串湿脚印往下绕了半层，广播楼后身就露了出来。

那不是一间完整机房。

更像一截被旧学校硬切出来的后勤肺叶。墙里嵌着两排玻璃窗，左边窗内堆满老式开盘机、监听喇叭和挂得发灰的耳机，右边则只留一张窄桌、一道半开的玻璃格口和一把给对象临时站定的铁脚椅。

最外沿白牌只写三行：

`旧播控不放全句`

`补听对象只认后半声`

`右窗不收自己认来的亲属`

沈辞目光停了一瞬。

第三条像是专门写给外面的人看的。

广播楼显然知道，被拖到这里的从来不只是待听对象。很多人都会沿着学号、牌子、签转单一路追过来，追到最后，站在窗外的家属和熟人本身也会变成这层流程的一部分。

17已经被带到右窗前。

知微站着，肩背还维持着刚才门前待签时那种很轻的绷紧，没有抬头，也没有往四周看。她右手食指侧面的旧创可贴已经被水泡得更皱，手里却空了，说明门前那支黑笔没跟过来。

这里暂时不认笔。

只认声。

玻璃格口下沿还开着三道细槽。

最左边贴`前声`，中间贴`后声`，最右边贴`续认`。17那张灰条正斜插在中间槽里，边角被打湿了一半，最底下还压着一小块半透明复写页，隐约能看见一道单独的竖线。

那是门前没写完的第一笔。

原来签字口那一笔，也会被送到这里。

广播楼根本没打算让任何一步真正断开。门前待签、第二层旧称、右窗补后声，只是把同一个“本人成立”拆成不同部件，换着地方往前推。

玻璃后坐着的是个头发稀疏的老男人，穿着洗得发黄的灰工作服，耳朵上挂一只单边监听。他没看知微，只翻右手边那摞窄条，声音像砂纸磨过木边：

“十七，补后声。”

“前认挂起，不补前句。”

桌边红灯亮了一下。

左窗机房里立刻传来倒带声。

那声音很慢，不像新设备按钮一按就退回去，更像有人用旧时代的办法，把一整段被剪过的磁带重新绕回接缝前头。沈辞听得太熟了，熟到后背那根弦无声一紧。

他以前在修复室里，最常处理的就是这种老带。

霉、潮、卷边、脱磁、断裂，哪一种声音会在什么时候露出来，他不需要想，耳朵自己就认得。

左窗里灯偏暗。

可还够他看清最靠外那排开盘机边，挂着一排细木牌。

`学号前声`

`旧称壳`

`家属后半`

`补名样带`

每一类下面都垂着不同颜色的引带，像把人的名字、关系和反应拆成一小段一小段，再按用途挂回墙上。

沈辞眼底一点点冷下去。

第二层男声不是凭空来的。

它有壳，有库，甚至有专门送到右窗来补后半句的续段。

知微面前那只小喇叭这时轻轻炸了一下底噪。

先出来的还是那口熟得过分的声线。

不是完整一句。

只有半截。

“……辞。”

沈辞呼吸微滞。

这次不像第二层那样完整到足够把人直接拖进家里。它先把前半句切掉了，只留最尾端那一点尾音，像故意避开最强的第一钩，只拿后半截去试对象会不会顺着补全。

知微没有反应。

可她压在玻璃下沿的那只手，指节还是很轻地白了一下。

不是想应。

是身体已经认出了这层壳。

广播楼比东口更阴的地方就在这里。它不逼你立刻开口，也不逼你立刻落笔，它先拿一小段最熟的尾音在你身上试，试你会不会自己把剩下那一半记忆补齐。

老男人抬手记了一笔，又把监听往耳边压紧了些。左窗里倒带声停了半拍，第二段很快又被送出来：

“……过来。”

还是那道男声。

还是像父亲。

可这回沈辞听清了不对。

第一个尾音出来时，底噪厚一点，像在小房间里录下的旧家用机；等到“过来”落下来，房间感却忽然薄了，像麦克风离嘴更近，连磁带摩擦的细沙音都换了一个方向。

不是同一句。

甚至不像同一盘带。

更关键的是，两段中间少了一口气。

父亲以前叫他，从不会把“小辞”和“过来”切得这么死。哪怕语气再平，中间也总会有个很轻的换气，像手上还在忙别的事，人却先把目光抬过来一下。

这里只有壳。

没有那口真正属于活人的气。

林照显然听不出这些细缝，却看见了更直接的东西。她轻轻碰了碰沈辞手背，朝左窗最底层示意了一下。

那里堆着几只旧纸箱。

其中一只没合严，露出半截卷盘外缘。卷盘纸套边上盖着淡红色借转章，已经被潮气泡开一半，还是能辨出最上面的字：

`海陵地方文书修复中心`

沈辞瞳孔微微一缩。

那是他的地方。

不是巧合。

机房里这些拿来拼旧称、补后声的样带，有一部分，是从他工作的修复中心流出来的。

而纸箱旁边还钉着一块旧夹板。

夹板上压着两页发潮的调拨单，最上头那页已经糊了，只剩下几行还能认：

`异常转录残带`

`只取前半，不放全句`

`关系样声，限夜间补听`

沈辞看着那行`只取前半`，后背一点点发冷。

第二层和右窗不是临时失真。

它们本来就只打算拿走“最像”的那半句。

而左窗里那个灰工作服男人已经抽出另一张窄条，夹到17那一栏后头。窄条顶端印着一行更细的小字：

`后声不足，转续认位`

林照顺着他目光看过去，声音压得极低：

“它在补你们家的声音。”

沈辞没答。

因为他正盯着那只露出半边的卷盘纸套。

纸套右下角，还有一串更旧的红码。

那种编号格式，他今天上午才在修复室里手写过一次。
