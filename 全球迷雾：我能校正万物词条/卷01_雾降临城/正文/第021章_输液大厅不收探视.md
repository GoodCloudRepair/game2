# 第021章 输液大厅不收探视

`内核复核序列`开启后，17号没有立刻被送进更亮的地方。

床先沿着一条向下微斜的通道走了很久。

头顶灯管越来越低，墙面也从观察二室那种冷白玻璃，慢慢变成旧医院常见的浅黄瓷砖。只是这些瓷砖都被重新打过蜡，亮得像把许多年前的病房又封回了某种更硬的壳里。

空气里的药水味越来越重。

不是急诊和病区里那种表面的消毒水味，而是长年输液、配药、金属针头和塑料软管混在一起的潮冷气。吸进肺里，会让人错觉自己不是往下一层走，而是在走进一只巨大的药瓶。

推床的人始终没说话。

直到前面最后一道门滑开，沈辞才看见所谓的`输液大厅下层`。

那不是普通大厅。

更像一片被无数输液架和透明软管撑起来的白色林子。

一排排旧儿童床被分成细窄长列，每张床头都挂着两到三只输液瓶。软管并不都接进病人手背，有些直接沿着墙边的凹槽汇进地面，又从另一端爬向大厅尽头一扇半掩的白门。

那些床之间没有帘子，也没有隔板。

可每一列又都隔得很开，像故意让床上的对象彼此看得见，却够不着。有人头偏向外侧，有人把手死死压在身下，还有几张床上只是鼓着被子，连脸都没露，像输液大厅并不在乎你是清醒、昏迷还是已经没力气动，只在乎这具对象还能不能继续挂着。

那扇门后没有光。

只有很淡的雾，从门缝里一丝一丝往外渗。

大厅正中的电子板缓慢刷着三行字：

`输液大厅不收探视。`

`未挂接对象不得靠门。`

`滴尽前不得呼叫接领人。`

沈辞看着最后一条，眼神微微一沉。

这里不说“家属”，不说“陪护”，甚至不说“接人”，只说`接领人`。像病床上的对象已经不是谁的亲人，而是某种待领回、待归档的东西。

17号床被推向最里面一列。

路过第八列时，侧边忽然传来一声压得很低的抽气。

一个中年女人半躺在床上，腕带勒得发红，眼睛却直勾勾望着大厅入口。她像终于听见了什么，嘴唇颤了一下，轻声喊了句：“我在这儿。”

声音一落，她床头那只正在缓慢滴落的瓶子立刻震了一下。

不是有人碰它。

而是瓶中液面自己轻轻晃开一圈纹，像有什么看不见的东西从里面抬头看了她一眼。

下一秒，床尾小板直接跳出新状态：

`探视误接`

两个穿灰白罩衣的人很快从另一列转过来，一人按住她肩，一人抬手掐掉了她头顶第二瓶还未完全开通的输液。女人这才像骤然醒过来，脸色一下白了，急忙摇头：“不是，我没看见人，我只是”

后半句还没说完，电子板上属于她的床号已经改成了：

`退探后挂`

床被整个调转，朝大厅更外侧推走。

她被推出去时，头顶那只瓶子里的液面仍在轻轻颤。

像刚才她喊出口的那一句“我在这儿”并没有真的散掉，而是被留在了瓶里，成了这次误接留下的残响。旁边几张床明显都听见了，却没有一个人敢接着往入口那边看，整列床像被那一声轻轻压矮了一寸。

沈辞看着那几行字，心里一点点发冷。

原来这里所谓“不收探视”，不是因为没有探视的人。

而是任何把外面的人、外面的关系、外面的“来接我了”带进大厅里的对象，都会立刻被当成挂接错误先退出来。

17号床终于停下。

有人给他换上了新的输液瓶。

瓶身上没有药名，只有一道窄窄的白签：

`17 / 挂接待定`

另一个人则把床脚灯从青色调成了更浅一点的白蓝，像17号现在还不够格靠近门，只能先挂在这一列继续等。

提送人员离开后，大厅很快恢复成之前那种压低了所有人声的安静。

可这种安静并不平。

每只输液瓶里都像藏着一小片晃动的光，随着滴速一下一下微微颤。整片大厅里看不见谁在说话，却总有些极轻的细响贴着床沿、瓶壁和软管来回走，好像有人把无数句快要出口的低语都拆碎了，灌进了这些透明液体里。

更前面的几列还不时亮起小小的床脚灯。

有的是白的，有的是浅黄，有的是偏冷的蓝。每一种灯色都对应着不同的床尾状态。沈辞很快看懂，这里和二室一样，真正支配对象命运的不是床上的人，而是外面那一整套颜色、标签和滴速。

靠近白门最里侧的一列，甚至已经看不见完整床尾，只能看见高高低低的瓶架和不断被牵紧的软管。像那片区域已经不再是病区，而是门前某种更深的供给口。

沈辞没有抬头数瓶，只借着床边金属护栏上一点反光，去摸大厅的轮廓。

门在最前面。

靠门那几列的瓶子滴得更快，床尾状态也不再是`待挂`，而是`门前观察`、`回声校准`、`接门准备`之类更深一层的字样。

知微如果已经被提前送下去，大概率就在那些位置更前的地方。

问题是17号现在还在外圈。

而外圈和内圈之间，并不只是距离。

中间至少隔着两轮筛选。先是大厅这一层的探视和挂接，再往里才轮得到回声校准、门前观察。17号如果想直接往门那边靠，不管是起身还是开口，都只会先被这片大厅看成“挂接错误”重新推出去。

正想着，头顶那只新挂上的输液瓶忽然轻轻晃了一下。

大厅里明明没有风。

可液面却自己倾了一点，映出一小块不该出现在那里的人脸轮廓。

不是他。

是一个女孩子垂下来的眼睛。

沈辞呼吸一滞，没有立刻抬头。

下一秒，瓶签背面随着瓶身转了半圈，露出极细的一行手写字：

`别看瓶里的脸`

字写得很匆忙，末尾那一撇甚至有点发抖，像留字的人当时已经快被带往更前一层，却还是硬从流程里抠出一秒，把这句话塞回了17号头顶。
