# 第013章 广播会替你认人

腕带扣上的瞬间，沈辞先感觉到的不是疼，是紧。

像有一圈看不见的字顺着塑料边缘收紧，贴着腕骨往里压，把他原本还残着的`急诊暂留`一点点盖掉。那条淡蓝腕带很快冷下去，颜色像退了一层灰，连上面的词条都变得发虚。

林照没有耽搁，直接把他按上转运床。她先压了一下床栏，确认暗扣没提前咬死，才拉过灰白床单盖到胸口，只留出一截手腕，正好让`共鸣体-17`露在外面。

“听广播，不回。”她低声说。

“看见熟人，不认。”

沈辞点了下头。

储物柜里的男孩缩在最里面，抱着那块`12床`床牌，一声不敢出。林照把半瓶葡萄糖水塞进柜角，又把门虚掩到只剩一道能透气的缝，指节在柜门上轻敲了一下，才双手握住床栏，推着沈辞往外走。

护士站外的灯比刚才更白。

像医院终于确认了该送下去的是谁，整条右侧通道都跟着亮了。原本模糊不清的箭头在沈辞眼里开始重新排列，最前方那块掉漆导向牌浮出一行新字：

`B1交接通道`

不是因为牌子变了。
是因为他现在戴着这条带子，所以终于“看得见”了。

连两侧病房门口那些原本糊成一片的护理板，也在他眼里重新分开。上面没有姓名，没有年龄，只有床号、滴速和一列列空白观察格。最底下一行则都写着同样的四个字：

`待送下层`

像从这一层往下，人已经不算“住院”，只是等待被继续归类。

床轮压过地砖，一路没有回声。两侧病房门却一扇扇半掩着，门缝里垂着输液管、病号服下摆和看不清脸的影子。有人在里面轻轻咳，有人在笑，还有一个很低的童声贴着门板说：

“哥。”

林照手上力道没变，推床的速度也没变，手腕筋却绷得很清楚。

沈辞连眼皮都没抬。

前方天花板上嵌着旧喇叭，忽然发出轻微电流声。随后，是之前在护士站里出现过的那道女声，温柔、平稳，像夜班广播真的只是在走一套普通流程。

“十七号转入，意识确认中。”

“听见请回答姓名。”

沈辞胸口微微一沉。

下一秒，左侧一扇门里忽然传出男人干涩的应声：“我叫”

只来得及三个字。

门内立刻响起床轮猛地撞墙的闷声，随后是束缚带被骤然拉紧时发出的“啪”一响。那人后半句名字没能说完，就像被什么东西整个勒回了喉咙里。

广播安静两秒，又若无其事地继续：

“十七号未答，默认静置。”

林照低声骂了句脏话，声音压得极轻，推床的手更稳了：“真拿广播认人。”

沈辞后背起了一层冷汗，却还是没出声。

通道尽头是一部老式病床电梯，门半开着，里面站着个穿深蓝工装的男人，帽檐压得很低，胸口挂着一块弯掉的牌子。牌子上的字早烂了，只剩最后两个还能勉强辨出：

`核单`

他没有看沈辞的脸，只伸手去摸床栏外那条腕带。手指碰上去的一刻，腕带下方浮出一行极淡的黑字：

`共鸣体-17·待静置`

沈辞呼吸停了半拍。

那不是他想看的词条。
像这条带子已经替他决定了该怎么看他。

更糟的是，词条浮出来那一瞬，他脑子里竟有极短的一空。不是忘记全部，只像“沈辞”这个名字突然被压远了一寸，前面顶上来的，反而是一个冰冷得毫无归属感的数字。

“陪送？”核单员声音沙得像磨过铁。

林照把那张`B1静置区交接单`存根递过去，手指压住纸角不让它抖，语气没有一点多余起伏：“送到门，交接无陪送。”

核单员接过存根，只看最下方那行`陪送身份：无`。看完，他把卡片拍回床单上，侧身让出电梯。

“下去以后，不留人。”

“知道。”林照说。她没看沈辞，目光只盯着电梯门缝。

电梯门合上时，外面的灯光被一寸寸切断。狭窄轿厢里只剩头顶一盏惨白壁灯，照得沈辞脸色比病人还淡。核单员站在脚头，始终低着头，像整个人只是流程里负责按下楼层的那只手。

负一层还没到，广播又响了。

“十七号，是否仍认得家属？”

沈辞眼睫动了一下。

林照先开口，下颌绷得很紧，声音比平时更硬，也更像在对系统说话：“无陪送，执行交接。”

广播停了一秒。

像在判断她这句话算不算回答。

随后，轿厢上方跳出一行红字：

`交接通过`

电梯继续下沉。

那短短两秒里，沈辞几乎能听见自己耳后血管发胀的声音。他很清楚，如果刚才林照多说一个字，比如他的名字，比如“我是医生”，这部电梯现在可能就不会再往下了。

“到门以后，我进不去。”林照盯着楼层数字，没看他，指尖在床栏上轻轻点了一下，“你进去先别下床。床停了，人也别先动。”

沈辞“嗯”了一声。

“广播再问，就当没听见。”

“嗯。”

“看见知微，也别第一时间喊她。”

这一次，沈辞沉默了一下，才应：“我知道。”

电梯终于一震，停住。

门开的瞬间，寒气像水一样灌了进来。

林照腕上的`陪护通行`几乎同时闪了一下，原本灰白的塑料带边缘浮出极淡的红，像已经走到它能往下送人的极限。

床头刚过门线，地面暗槽里就弹起两道金属卡扣，把床轮轻轻咬住。再往前，这张床就不再归林照推了。
她的手还扶着床栏，指节压得发白，却没再多送半寸。
像送到这里，已经是例外。
不是规矩里本该有的那种。

外面不是普通病区。是一条更长、更低的走廊，灯带嵌在地脚线里，把每张推床的轮印都照得发青。墙两侧没有病房号，只有一排排磨旧的玻璃窗，窗后垂着一样的白帘。帘子后面有人影轮廓，却没有半点翻身声，静得像全都只剩呼吸。

走廊最前方立着一扇半透明的移门。

门上没有“B1静置区”。

只有一行像后来贴上去的黑字：

`静置观察区`

而在那行字下面，一块独立床牌被夹在栏杆上，像专门放给刚到这里的人看。

牌子上只有一个数字。

`17`
