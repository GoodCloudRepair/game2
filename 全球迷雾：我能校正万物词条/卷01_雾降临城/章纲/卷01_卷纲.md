## 一、第一卷定位

- 卷名：雾降临城
- 目标字数：约25万
- 章节规模：120章
- 单章字数：不少于2000字，软目标2000-2200字，可上浮至2300字
- 预计总字数：约24.6万到25.2万
- 时间跨度：`D1-D7`
- 默认POV：沈辞

第一卷承担的任务只有四个：

1. 把灰雾、门域、失名、识名这套核心规则立住
2. 把沈辞从旁观者推成真正入局的人
3. 把妹妹线和“沈辞为什么是例外”这两条悬念线埋稳
4. 以“封住第一扇城市级灰雾门，被城防局吸纳”作为卷尾落点

## 二、第一卷已确定的关键决策

### 1. 城市与开篇地点

- 城市名：海陵市
- 开篇地点：海陵市地方文书修复中心
- 连接场景：文昌路地铁换乘站

这样开篇既能承接沈辞的职业身份，也能自然把他推入地铁、旧城区、医院三线合并的大门域。

### 2. 沈知微的第一次失联方式

不是单一原因，而是分成两段：

1. 灰雾降临时，她在文昌路换乘站附近与沈辞失联
2. 后续她被发现存在异常共鸣，被城防局特殊转移

这样前期有即时寻找压力，中期又能反转“她不是单纯被困，而是被卷入更深层事件”。

### 3. 识名第一次觉醒代价

第一次强行识名时，沈辞会出现：

- 剧烈头痛
- 听见像“命名声”一样的低语
- 短暂耳鸣和轻微耳血

这让能力从一开始就带代价，不会显得太轻。

### 4. 第一卷封门机制

第一卷卷尾封门采用组合式机制：

- 城防局的封门装置负责物理层面的稳定
- 沈知微的异常共鸣负责短暂锚定门域核心
- 沈辞通过识名读出“门”的核心词条，完成最后一步破局

这样既保留官方力量，也保留主角与妹妹的独特性。

### 第一卷中段转折

第一卷的真正中段转折点不只是“找到更多线索”，而是：

`沈辞逐渐意识到，沈知微不是单纯失踪，而是被门域和官方同时盯上的异常个体。`

这会让第一卷后半程从“单纯找人”升级成“在更大局面里找人”。

## 四、阶段详细拆解

## 第一阶段：失序开篇

- 章节：1-10
- 时间：`D1 上午至傍晚`
- 场景重心：档案馆、文昌路换乘站、旧城区地铁出口、儿童医院外围
- 阶段目标：
  - 把沈辞、沈知微、父亲线的情绪底色立住
  - 把“不要回答门里的名字”立成开篇规则
  - 让识名第一次出场并完成第一次有效破局
  - 让读者对儿童医院和沈辞的异常性产生强钩子

### 阶段推进

1. 沈辞在档案馆修复一份异常旧档，第一次碰到与父亲有关的模糊痕迹。
2. 灰雾从文昌路换乘站和地下通风系统反涌上来，城市秩序瞬间失控。
3. 沈知微在换乘站失联，留下最后一条不完整信息。
4. 沈辞被迫进入站内寻找她，并第一次直面“失名者”。
5. 识名初次觉醒，沈辞第一次靠规则理解逃出生路。
6. 他在林照出现前，先接触最小规模幸存者群体，奠定“并非只救自己”的基调。
7. 阶段尾声抵达儿童医院外围，并出现“门后有人认出沈辞”的强钩子。

### 这一阶段必须放出的信息

- 规则1：不要回应门里喊你名字的声音
- 规则2：灰雾会改变路线、广播、标识和身份指向
- 信息1：沈知微疑似被带向三号线儿童医院方向
- 信息2：沈辞的识名并非普通觉醒，某些失名者会对他产生反向异常

### 关系推进

- 兄妹线：从日常依赖快速切入强失联压力
- 沈辞 / 幸存者：从旁观，到第一次承担临时决策
- 沈辞 / 林照：仅做前置信息铺垫，不急着深入感情线

### 阶段尾钩子

沈辞在儿童医院外，通过识名看向一具尸体时，对方反过来叫出了他的名字。

## 第二阶段：儿童医院

- 章节：11-24
- 时间：`D1 傍晚至D1 夜间`
- 场景重心：旧儿童医院、急诊楼、输液大厅、地下储药库
- 阶段目标：
  - 正式引入林照
  - 建立“医院门域”的规则性恐怖
  - 让沈辞确认沈知微曾经到过这里，但已被进一步转移

### 阶段推进

1. 儿童医院已经和正常建筑脱节，部分区域进入门域化，时间、病历、身份反复重排。
2. 林照带着一个小型医疗救援点坚守，缺药、缺电、缺秩序，但仍在维持伤员和孩子。
3. 沈辞与林照因是否立刻深入医院核心产生摩擦，随后通过共同救人建立初步信任。
4. 医院内出现“被重新命名的病人”“只认手环不认人的护士站”等规则型危险。
5. 沈辞多次使用识名，代价逐渐加重，但也越来越能看懂局部规则。
6. 在病案室和转运记录里，沈辞查到沈知微被标成`共鸣体-17`，并由白防护组转移。
7. 阶段尾声，医院深层广播提到“门需要新的锚点”，把妹妹线抬到更高层次。

### 这一阶段必须放出的信息

- 规则3：在医院门域里，名字会被病历和腕带替换
- 信息3：城防局或某支官方力量已经比沈辞更早发现沈知微
- 信息4：沈知微的异常不是普通幸存者级别，而是“共鸣体”

### 关系推进

- 沈辞 / 林照：从互相试探进入临时合作
- 沈辞 / 沈知微：从“她在哪里”升级为“她为什么会被特殊标记”

### 阶段尾钩子

沈辞在医院地下储药库里看见一份转运单，上面有父亲早年参与过的项目代号。

## 第三阶段：旧城追索

- 章节：25-38
- 时间：`D1 傍晚至D1 深夜`
- 场景重心：旧居民区、殡仪街、社区文化馆、废弃公交总站
- 阶段目标：
  - 把城市地图从地铁和医院扩到旧城区
  - 建立幸存者生存压力和物资逻辑
  - 让沈辞第一次听到外部官方广播

### 阶段推进

1. 沈辞和林照离开医院，按转运路径追向旧城区。
2. 旧城区的危险不是单一怪物，而是“失名后被固定习惯支配的人群和建筑”。
3. 沈辞帮助幸存者建立一个短暂避难点，也第一次开始不是为了自己一个人做决定。
4. 他通过识名看出公交总站、社区门禁、殡仪街的不同规则，体现能力的连续破局价值。
5. 第一次官方广播接入，提到临时收容点、封锁线和“严禁接近文昌-中心医院-旧城三域重叠区”。
6. 阶段尾声拿到新线索：沈知微曾被转往回澜中学临时收容点。

### 这一阶段必须放出的信息

- 规则4：失名者会被自己生前最强执念固定
- 信息5：文昌站、儿童医院、旧城区并不是三个独立灾点，而是正在合并
- 信息6：城市里存在仍在运作的半官方秩序

### 关系推进

- 沈辞 / 林照：林照开始相信沈辞不是碰巧，而是真的能看懂规则
- 沈辞 / 幸存者群像：第一次建立主角的“别人愿意跟着他走”的基础

### 阶段尾钩子

回澜中学传出的旧广播里，出现了一句只在沈家内部用过的称呼。

## 第四阶段：回澜中学

- 章节：39-70
- 时间：`已写部分（039-060）为 D1 23:23 至 D2 01:49；061-070 承接学校线，继续以时间轴回写为准`
- 场景重心：中学校园、操场临时棚、教学楼、考试门域
- 阶段目标：
  - 做第一卷最重要的一次学生群像救援
  - 让兄妹线得到第一次情感爆点回收
  - 让沈辞从“找妹妹”走向“开始承担更多人”
  - 完成学校线从广播、补名、待尾到名单抹名的整段收束

### 阶段推进

1. 回澜中学表面是收容点，实际上教学楼内部被异化成“考试规则场”。
2. 许多学生和老师被困在不同楼层，无法随意说话、离座或改写答案。
3. 沈辞靠识名读出规则关键，带队破掉“铃声、座位、缺考”三层限制。
4. 他救下一批学生，也在这里第一次得到明显的“他做对了，所以别人活了下来”的回报。
5. 沈知微没有留在学校，但留下手账、录音或一张故意写错的校门通行单。
6. 线索显示她被转往体育馆收容点，但转运理由不是“学生”，而是“门域共鸣观察对象”。
7. 阶段尾声，沈辞在校广播室找到一句录音：`哥，别让它替你命名。`

### 这一阶段必须放出的信息

- 规则5：某些门域会把“社会秩序”异化成硬规则
- 信息7：沈知微在失联后仍保有较强自我意识
- 信息8：她知道比沈辞更多的异常

### 关系推进

- 沈辞 / 沈知微：虽然未见面，但情感连接大幅拉高
- 沈辞 / 林照：林照第一次看到沈辞为了群体主动站到前面

### 阶段尾钩子

体育馆收容点的官方名单里，沈知微的名字被整个抹掉了。

## 第五阶段：官方收容点

- 章节：71-80
- 时间：`承接回澜中学后续，以时间轴回写为准`
- 场景重心：体育馆收容点、城防局临时封控线、器材仓、监测车
- 阶段目标：
  - 正式引入城防局一线体系
  - 让沈辞和“官方秩序”建立复杂关系
  - 父亲线第一次明显上浮

### 阶段推进

1. 体育馆并不是完全安全的地方，它只是更有秩序，也更有筛选。
2. 城防局已经注意到沈辞的路线判断能力，试图短暂征用他协助定位失控区域。
3. 沈辞在体育馆收容数据库里查不到妹妹，却找到一条被加密覆盖的转移日志。
4. 通过林照的医疗权限和城防局残缺数据，他第一次接触到“城市级灰雾门”的概念。
5. 一份旧档出现父亲代号，说明十二年前类似事件可能发生过前兆。
6. 沈辞开始意识到，自己不是只被灾难卷入，而是可能从一开始就站在某条旧线索上。
7. 阶段尾声，文昌站、中心医院和旧城区的重叠区出现第一次全城级异动。

### 这一阶段必须放出的信息

- 信息9：城防局并不全知，也在摸索
- 信息10：父亲曾接触过与灰雾、命名相关的前置研究
- 信息11：妹妹被转入“非公开处理链”

### 关系推进

- 沈辞 / 城防局：从被动配合走向短期互用
- 沈辞 / 林照：从临时合作变成可以背靠背做事的关系

### 阶段尾钩子

监测车上的仪器在沈辞靠近时自动跳出一串不属于任何已知序列的词条编号。

## 第六阶段：财团与旧档

- 章节：81-90
- 时间：`承接官方收容点后续，以时间轴回写为准`
- 场景重心：旧档案库、河道管廊、财团前哨点、三域边缘
- 阶段目标：
  - 引入财团线与顾焰
  - 确认沈辞的异常远超普通觉醒
  - 把第一卷后半程彻底推向门域核心

### 阶段推进

1. 城防局和财团都在抢一份旧档与一件门域遗物。
2. 沈辞因为识名能力，被迫卷进双方争夺。
3. 顾焰短暂登场，不需要抢走戏份，但必须足够强、足够危险、足够记住。
4. 沈辞在旧档案库里找到父亲留下的隐语痕迹，确定父亲曾经试图阻止某种“命名流程”。
5. 财团仪器对沈辞的能力产生异常反应，首次暗示他可能不是普通的编目师序列。
6. 三域边缘开始不稳定扩张，门域核心即将彻底暴露。
7. 阶段尾声，沈辞得到最后一条明确线索：沈知微就在三域重叠核心。

### 这一阶段必须放出的信息

- 信息12：财团知道一些城防局不知道的旧研究
- 信息13：沈辞的识名能力有被“识别”回去的风险
- 信息14：父亲留下的线索并不是引导他逃，而是引导他看见

### 关系推进

- 沈辞 / 顾焰：建立第一次对照和张力
- 沈辞 / 城防局：互相依赖加深，但不完全信任

### 阶段尾钩子

在进入三域重叠区前，沈辞识名看见自己的影子上浮出了一行并不属于“沈辞”的词条。

## 第七阶段：门域核心

- 章节：91-102
- 时间：`待按时间轴回写`
- 场景重心：文昌站-中心医院-旧城区重叠核心、深层门域、灰雾心脏区
- 阶段目标：
  - 找到沈知微
  - 炸开第一层“沈辞为什么特殊”的真正悬念
  - 让第一卷从求生升级到承担城市级责任

### 阶段推进

1. 三域重叠区内部不再遵守普通空间逻辑，车站、病房、街道像被拼接在一起。
2. 沈辞识名能力进入第一卷最密集的高压使用区，代价显著上升。
3. 门域核心并不是单纯怪物巢穴，而是一个不断试图“重新命名一切”的结构。
4. 沈知微仍活着，但她的异常共鸣正在被核心利用。
5. 沈辞救到妹妹，却发现只带她离开会加速核心失控。
6. 一段父亲留下的音频或规则残痕，首次说明“门不是靠蛮力关的，得先读懂它是什么”。
7. 阶段尾声，城防局决定启动封门方案，但必须有人留下完成识别与锚定。

### 这一阶段必须放出的信息

- 信息15：门域核心像在“整理”人和物，而不是只想毁掉它们
- 信息16：沈知微与门域的共鸣能短暂稳定核心
- 信息17：沈辞的能力与“正确命名”存在深层联系

### 关系推进

- 沈辞 / 沈知微：第一卷真正重逢，但重逢立刻附带更大代价
- 沈辞 / 林照：林照看见沈辞愿意留下承担城市级风险

### 阶段尾钩子

沈辞在核心里读出一行词条：`海陵市第一灰雾门·未完成编目`。

## 第八阶段：封门之夜

- 章节：103-120
- 时间：`待按时间轴回写`
- 场景重心：核心外圈、封门阵地、灰雾门本体、城市封锁线
- 阶段目标：
  - 完成第一卷最大高潮
  - 让沈辞在不正式掌握改名的前提下，完成识名级别的极限爆发
  - 以城市级代价换来阶段性胜利

### 阶段推进

1. 城防局启动封门装置，但外部怪潮、门域回涌、财团干扰同时发生。
2. 沈知微成为短暂锚点，她必须撑住，沈辞必须读懂门。
3. 沈辞在极高代价下，用识名读取整扇门的核心词条和缺口。
4. 某一瞬间他会出现“近似改名”的异常征兆，但这一卷不把它稳定下来。
5. 林照负责救援和外圈秩序，城防局付出明显伤亡，顾焰或财团势力完成短暂但有代价的协作。
6. 沈辞完成最后的识别与指向，让封门装置命中正确“名字”。
7. 灰雾门主体关闭，海陵市保住核心城区，但旧城区西侧和地下深段列入长期封控。
8. 卷尾，沈辞与沈知微幸存，知微有源位后遗症，城防局以临时证将沈辞纳入监管借用。

### 这一阶段必须放出的信息

- 信息18：封门不是物理摧毁，而是重新锁定它的正确命名
- 信息19：沈辞的异常远未解释完，只是第一次被大规模验证

### 关系推进

- 沈辞 / 城防局：从被观察对象转为必须争取的人
- 沈辞 / 沈知微：兄妹重聚，但妹妹身上的后遗症明确留下

### 卷尾最终钩子

封门完成后，沈辞在废墟里看到一张新的转移名单。  
名单最底部只剩一个简短备注：

`编目程序已重启。`

## 五、第一卷的能力成长控制

第一卷只做三件事：

1. 把识名写出层次
2. 把识名的代价写出真实感
3. 埋下改名的异常征兆，但不正式稳定掌握

### 第一卷内识名成长节奏

- 1-10章：第一次觉醒，靠直觉和痛感触发
- 11-24章：开始能主动聚焦单一目标词条
- 25-38章：能从环境中抓重点规则，而不只是看怪物
- 39-70章：能持续拆解“名字 / 位置 / 来源”这类社会规则门域
- 71-90章：能辅助群体行动、路线判断、封控决策
- 91-120章：在门域核心进入极限识名状态，完成卷尾大破局

## 六、第一卷主要情绪线

### 兄妹线

从“失联焦虑”推进到“线索回收”，最后落到“重逢即更大代价”。

### 沈辞的成长线

从“我只想找到妹妹”推进到“我不得不参与决定这座城能不能活下去”。

### 悬念线

第一卷只回答“沈辞确实是例外”，不回答“他为什么是例外”。

## 七、第一卷结尾状态

第一卷结束后，必须稳定留下这些结果：

- 沈辞和沈知微活下来（已完成）
- 沈知微出现明确后遗症或异常残留（已完成：源位灰痕、完整姓名和完整亲称反应迟钝）
- 海陵市没有完全毁灭，但旧城区被重创（已完成：旧城西侧和地下深段长期封控）
- 城防局注意到沈辞，并准备将他纳入体系（已完成：城防局临时证 / 监管借用）
- 财团也记住了沈辞（已完成：顾焰复核标记未撤）
