## 第031章 临设只认交接单

- 章节功能：把`澜中临设`从学校外沿推进到实际交接口，并确认17号已经先入`操场北棚`
- 核心冲突：沈辞和林照必须在不被当成`自己下来的人`写进交接口流程的前提下，摸清17号在临设里的第一落点
- 主要剧情：沈辞与林照跟着`四号短驳`摸进澜中临设侧签口，确认17号早在`21:12`就先入`操场北棚`，眼前这趟车送来的只是补回执
- 信息点：`临设只认交接单 / 改签件不走正门 / 交接口不收自己下来的人`；17号已于`21:12`先入北棚；`北棚今晚不见家属，不收补认`
- 章末钩子：北棚旧广播直接报出`二排十七号，待核，不见家属`

## 第032章 北棚不见家属

- 章节功能：把17号从“北棚外的广播提示”推进到“北棚内的具体空床、具体去向和颜色分流”
- 核心冲突：沈辞必须在`北棚不见家属`的规则下，尽可能接近17号位，又不能让自己被棚内秩序认成探视者
- 主要剧情：沈辞潜进北棚摸到17号空床，确认知微`21:12`先入棚、现已被提往东口复听，同时看清北棚是按颜色和“落班”往下分人的暂置棚
- 信息点：`北棚不见家属 / 待核位不答探视 / 换床先看床尾牌`；17号于`21:12`先入北棚；状态为`复听待提`；北棚内存在蓝/白/红/灰的颜色分流；黑板写着`未落班，不入教室`
- 章末钩子：知微纸片和灰牌都把路线指向`东口`

## 第033章 班牌不要写本名

- 章节功能：把东口灰牌线具体化为“班牌 + 卡槽 + 去向”的学校分班流程
- 核心冲突：沈辞不能正面拿牌挂到自己身上，又必须尽快看懂灰列如何被认、如何被通知进楼
- 主要剧情：沈辞与林照在旧旗杆分牌桌前看懂班牌、卡槽和东口通知逻辑，确认17号已被写进灰列东口待听顺序
- 信息点：`班牌不要写本名 / 未落班先挂颜色 / 借牌不进东口`；铁架卡槽才是实际登记口；灰列对象会被写入`东口待听`；知微留有`别抬头看二楼喇叭`提示
- 章末钩子：广播楼东口老喇叭开始积起沙响

## 第034章 广播先叫学号

- 章节功能：正式揭开广播楼东口复听规则，并把17号推进到“门前待签”这一步
- 核心冲突：沈辞不能直接出声、不能正面入列，却必须在短时间里看懂第二声到底怎么认人
- 主要剧情：沈辞在广播楼东口亲眼看懂第二声如何把学号拖回旧名，并确认17号状态已是`灰列 / 二次复听 / 不归班列`，只差门前待签
- 信息点：`广播先叫学号 / 复听期间不纠正称呼 / 第二声不应旧名`；复听不仅记回应，也记抬头等本能反应；17号状态为`灰列 / 二次复听 / 不归班列`；门前还差一道签字
- 章末钩子：白牌立起`不要替十七号签字`

## 第035章 不要替十七号签字

- 章节功能：揭开17号签字口的真正逻辑，并把第二层男声与父亲旧录音声线硬接起来
- 核心冲突：沈辞必须理解为什么17这一签绝不能被别人代落，同时还要忍住不在第二层男声响起前抢先乱动
- 主要剧情：沈辞贴近签字口后确认，17号这一步真正要逼的是“本人笔认”，代签会被直接写成代认，同时知微留下`别听男声第二层`警告
- 信息点：`不要替十七号签字 / 复听对象只认本人落笔 / 旁听人不留原名`；灰皮册按`学号 / 笔认 / 签入人 / 层`分栏；17号行写着`本人笔认后进二听 / 代签作代认`；知微留下`别听男声第二层`
- 章末钩子：第二层男声以与父亲旧录音极像的起声方式出现

## 第036章 别答喇叭里的小名

- 章节功能：让第二层男声真正落下，并第一次用只属于沈家的称呼反向钩住沈辞
- 核心冲突：沈辞必须在确认门内坐着的就是知微的同时，硬压住自己对那声`小辞`的本能回应
- 主要剧情：沈辞透过内窗认出门前待签位上的人就是知微，并亲耳听见第二层男声喊出只在沈家内部使用过的`小辞`
- 信息点：`别答喇叭里的小名 / 第二层只听旧称 / 未签者坐门前`；17号此时还没落笔；知微的手势与创可贴坐实身份；第二层男声用的是与父亲旧录音高度相似的壳；`小辞`只在沈家内部使用
- 章末钩子：男声进一步说出`小辞，过来`

## 第037章 别替喇叭认亲

- 章节功能：打断第二层旧称诱导，把17号从门前待签硬转进`旧播控右窗补听`
- 核心冲突：沈辞必须在不应声、不闯门的前提下，打断第二层完整放送，保住知微那一笔没有真正写完
- 主要剧情：沈辞在第二层旧称已经钓到门里门外的边缘，用右声道回授强行打断二听，让17号从门前待签改挂`缓签复排`并转去旧播控右窗
- 信息点：第二层不只钓门内对象，也会听门外真正相关者；器材侧窗下红针会记录近场反应；知微留下的`别听男声第二层`本质是“别让这一层完整放完”；二听受扰后会触发`复排后听 / 缓签复排 / 右窗补听 / 补听前不再呼名`
- 章末钩子：知微用墨迹箭头把线继续指向`旧播控右窗`

## 第038章 旧播控不放全句

- 章节功能：揭开旧播控右窗只补“后半声”的机制，并把修复中心残带正式接进学校广播楼后台
- 核心冲突：沈辞必须在不暴露自己的前提下，弄清父亲似的声音为什么到了右窗只剩“后半句”
- 主要剧情：沈辞与林照追到旧播控右窗，发现这里专门用`后半声`补对象记忆，第二层父亲似男声也只被拆成最像的半句残壳
- 信息点：`旧播控不放全句 / 补听对象只认后半声 / 右窗不收自己认来的亲属`；门前第一笔会被一起送到右窗；样带分成`学号前声 / 旧称壳 / 家属后半 / 补名样带`；父亲似的声音在右窗里缺少同一口气；修复中心残带被调进来并写着`只取前半，不放全句`
- 章末钩子：沈辞看见纸套上的编号格式与自己上午在修复室里使用的完全同源

## 第039章 第二笔才算本人

- 章节功能：拆开右窗“第二笔本人续认”逻辑，并揭出教学楼后续`入列 / 补名`两条分支
- 核心冲突：沈辞必须在不出手硬抢的前提下，弄清第二笔为什么比第一笔更危险，以及知微一旦被续认成功会被送去哪
- 主要剧情：沈辞看清右窗真正要补的不是一句声音，而是17号的第二笔本人续认；只要知微落下这一下，就会被直接送进教学楼的`入列`秩序
- 信息点：`前笔挂起 / 本人续认 / 第二笔生效`；透明薄片保存第一笔复写；`本人笔认只收第二笔 / 前笔不断，不得退号 / 补听成立，再续全号`；右窗已备好`续认完成 / 直送入列`与`样声退回 / 转补名`两套路；更深处存在`西教学楼二层 / 夜间缺考列`
- 章末钩子：沈辞认出拼父亲声壳那卷带子边缘还贴着修复中心临时试接用的检修色条

## 第040章 样声里少了一口气

- 章节功能：让沈辞用自己的修复经验精准拆穿样带接缝，把17号从右窗续认硬转去`缺考补名`
- 核心冲突：沈辞必须在不触发正面对抗的前提下，把右窗这次补后声做成一场流程内部认定的失败
- 主要剧情：沈辞用自己对旧带气口和接缝的判断，让右窗拼接样声在过轮时自行露馅，17号因此被判`样声不全`并改送`西教学楼一层 / 缺考补名处`
- 信息点：父亲真声和样带之间差的不是字，而是同一口气；林照为沈辞掩掉探手动静；`旁听过门`小条可当临时拨片；右窗一旦判`样声不全`，会同步写下`关系样退库 / 本人续认中止 / 前笔封存 / 转补名`；左窗人明确提到`文修那边送来的东西`
- 章末钩子：17号被旧教工雨衣接走，续送单去向改为`西教学楼一层 / 缺考补名处`
