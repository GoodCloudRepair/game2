# 《全球迷雾：我能校正万物词条》卷01_001-010章章纲

## 使用说明

- 默认POV：沈辞
- 单章字数：2000-2100字
- 时间范围：`D1 上午至傍晚`
- 当前状态：正文已完成，本文件保留为卷一开篇执行版细纲
- 开篇目标：
  - 抓住读者
  - 让灰雾规则迅速落地
  - 让识名第一次出场就有价值
  - 让“找妹妹”和“沈辞为什么是例外”两条线同时启动

## 第001章 没有编号的档案

- 预计字数：2050字
- 时间：`D1 10:40-11:00`
- 地点：海陵市地方文书修复中心
- 章节功能：开场、立主角、埋父亲线、埋规则钩子
- 核心冲突：日常秩序正在崩坏，但沈辞还没意识到严重性

### 主要剧情

1. 交代沈辞的职业和气质，他在修复一份没有编号、来源异常的旧录音档。
2. 旧档内容混乱，只能勉强听清一句类似警告的话：`不要回答门里的名字。`
3. 沈辞收到沈知微的消息，她说补习结束，准备从文昌路站过来和他汇合。
4. 文书中心外部信号、钟表和广播同时出现轻微异常，但大部分人都当成普通故障。
5. 结尾时，修复台上的旧录音在无人操作的情况下自行播放，且声音像是父亲留下的。

### 本章信息点

- 沈辞的工作和性格
- 父亲失踪留下的情绪底色
- “不要回答门里的名字”作为第一条关键规则

### 结尾钩子

旧录音里传出一句完整话：`如果雾提前来了，别去站里找名字。`

## 第002章 雾从地铁口上来

- 预计字数：2050字
- 时间：`D1 11:00-11:20`
- 地点：修复中心、地下通道入口
- 章节功能：灾变爆发、兄妹线启动
- 核心冲突：沈辞想保持冷静，但外部秩序开始极速崩塌

### 主要剧情

1. 文昌路站方向传来混乱人流和警报，修复中心灯光失真，名字牌开始出现错字和重影。
2. 灰雾并不是从天空压下来，而是从通风井、地下口、走廊尽头缓慢倒灌。
3. 沈知微发来最后一条语音，说站里广播正在反复念她的名字。
4. 中心内第一起异化事件出现，有人因为回应了雾中的呼唤而失踪。
5. 沈辞决定逆着人流进入地下通道，去文昌路站找沈知微。

### 本章信息点

- 灰雾降临方式
- 回应“名字”会出事
- 沈知微最后已知位置在文昌路站

### 结尾钩子

地下通道入口上方原本写着“文昌路换乘站”的牌子，已经变成了：`请从正确的名字进入。`

## 第003章 别应门里的名字

- 预计字数：2050字
- 时间：`D1 11:20-11:40`
- 地点：文书中心地下通道、换乘站服务走廊
- 章节功能：建立第一波规则恐怖
- 核心冲突：沈辞必须在不懂规则的情况下活着靠近车站

### 主要剧情

1. 沈辞和几名临时幸存者结伴前进，包括保安、地铁值班员和一名抱着孩子的女人。
2. 通道里不断传来熟人或家属的声音，试图诱使他们回头、答话或开门。
3. 有人忍不住应声，整个人被灰雾后的门缝“吞掉”，只留下一只鞋。
4. 沈辞第一次意识到旧录音不是疯话，而是真规则。
5. 他带着剩下的人绕进服务走廊，逼近真正的换乘站。

### 本章信息点

- 规则1正式成立：不要回应门里的名字
- 灰雾会借熟悉的声音诱导

### 结尾钩子

服务走廊尽头的电子钟全部停在同一个时间：`11:17`

## 第004章 第一个词条

- 预计字数：2050字
- 时间：`D1 11:40-12:00`
- 地点：文昌路换乘站外围
- 章节功能：识名第一次觉醒
- 核心冲突：沈辞第一次面对真正的失名者

### 主要剧情

1. 一名看上去只是受惊的乘客突然异化，反复说“我要回家”，扑向有名字回应的人。
2. 沈辞在极度危险中第一次看见浮动词条：`失名乘客·回家`
3. 识名带来剧烈头痛、耳鸣和一种像谁在替世界重新命名的低语。
4. 沈辞抓住词条里的执念，利用站内指示牌和列车广播误导怪物方向。
5. 他带着剩下的人逃过第一轮扑杀，第一次确认自己能“看见别人看不见的东西”。

### 本章信息点

- 识名首次觉醒
- 失名者并非纯随机怪物，而是被执念固定
- 识名的代价真实存在

### 结尾钩子

怪物倒下后，地上掉出一张沈知微的地铁月票夹。

## 第005章 她在三号线

- 预计字数：2050字
- 时间：`D1 12:00-12:20`
- 地点：换乘站闸机区、三号线入口
- 章节功能：给出追踪方向
- 核心冲突：沈辞要在继续救人和继续追妹妹之间做取舍

### 主要剧情

1. 月票夹证明确实是沈知微的，里面夹着她今天写的便签。
2. 沈辞通过残缺广播和乘车信息，推断沈知微上了通往旧儿童医院方向的三号线支线。
3. 他顺手救出被困在闸机区的两名学生，第一次出现“不是只救妹妹一个人”的选择。
4. 他发现站内某些门不能随便开，`员工通道`反而比`紧急出口`更安全。
5. 他决定沿三号线方向继续追。

### 本章信息点

- 沈知微曾往三号线方向移动
- 识名不只对怪物有用，也能帮助判断环境规则

### 结尾钩子

闸机尽头的灰雾上，像被手指写出了一行新字：`不要来东端，门在学我的声音。`

## 第006章 灰雾里的检票员

- 预计字数：2050字
- 时间：`D1 12:20-13:00`
- 地点：三号线旧站台、废弃检票区
- 章节功能：第一次真正规则战
- 核心冲突：沈辞必须用规则而不是蛮力过关

### 主要剧情

1. 一名已经半失名的旧站检票员守在通道里，违反“检票规则”的人都会被拖回雾里。
2. 幸存者里有人想强冲，导致伤亡，沈辞第一次承担“你们先别动，听我的”。
3. 他通过识名读出检票员核心规则不是“杀人”，而是“验票后放行”。
4. 沈辞利用站务废票、旧员工章和顺序欺骗，带众人通过检票区。
5. 他第一次真正尝到识名破局的主动收益。

### 本章信息点

- 规则型怪物可以被“顺着规则欺骗”
- 沈辞开始承担临时带队职责

### 结尾钩子

通过检票区后，出站口不再通往街道，而是直接通向一条通往旧城区的黑色楼梯。

## 第007章 地上的医院

- 预计字数：2050字
- 时间：`D1 13:00-13:40`
- 地点：旧城区出口、临时救护点
- 章节功能：从地下转到地面，引入林照
- 核心冲突：沈辞需要新线索，但地面秩序同样已经坏了

### 主要剧情

1. 沈辞一行人从旧城区出口爬上地面，发现白天像被压成黄昏。
2. 旧儿童医院方向有大量伤员和混乱人流，一个临时救护点正在勉强运转。
3. 林照首次出场，正在抢救一个因回应“亲人名字”而精神失控的孩子。
4. 林照不信任来历不明的人，沈辞则只想要妹妹的线索，二人第一次正面碰撞。
5. 一名学生幸存者提供信息：他见过一个像沈知微的女孩，被穿白色防护服的人带往医院。

### 本章信息点

- 林照登场
- 地面并不比地下安全
- 白防护组第一次被提及

### 结尾钩子

林照拿出一张临时分诊单，上面沈知微的名字已经被人用红笔划掉。

## 第008章 不要被重新命名

- 预计字数：2050字
- 时间：`D1 13:40-14:20`
- 地点：临时救护点、废弃药房
- 章节功能：强化“命名”主题，推进林照关系
- 核心冲突：想救人，就得进入已经变得不讲理的药房

### 主要剧情

1. 林照需要药，沈辞需要医院入口信息，两人决定短暂结伴。
2. 废弃药房里的药名、标签和存放柜会不断变化，普通人几乎无法分辨。
3. 沈辞用识名帮林照找到真正可用的药和错误命名的危险物。
4. 他在高压使用识名后再次听到低语，仿佛有人称他为`编目员七号`。
5. 林照第一次正面意识到沈辞的能力不是普通直觉。
6. 两人在药房后台找到一张转运单，沈知微的名字被改成了`共鸣体-17`。

### 本章信息点

- “重新命名”不仅发生在人身上，也发生在物品上
- 共鸣体-17第一次出现

### 结尾钩子

转运单上写着下一站：`旧儿童医院住院楼负一层。`

## 第009章 旧城天桥

- 预计字数：2050字
- 时间：`D1 14:20-15:20`
- 地点：旧城区天桥、医院外圈道路
- 章节功能：推进伙伴关系，补外部秩序信息
- 核心冲突：去医院的路被灰雾和失名群挡住

### 主要剧情

1. 地面道路已经被雾吞掉，只能从老旧天桥和高架连廊绕向医院。
2. 沈辞在路上又救下少量幸存者，但开始学会只救“能救的人”，体现克制。
3. 林照问沈辞到底看见了什么，沈辞只部分坦白，二人建立初步信任。
4. 一具疑似城防局成员的尸体被发现，身上有医院区域图和收容点标识。
5. 地图证实儿童医院已被划为`A级异常区`，且早于公众认知。

### 本章信息点

- 城防局已提前介入
- 儿童医院是重点异常区
- 沈辞和林照关系从利用转向合作

### 结尾钩子

医院外墙上，本该固定的楼名正在一点点变成：`第三收容门`

## 第010章 门后有人认出他

- 预计字数：2050字
- 时间：`D1 15:20-16:00`
- 地点：旧儿童医院外围、住院楼入口
- 章节功能：前10章收束，打出强悬念
- 核心冲突：沈辞终于抵达关键地点，但发现事情比“找妹妹”大得多

### 主要剧情

1. 医院外部结构已经开始门域化，楼层、窗户、通道不再对应正常空间。
2. 沈辞通过识名判断一条暂时可进的路径，和林照进入住院楼前厅。
3. 前厅里横陈着被拖行过的尸体和废弃转运设备，明显发生过官方级别行动。
4. 沈辞对一具半失名尸体使用识名，试图读取它死前状态。
5. 尸体却在识名反应中短暂睁眼，直勾勾地看向沈辞，说出一句不该有人知道的话。

### 本章信息点

- 医院不是普通灾点，而是门域核心的重要边缘
- 沈辞的能力会被某些存在“反向识别”

### 结尾钩子

那具尸体张嘴说道：  
`编目师……你终于回来了，沈辞。`

## 前10章完成后的效果检查

前10章完成后，读者应该明确知道：

1. 灰雾不是普通灾难，而是有规则的异常
2. 沈辞的能力很特殊，但有代价
3. 沈知微不是普通失联者，她被卷进了更深层事件
4. 林照已经进入主线，但不是感情挂件
5. 儿童医院会是第一卷前半段的第一个大场
6. 沈辞身上藏着比“找妹妹”更大的问题
