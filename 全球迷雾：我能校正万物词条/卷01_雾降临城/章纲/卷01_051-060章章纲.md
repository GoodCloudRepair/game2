## 第051章 旧称不收自己报

- 章节功能：把旧名室的第一层处理逻辑坐实，并明确`小辞`这张旧称白签已经先挂到了`沈`的外格
- 核心冲突：沈辞必须贴着门缝看清旧名室到底先收哪一种称，同时压住自己在`小辞`被挂出来那一刻的本能反应
- 主要剧情：旧名室第一步不是逼人认亲，而是先用`自报旧称`规则筛反应，再把广播楼带来的`小辞`白签正式挂进`沈`的外格
- 信息点：`旧称不收自己报 / 门外不替交 / 先挂旧称，再核来处`；自报旧称会被压到后核；旧名室把`沈`拆成内外两格；`小辞`来自东口流程，被先挂到`沈`外格
- 章末钩子：旧名室已经不只是等知微交旧称，而是先把门外的沈辞挂进了`沈`的外格

## 第052章 门外称先挂外格

- 章节功能：坐实旧名室的内外两层挂格结构，并把`沈外格`正式推进到靠窗核来处这一步
- 核心冲突：沈辞必须尽快弄清`小辞`这一格现在到底危险到什么程度，是否已经等同于自己被旧名室认定
- 主要剧情：旧名室坐实“门外称只先挂外格、不替内认”，知微左边内格因旧位未清暂空，`沈外格`则被推进到侧窗核来处
- 信息点：`门外称先挂外格 / 外格只存，不替内认 / 外格靠窗，不应第二遍`；外格分主称层与旁格层；知微因`旧位未清`导致左边内格仍暂挂不上；`沈外格，靠窗核来处`
- 章末钩子：旧名室的下一步，已经不是逼知微，而是隔着侧窗继续往门外摸沈辞

## 第053章 主称不能借别人叫

- 章节功能：坐实`主称不能借别人叫`这条关键限制，并把`小辞`从能直接收人的危险称呼暂时压成`借声挂存`
- 核心冲突：沈辞必须在不靠窗、不应声的前提下，判断`小辞`这张签现在还有没有办法被旧名室直接用来收自己
- 主要剧情：侧窗规则坐实`主称不能借别人叫`，`小辞`因属于广播楼带入旧声而被压成`借声挂存`，旧名室随即改推`先并来源`
- 信息点：`主称不能借别人叫 / 借叫只留旁格 / 借声不作主称 / 带入只挂外格`；顾姓女生门外那声被压成`借叫`；`小辞`因源于广播楼旧声而只能记作`借声挂存`；旧名室决定`先并来源`
- 章末钩子：`小辞`虽然没能直接坐成主称，却被继续带向“并来源”这一层

## 第054章 并来源前不翻沈字

- 章节功能：把旧名室的“来源并合”效果真正亮出来，并明确`沈`因为左格空缺与旧位不清而暂时无法在这一层并成一处
- 核心冲突：沈辞必须看明白“来源并”到底意味着什么，以及第49章拖出来的`旧位不清`现在还有没有继续挡住知微
- 主要剧情：旧名室用周姓对象展示“来源并成就会露尾并被立刻取走”，而`沈`因左格空缺与旧位不清暂时并不成，最终被改记`未并走东回廊 / 外应先送`
- 信息点：`并来源前不翻沈字`；另一姓对象来源并合后露出尾角并被取走；`旧位不清`让`沈`左格仍挂不上；`未并走东回廊`与`旧生未定，外应先送`
- 章末钩子：旧名室决定不在这一层继续耗`沈`，而是把外应先转去东回廊

## 第055章 外应先走东回廊

- 章节功能：正式亮出东回廊这条外应转送线，并把沈辞从“挂在门外”推进到“已有一只空夹在等他”
- 核心冲突：沈辞必须面对一个比之前都更明确的现实，即学校已经不满足于在纸面挂他，而是在物理路径上给他备好了下一格位置
- 主要剧情：东回廊白单与压着`辞`字的空黑夹被提前备好，喇叭直接点出`沈外应，候并`，学校第一次在真实路径上迎接沈辞
- 信息点：`东回廊 / 外应先走 / 内格暂存 / 并称不过门`；空黑夹先压`辞`字；`来处不在门内者，走东`；播报`沈外应，候并`；`过东廊者，不回旧名室门前`
- 章末钩子：门里知微还在，门外属于沈辞的东回廊空夹已经挂出来了

## 第056章 回头的人算退件

- 章节功能：正式让东回廊从“被点亮的新路径”变成“沈辞必须亲自走进去的单向线”
- 核心冲突：东回廊要求沈辞一旦跟入就不能再按来路退回，而知微又还被留在旧名室里
- 主要剧情：`辞`字空黑夹自行启动，东回廊以`回头的人算退件`逼沈辞正式离开旧名室门前，并与林照分成内线 / 外窗两条平行线
- 信息点：`外应先走 / 陪行不过内线 / 回头的人算退件`；林照走外窗维修道平行跟线；`辞`字空黑夹先自行启动
- 章末钩子：第一拐角翻出新的动态规则`空夹先过 / 人后贴线 / 不得并行`

## 第057章 空夹先过人后贴线

- 章节功能：把东回廊的动态跟线规则具体化，并解释为什么沈辞不能和自己的空夹并着走
- 核心冲突：沈辞既要继续跟住自己的流程夹，又不能离它太近，否则会被直接按`到位`或`并件`处理
- 主要剧情：东回廊中段坐实`并夹作到位`与`并件口`规则，沈辞看见其他姓氏外应夹与`右窗补退`等边线残片仍在继续往里送
- 信息点：`人不得并夹 / 并夹作到位`；`并件口`；其他外应夹如`何 / 周 / 顾`；`右窗补退`等边线来源继续被东回廊承接
- 章末钩子：走廊尽头出现`候并格`，并亮出规则`候并不站本位`

## 第058章 候并不站本位

- 章节功能：正式揭开`候并格`的结构，并明确“候并”不是站进自己的位置，而是站在旁边等被扣进去
- 核心冲突：候并格试探沈辞会不会自己站进`本位`，而那一步一旦踩错就可能直接省掉缓冲层
- 主要剧情：候并格把`辞`字空夹挂成本位，把沈辞本人逼去`沈外应代线`，并正式吐出只认`姓 / 称 / 来处`的`来处单`
- 信息点：`候并不站本位 / 外应先立代线 / 并称不过门`；`本位 / 代线 / 退挂`三类站位；`借声挂存者，先核来源`
- 章末钩子：抽屉弹出`来处单`，并写明`现名不过柜`

## 第059章 外应只听半个姓

- 章节功能：把`来处单`正式展开，并让修复中心 / 父亲线通过`文修前声`这一硬线索再次抬头
- 核心冲突：沈辞不能把完整现名递进柜里，只能沿着`半个姓 + 来源`这条更危险的材料线继续往里拆
- 主要剧情：候并格来处单亮出`文修前声`与文修中心风格红章，学校后台、旧播控与修复中心前史被硬接成同一条来源线
- 信息点：`外应只听半个姓 / 全称作自认 / 来处先于现人`；`东口旧声带入 / 右窗补退后挂 / 门外称留存 / 文修前声`；文修中心章型再次出现
- 章末钩子：喇叭播报`沈外应，前移，文修前声，候核`

## 第060章 并门前先交来处

- 章节功能：把东回廊这一小段推进到`并门前`，并正式打开更深的`来源柜 / 文修线`
- 核心冲突：沈辞不能把自己或完整现名送进并门，只能先把`来处`递进去，让学校自己把更深来源线翻出来
- 主要剧情：并门前柜墙坐实`样声不过门`与`先交来处`，沈辞没有把自己递进去，而是借规则逼出更深一层`来源柜`
- 信息点：`并门前先交来处 / 样声不过门 / 无源称退挂`；`文修 / 家录`等来源柜分类；文修格式窄盒与档案封套
- 章末钩子：右侧小门翻出`文修前声，请入来源柜`
