# 卷01 阶段02（031-060章）大纲

## 阶段定位

- 章节范围：031-060
- 对应内容：旧城追索收束（031-038）+ 回澜中学前中段（039-060）
- 当前状态：正文已完成
- 时间跨度：以`08_时间线约束与时间轴.md`为准

## 阶段主任务

1. 把追索重心从“物流后车线”切到“学校规则场”
2. 完成广播楼、补名室、待尾区、旧名室的层层推进
3. 让兄妹线第一次进入“同姓 / 后字 / 旧称”这些更私密、更危险的命名层
4. 让沈辞从单纯找妹妹，进一步升级为读懂并撬动一整套社会规则门域
5. 让林照与沈辞的搭档进入高度默契、可无声协作的阶段

## 起点与终点

- 起点状态：17号线已进入学校外沿，沈辞与林照刚摸到临设 / 北棚 / 东口线
- 终点状态：沈辞被东回廊 / 候并格 / 来处单一路往里接走，自己也被正式写进学校流程

## 三段推进

### 一、031-040章：学校外沿与广播楼

- 临设、北棚、分牌桌、东口待听顺序逐步坐实
- 广播楼用学号、旧称、样声和续认规则层层逼近17号
- 沈辞第一次正面承受“家族记忆被规则调用”的冲击
- 他靠声音、设备和职业经验把17号从本人续认线上硬拖去补名线

### 二、041-050章：补名室与旧名室前段

- 西教学楼补名口建立“先认位置、后认名字”的教楼秩序
- `沈`姓先被抬出来，后字与尾卡成为真正危险核心
- 教学楼不再只处理知微，开始反向套住沈辞
- 旧名室打开后，`沈姓两位`被拆成门内旧生 / 门外外应两半

### 三、051-060章：旧称、来处与候并链

- 旧称层证明学校不只认名字，还会拆主称、借声、外格和来源
- `小辞`被挂进外格，主角身份第一次在纸面流程中被明确定义
- 旧名室把知微与沈辞拆成不同流程位，但又强行归到同姓线下
- 东回廊、并件口、候并格、来处单等动态流程建立起“人会被一步步送成文件”的压迫感

## 本阶段必须兑现的信息

- 规则5：某些门域会把社会秩序异化成硬规则
- 学校不只认学生身份，还会把“姓 / 学号 / 后字 / 旧称 / 来源”拆成多层流程
- 沈知微在失联后仍保有极强自我意识，并持续给沈辞留线
- 沈辞本人不再是纯追索者，而是被学校规则场反向接管的对象之一

## 人物推进重点

- 沈辞：从追人变成追规则、追列次、追座位、追旧称来源
- 沈知微：从沿路留痕升级到和沈辞“隔空协同改流程”
- 林照：从高压搭档升级到“现实侦察 + 边线支撑 + 情绪制动”的稳定支点

## 阶段爆点

- 第038章：旧播控右窗与修复中心调拨线并轨
- 第043章：17号只被允许先写下`沈`
- 第050章：旧名室先收半个人
- 第060章：来处单坐实，主角正式被流程往内层接走
