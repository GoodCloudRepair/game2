# 第086章：连带项不单独入域

## 1. 本章定位

- 所属卷：卷01 雾降临城
- 所属阶段：阶段03 / 财团与旧档起步（进入三域前沿）
- 章节功能：用失败实验坐实中位活线不能被清成单独项，沈辞成为原链成立条件
- POV人物：沈辞
- POV类型：近景限知

- 本章结束时间：D2 04:39

## 2. 本章目标

- 本章要推进的主线层级：保护知微 / 主角身份线 / 世界观格局线
- 本章核心冲突：前置口尝试把沈辞从中位清成“单独项”，而17柜与样带系同时出现不稳
- 这章结束后局面具体变化：系统正式给出`连带项不单独入域 / 中位保留`
- 这章会让读者对谁产生新的认知：对沈辞在链上的不可替代性、对杜衡对局面的再判断、对前置口比人更冷的处理方式
- 本章必须交代的信息：连带活项不是陪同者，而是原链里缺了就断的一节
- 本章新增物品/能力/规则/伏笔：`清单未成 / 原链复位`、`连带项不单独入域`、`中位保留`
- 本章结尾钩子：前沿廊深门打开，只收同链三节
- 本章风格目标：用抽项失败造成的设备反应来落地规则，而不是由角色讲解
- 本章对应卷目标：把沈辞从“被借用”推进到“被链条正式需要”

## 3. 本章剧情梳理

- 开场状态：旧借图已提示`中位留活`，前置口进入校位阶段
- 事件一：桥底翻出`清单独项 / 保链，后校位`，准备先把中位活线剥出
- 事件二：沈辞被半抽离时，17柜、样带系同时出现偏移与回收征兆
- 事件三：前置口记录`清单未成 / 原链复位`，并正式保留中位
- 结尾结果：前沿廊开启，条件变成`只收同链三节`
