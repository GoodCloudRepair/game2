- 所属卷：第一卷

- 章节功能：把`联络单`这一层正式立住，并确认学校真正要核的是“知微在校内需要被带走时，最后谁会来接”

## 2. 本章目标

- 本章要推进的主线层级：从“知微在学校里留下过`哥`这一笔”推进到“学校确认真正会到场来接她的人，也是`哥`”
- 本章核心冲突：沈辞不能沉进情绪，只能在多张旧联单里看清学校真正记录的是谁实际到过、谁能换来放行
- 这章结束后局面具体变化：学校不再只承认`哥`是一个留笔称呼，而是把它进一步核成一个真实到场、真实接人的外接位置
- 这章会让读者对谁产生新的认知：对知微在校内遇事时的默认联络方向、对学校流程收人的实际逻辑、对父称线当前为何仍被压后
- 本章必须交代的信息：`联络单先核谁来接 / 家属栏后压 / 空来作失联`；`哥`在`来接人`栏被反复记成真实到校者
- 本章新增物品/能力/规则/伏笔：蓝皮联络单、`来接核过 / 外接作真 / 转回听格`翻牌、`回听格`棕色小牌
- 本章结尾钩子：`回听格`翻出，下一步开始核“知微真正会对什么称呼起应”
- 本章风格目标：把“关系”继续落到冷纸证和放行痕迹上，而不是直接抒情
- 本章对应卷目标：继续收束学校线，把兄妹线从留笔层推进到校内现实接应层

## 3. 本章剧情梳理

- 开场状态：蓝皮`联络单`已经被推出，上一章刚核过`旧生留笔`
- 事件一：联络单亮出`先核谁来接 / 家属栏后压 / 空来作失联`
- 事件二：沈辞翻到`沈`姓相关旧联单，发现正式家属栏被压后，真正反复被门岗章认下的`来接人`始终是`哥`
- 事件三：他将带有门岗回签的底联送入细口，联络层翻出`来接核过 / 外接作真 / 转回听格`
- 结尾结果：棕色`回听格`小牌升起，流程从“谁会来接”转向“她真正认哪一种称”
