# 第049章：不要替十七排旧位

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：在`知 / 辞`双来源压桌之后，把17号从`旧位成`的临界线拖成`旧位不清`
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 00:38
- 本章结束时间：D2 00:44
- 时间跨度：6分钟
- 与上一章间隔：0分钟
- 地点：二层旧位核对长桌、墙角旧座次图前、长桌外白线区域
- 目标字数：2000-2100字
- 实际字数：2001字

## 2. 本章目标

- 本章要推进的主线层级：从“沈姓旧生 / 外应双来源成立”推进到“17号旧位未定，被继续送向旧名室”
- 本章核心冲突：沈辞必须在不能出声、不能靠近、不能代指的前提下，阻止知微在二层把旧位先认下来
- 本章必须交代的信息：`排旧位不许代指 / 靠窗靠门都算认位 / 旧位不清，送旧名室`；线外不代位；旁声也会被当作`旁声作证`；林照扶正压片后露出`门位调换，以后册为准`与斜改线，使靠门与中列变得无法明确判断；最终17被判`旧位不清`
- 本章结尾钩子：17号被记去向`旧名室`
- 本章风格目标：写一场几乎没有动作、但每个人都不能多说半句的极静高压对抗
- 本章对应卷目标：让沈辞和林照第一次在二层核位桌前完成一场“只靠不说话也能改结果”的配合

## 3. 本章剧情梳理

- 开场状态：知微已经站到核位桌前，`知 / 辞`双来源压在她眼前
- 事件一：长桌规则与顾姓女生的`代指作代排`说明任何多嘴都可能成为核位证据
- 事件二：沈辞确认自己既不能说、不能靠、也不能踩线，只能等规则里出现真正的模糊地带
- 事件三：林照扶正旧座次图透明压片，露出`门位调换，以后册为准`与斜改线，让靠门 / 中列变得无法判清
- 结尾结果：老女人不敢代排，17被判`旧位不清`，送往旧名室
