# 第044章：二铃之前只点姓

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：把一号补名室从“先写姓”推进到“二铃前只点姓”，并正式抬出同姓未齐的锁与`沈`姓下的`知 / 辞`尾卡
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 00:11
- 本章结束时间：D2 00:16
- 时间跨度：5分钟
- 与上一章间隔：0分钟
- 地点：西教学楼一号补名室课桌区、后柜抽屉区、窗边红纸空桌区
- 目标字数：2000-2100字
- 实际字数：2004字

## 2. 本章目标

- 本章要推进的主线层级：从“17号先写入`沈`姓”推进到“补名室会按姓把人收进格子里，并在第二铃前用同姓未齐锁住后字”
- 本章核心冲突：沈辞必须在不暴露自己也被`沈`姓卷入的前提下，看懂同姓与后字之间的关系，并确认知微后面那一半名字究竟将如何被点
- 本章必须交代的信息：二铃前女老师只点姓，不点全名；`周姓未齐，只点前，不点尾`；`只认点到的姓，不替自己挑后字`；前字簿按姓记账；后柜抽屉按姓存放尾卡；`沈`字签下同时压着`知`和`辞`；窗边白卷题头是`夜间缺考列`，且带有`未齐`红框
- 本章结尾钩子：夜间缺考列的白卷与`沈`下并排的`知 / 辞`同时露出，说明名字层的危险还要继续往上走
- 本章风格目标：把补名室写成一间会先按姓记账、再慢慢补全人的冷房间，让规则更像文书流程而不是单纯惊吓
- 本章对应卷目标：让回澜中学的“补名”逻辑进一步贴到兄妹线核心，并为下一章的小规模规则对抗做准备

## 3. 本章剧情梳理

- 开场状态：17号已经把`沈`姓写进补名条，正处于`待尾`状态
- 事件一：女老师在第二铃前逐列点姓，补名室所有人都只能以姓回应列次，不得提前碰后字
- 事件二：`周姓未齐`的示范让沈辞看懂，同姓只要有一边未齐、迟到或待补，另一边就不能先并尾
- 事件三：在林照的提醒下，沈辞看见后柜抽屉按姓存放尾卡，并在`沈`字签下发现`知`和`辞`
- 结尾结果：窗边白卷露出`夜间缺考列`与`未齐`红框，说明补名室之后还有更深的夜间名单在等
