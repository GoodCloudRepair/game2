# 第059章：外应只听半个姓

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：把`来处单`正式展开，并让修复中心/父亲线通过`文修前声`这一硬线索再次抬头
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 01:36
- 本章结束时间：D2 01:42
- 时间跨度：6分钟
- 与上一章间隔：0分钟
- 地点：候并格第三排来处抽屉前、外窗平行观察位
- 目标字数：2000-2100字
- 实际字数：2025字

## 2. 本章目标

- 本章要推进的主线层级：从“候并格开始核来源”推进到“`文修前声`成为沈辞这一线的明确来处标签”
- 本章核心冲突：沈辞不能把完整现名递进柜里，只能沿着`半个姓 + 来源`这条更危险的材料线继续往里拆
- 本章必须交代的信息：`外应只听半个姓 / 全称作自认 / 来处先于现人`；`东口旧声带入 / 右窗补退后挂 / 门外称留存 / 文修前声`；文修中心章型再次出现
- 本章结尾钩子：喇叭播报`沈外应，前移，文修前声，候核`
- 本章风格目标：让父亲线和修复中心线以制度化材料痕迹出现，而不是直接人物揭秘
- 本章对应卷目标：把学校后台与修复中心前史更硬地接上，为后续更深线索提供转门

## 3. 本章剧情梳理

- 开场状态：沈辞站在候并格`代线`内，抽屉里弹出`来处单`
- 事件一：他读出`外应只听半个姓 / 全称作自认 / 来处先于现人`，确认此层只认拆开的姓与来源
- 事件二：来处单第三列露出`东口旧声带入 / 右窗补退后挂 / 门外称留存 / 文修前声`，并出现文修中心风格的半散红章
- 事件三：沈辞与林照共同确认学校后台正通过某条物料链调用修复中心相关的前声材料
- 结尾结果：来处单被流程认可，黑夹补出新条`并门前，核来处`，喇叭播报`文修前声，候核`
