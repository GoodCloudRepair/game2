# 第033章：班牌不要写本名

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：旧城追索
- 章节功能：把东口灰牌线具体化为“班牌 + 卡槽 + 去向”的学校分班流程
- POV人物：沈辞
- POV类型：主POV
- 时间：D1
- 本章开始时间：D1 22:10
- 本章结束时间：D1 22:21
- 时间跨度：11分钟
- 与上一章间隔：0分钟
- 地点：澜中临设旧旗杆分牌桌、东口长廊前、广播楼东口外沿
- 目标字数：2000-2100字
- 实际字数：2002字

## 2. 本章目标

- 本章要推进的主线层级：从“知道去东口”推进到“看懂东口要先过班牌分列和卡槽登记”
- 本章核心冲突：沈辞不能正面拿牌挂到自己身上，又必须尽快看懂灰列如何被认、如何被通知进楼
- 本章必须交代的信息：`班牌不要写本名 / 未落班先挂颜色 / 借牌不进东口`；铁架卡槽才是实际登记口；灰列对象会被写入`东口待听`；知微留有`别抬头看二楼喇叭`提示
- 本章结尾钩子：广播楼东口老喇叭开始积起沙响
- 本章风格目标：把学校分牌写成比北棚更“像学校、也更不像学校”的冷分班流程
- 本章对应卷目标：继续延迟真正的教学楼核心，只揭开灰列外壳

## 3. 本章剧情梳理

- 开场状态：沈辞和林照从北棚撤出，沿棚架和操场边绕向东口
- 事件一：两人在旧旗杆分牌桌看到班牌规则，确认这里的牌子不写本名，只写颜色、列、口和次
- 事件二：沈辞发现铁架卡槽才是真正的登记点，谁被插进哪一列，哪一列后面的门灯和喇叭就会提前知道
- 事件三：他从废旧胸牌与知微留言里确认`17 / 东`对应灰列东口，并拿到`广播维护`灰臂带作为边线掩护
- 结尾结果：两人盯着灰列对象往`广播楼东口`过去，老喇叭开始启动
