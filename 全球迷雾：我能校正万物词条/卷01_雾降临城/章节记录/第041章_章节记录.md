# 第041章：缺考的人先补名字

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：正式把17号线从广播楼推进到西教学楼，并立起“先认位置、再认名字”的补名秩序
- POV人物：沈辞
- POV类型：主POV
- 时间：D1
- 本章开始时间：D1 23:42
- 本章结束时间：D1 23:55
- 时间跨度：13分钟
- 与上一章间隔：0分钟
- 地点：旧播控后窄道、西教学楼侧门、补名桌外沿、一号补名室入口
- 目标字数：2000-2200字（可上浮至2300字）
- 实际字数：2150字

## 2. 本章目标

- 本章要推进的主线层级：从“17号被转补名”推进到“教学楼真正以补名、入座、缺考列的秩序接手对象”
- 本章核心冲突：沈辞必须在不惊动补名桌的前提下，尽快读懂西教学楼这一层到底先认什么，才能决定下一步怎么继续跟进去
- 本章必须交代的信息：`缺考的人先补名字 / 未落座前不问来处 / 借名者不得入列`；补名桌名录没有姓名栏，只认楼层、室号、列、号；黑板写`一铃入座 / 二铃封题 / 缺考补名，不得回头`；这里比广播楼更先认位置；林照摸到`考务后勤`蓝布外套和`一号补名室列签`
- 本章结尾钩子：第一铃后，17号被带向`一号补名室`，更深处真正的课桌秩序已经露出轮廓
- 本章风格目标：让教学楼的恐怖从“声音诱导”切到“安静、座位、列次和缺考”
- 本章对应卷目标：回澜中学核心秩序正式起步，但仍只打开第一层门

## 3. 本章剧情梳理

- 开场状态：17号从旧播控后送线被带往西教学楼侧门
- 事件一：沈辞与林照看见侧门白牌和地面分列胶带，确认教学楼先按“缺考 / 补名 / 入列”分对象
- 事件二：两人观察补名桌，发现名录先认座次和列号，名字反而滞后，且17号目前只被写成`待补`
- 事件三：林照摸到`考务后勤`外套和`一号补名室列签`，为继续潜入教楼留出壳；知微则主动踩进更安全的中偏右列
- 结尾结果：第一铃响起，活动牌翻出`第一铃后，缺考者入一号补名室`，17号被带往真正课桌秩序前
