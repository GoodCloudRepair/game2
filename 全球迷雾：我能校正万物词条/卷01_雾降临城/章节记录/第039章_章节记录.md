# 第039章：第二笔才算本人

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：拆开右窗“第二笔本人续认”逻辑，并揭出教学楼后续`入列 / 补名`两条分支
- POV人物：沈辞
- POV类型：主POV
- 时间：D1
- 本章开始时间：D1 23:19
- 本章结束时间：D1 23:31
- 时间跨度：12分钟
- 与上一章间隔：0分钟
- 地点：旧播控右窗检修缝、左窗机房、右窗续认台
- 目标字数：2000-2200字（可上浮至2300字）
- 实际字数：2131字

## 2. 本章目标

- 本章要推进的主线层级：从“旧播控补后半声”推进到“右窗真正要补的是17号的第二笔本人续认”
- 本章核心冲突：沈辞必须在不出手硬抢的前提下，弄清第二笔为什么比第一笔更危险，以及知微一旦被续认成功会被送去哪
- 本章必须交代的信息：`前笔挂起 / 本人续认 / 第二笔生效`；透明薄片保存第一笔复写；`本人笔认只收第二笔 / 前笔不断，不得退号 / 补听成立，再续全号`；右窗已备好`续认完成 / 直送入列`与`样声退回 / 转补名`两套路；更深处存在`西教学楼二层 / 夜间缺考列`
- 本章结尾钩子：沈辞认出拼父亲声壳那卷带子边缘还贴着修复中心临时试接用的检修色条
- 本章风格目标：把危险从“会不会应声”进一步收紧成“哪一笔会把你正式写进去”
- 本章对应卷目标：开始把学校真正的教楼秩序和广播楼流程接成一体

## 3. 本章剧情梳理

- 开场状态：沈辞已确认样带来自修复中心，继续贴在检修缝里观察右窗
- 事件一：他看见透明薄片和复写层，明白门前第一笔已经被保存，右窗这里只差第二笔就能把17完整写回本人
- 事件二：左窗灰工作服男人当场用剃须刀片和接带胶拼接样带，坐实父亲似声音是人工维护出来的后台材料
- 事件三：林照替沈辞摸到`西教学楼一层 / 缺考补名处`续送单；沈辞又看见更深一层的`西教学楼二层 / 夜间缺考列`
- 结尾结果：拼接样带露出修复中心检修色条，证明它甚至没清干净人工试接痕迹
