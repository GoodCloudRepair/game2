# 第091章：入核口不收完整人

## 1. 本章定位

- 所属卷：卷01 雾降临城
- 所属阶段：阶段04 / 门域核心起步
- 章节功能：把核心真正的入场条件钉成“只收未写圆的人”
- POV人物：沈辞
- POV类型：近景限知
- 时间：D2 清晨前
- 本章开始时间：D2 05:02
- 本章结束时间：D2 05:09
- 时间跨度：约7分钟
- 与上一章间隔：紧接第090章
- 地点：核心外窗前 / 入核口外缘
- 目标字数：2000-2100字
- 实际字数：2000

## 2. 本章目标

- 本章要推进的主线层级：保护知微 / 主角身份线 / 世界观格局线
- 本章核心冲突：入核口拒收完整身份，沈辞必须主动退掉门外挂识和完整关系
- 这章结束后局面具体变化：沈辞被白片接受为`活 / 在`，并获得进入“先校物后校人”过渡口的资格
- 这章会让读者对谁产生新的认知：对沈辞“不完整”的价值、对顾焰理解规则的深度、对林照在关键口的克制
- 本章必须交代的信息：入核口不收完整人
- 本章新增物品/能力/规则/伏笔：`入核口 / 不收完整人`、`完整旧称，退回外窗`、`先退门外挂识`、`活 / 在`
- 本章结尾钩子：侧缝亮出`先校物 / 后校人`
- 本章风格目标：让“放弃完整身份”写成冷流程选择，而不是热血宣言
- 本章对应卷目标：正式把沈辞从窗外观察者推进到可入核的候选项

## 3. 本章剧情梳理

- 开场状态：知微已经被推进`无名后入内圈`，窗下弹出入核白片
- 事件一：白片亮出`入核口 / 不收完整人`
- 事件二：顾焰说明核心不收能在门外自证完整的人，沈辞主动摘下`临时协查`并退掉旧权限压痕
- 事件三：白片认下沈辞剩余状态为`活 / 在`
- 结尾结果：侧缝开启，进入下一层“先校物后校人”
