# 第051章：旧称不收自己报

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：把旧名室的第一层处理逻辑坐实，并明确`小辞`这张旧称白签已经先挂到了`沈`的外格
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 00:50
- 本章结束时间：D2 00:55
- 时间跨度：5分钟
- 与上一章间隔：0分钟
- 地点：旧名室门口、门缝外、门边黑板前
- 目标字数：2000-2100字
- 实际字数：2080字

## 2. 本章目标

- 本章要推进的主线层级：从“旧名室先收半个人”推进到“旧名室先挂旧称，再分门里门外两格”
- 本章核心冲突：沈辞必须贴着门缝看清旧名室到底先收哪一种称，同时压住自己在`小辞`被挂出来那一刻的本能反应
- 本章必须交代的信息：`旧称不收自己报 / 门外不替交 / 先挂旧称，再核来处`；自报旧称会被压到后核；旧名室把`沈`拆成内外两格；`小辞`来自东口流程，被先挂到`沈`外格
- 本章结尾钩子：旧名室已经不只是等知微交旧称，而是先把门外的沈辞挂进了`沈`的外格
- 本章风格目标：把旧名室写成一间先钩反应、再慢慢分类的冷处理屋，而不是直接逼人认亲的高潮房
- 本章对应卷目标：继续把学校线从补名、待尾、旧位推进到更私密的旧称层

## 3. 本章剧情梳理

- 开场状态：旧名室在门内点出`沈姓两位，先交旧称`
- 事件一：周姓男生因脱口而出“早没人那么叫我了”被记`自报旧称`，旧名室的第一条硬规则被现场坐实
- 事件二：沈辞看见旧名室门边的玻璃瓶、黑板内外格和`旧称不收自己报 / 门外不替交 / 先挂旧称，再核来处`
- 事件三：老太婆从`外应带入`抽屉里抽出`小辞`白签，直接挂进`沈`的外格
- 结尾结果：知微内格仍空，沈辞却已经被旧名室先挂住了外格
