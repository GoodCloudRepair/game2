# 第047章：待尾区不收并肩人

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：立住二层待尾区按姓隔位、按未齐记账的规则，并把对面的空`沈`位真正写成威胁
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 00:27
- 本章结束时间：D2 00:32
- 时间跨度：5分钟
- 与上一章间隔：0分钟
- 地点：二层待尾区走廊、待尾长椅区、旧柜与小黑板前
- 目标字数：2000-2100字

## 2. 本章目标

- 本章要推进的主线层级：从“17号送上二层”推进到“二层按姓隔位，并明确存在另一张待补的`沈`牌”
- 本章核心冲突：沈辞必须在不能惊动待尾区执行层的前提下，尽快读懂这里到底在等什么齐
- 本章必须交代的信息：`待尾区不收并肩人 / 同姓先隔位 / 未叫到不看对面`；同姓不并肩也不照面；黑板记`到位 / 未齐`；`未齐不过册`；对面的`沈`牌下压着`待补位`
- 本章结尾钩子：老女人抱出`旧位册`，直接点到`沈姓，先核旧位`
- 本章风格目标：把二层写成比一层更安静、更像档口候件区的冷秩序
- 本章对应卷目标：继续把学校名字链从“补姓”推进到“等齐再过册”

## 3. 本章剧情梳理

- 开场状态：17号刚被送入二层待尾区
- 事件一：沈辞看见待尾长椅、白线、姓牌与天花铅坠，确认同姓对象被强制隔开
- 事件二：周姓与林姓的微小越线被立刻压回，坐实`并肩作并`与`同姓先隔位`
- 事件三：沈辞从黑板上的`到位 / 未齐 / 未齐不过册`与对面那张`沈`牌下的`待补位`看懂，知微当前最危险的是等另一个`沈`补齐
- 结尾结果：`旧位册`被搬出，二层流程从等待过渡到正式核位
