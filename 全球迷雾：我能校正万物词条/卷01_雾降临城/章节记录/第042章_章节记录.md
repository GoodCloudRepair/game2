# 第042章：一号补名室不收本名

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：让沈辞与林照借`考务后勤`和列签潜入一号补名室，并正式立住“补名只照座签抄”的规则
- POV人物：沈辞
- POV类型：主POV
- 时间：D1-D2
- 本章开始时间：D1 23:55
- 本章结束时间：D2 00:06
- 时间跨度：11分钟
- 与上一章间隔：0分钟

- 目标字数：2000-2100字

## 2. 本章目标

- 本章要推进的主线层级：从“17号被送进一号补名室”推进到“补名室真正认的是旧座签，而不是对象自己报出来的名字”
- 本章核心冲突：沈辞必须在不暴露潜入身份的前提下，看懂一号补名室怎样把人和旧座位残名绑在一起，同时继续跟住知微
- 本章必须交代的信息：`一号补名室不收本名 / 补名只照座签抄 / 写满者先入空卷`；黑板写`先坐后抄 / 未点到列不落第一字 / 空卷不改名`；补名室用的是旧年级旧班列裁下来的座签残名；写自己完整本名会被判入`空卷`；中右列当前比左列`空卷待收`更安全；17号桌签下先露出的是`沈`
- 本章结尾钩子：知微补名桌角的黑纸掀起后，17号座签下露出沈家的姓
- 本章风格目标：把教学楼的恐怖从“流程口”继续推进到“座位认名”，让危险更安静、更规整
- 本章对应卷目标：正式把教学楼写成“社会秩序异化”的第一层具体形态

## 3. 本章剧情梳理

- 开场状态：第一铃刚过，沈辞与林照拿到`考务后勤`外套和`一号补名室列签`，顺着侧门潜进补名室
- 事件一：沈辞看见门牌、黑板和三列课桌结构，确认这里不收对象自报名字，而是只认旧座签缺口
- 事件二：一名中列女生按座签成功补字，另一名左列男生抢写本名，被当场判`写满 / 空卷`
- 事件三：沈辞从`空卷待收 / 待补 / 迟到补列`三色夹和知微的落列选择中，看懂中右列暂时更安全
- 结尾结果：女老师叫到17号并掀开座签黑纸，底下露出`沈`
