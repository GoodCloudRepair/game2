# 第057章：空夹先过人后贴线

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：把东回廊的动态跟线规则具体化，并解释为什么沈辞不能和自己的空夹并着走
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 01:24
- 本章结束时间：D2 01:30
- 时间跨度：6分钟
- 与上一章间隔：0分钟
- 地点：东回廊中段、并件口前、候并格外沿
- 目标字数：2000-2200字（可上浮至2300字）
- 实际字数：2202字

## 2. 本章目标

- 本章要推进的主线层级：从“进入东回廊”推进到“看清东回廊如何把空夹和活人故意分出半步”
- 本章核心冲突：沈辞既要继续跟住自己的流程夹，又不能离它太近，否则会被直接按`到位`或`并件`处理
- 本章必须交代的信息：`人不得并夹 / 并夹作到位`；`并件口`；其他外应夹如`何 / 周 / 顾`；`右窗补退`等边线来源继续被东回廊承接
- 本章结尾钩子：走廊尽头出现`候并格`，并亮出规则`候并不站本位`
- 本章风格目标：把压迫感写成“你连和自己的位置走平都不被允许”
- 本章对应卷目标：继续深化学校线的流程精密感，让追线感强于房间重复感

## 3. 本章剧情梳理

- 开场状态：沈辞刚过东回廊第一拐角，看到`空夹先过 / 人后贴线 / 不得并行`
- 事件一：沈辞在反字竖牌与林照外窗提醒下，确认`人不得并夹 / 并夹作到位`
- 事件二：他略微跟平空夹时，左墙`并件口`立即弹开竹夹，几乎把人和夹一并收口
- 事件三：沈辞看见`何 / 周 / 顾`等其他外应夹，以及`旧口带入 / 门外称挂存 / 右窗补退`等来处标签，确认东回廊在承接学校前层边线残片
- 结尾结果：沈辞跟着`辞`字空夹走到候并格前，规则翻出`候并不站本位`
