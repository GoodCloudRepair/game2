# 第050章：旧名室先收半个人

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：打开`旧名室`这一层，并把二层的尾字、旧位问题继续压进`旧称`与`来源未并`的新危险里

## 2. 本章目标

- 本章要推进的主线层级：从“17号旧位不清”推进到“旧名室开始同时处理`旧生`与`外应`两种`沈`来源，并要求先交旧称”
- 本章核心冲突：沈辞必须继续贴着门缝看清旧名室的处理逻辑，同时压住自己在`旧称`层面被点出来的本能
- 本章必须交代的信息：17号被加挂`位失`灰扣；`旧名室先收半个人 / 未核旧位，不翻全名 / 先认缺口，再认人`；门边写`尾未定者，先交旧称`；旧名室内收集乳名、小字、旧喊法、门外称；`沈`下被拆成`旧生 / 外应`两类来源；`来源未并，不翻全名`；屋内直接报出`沈姓两位，先交旧称`
- 本章结尾钩子：旧名室不仅在等知微补完后字，也在等沈辞把自己的“旧称”一起交进去
- 本章风格目标：把学校线从“名字拆字”再推进到“旧称拆人”，让危险更贴近兄妹和沈辞本人
- 本章对应卷目标：把学校深层规则从二层待尾进一步推进到旧名室，为下一段更深的身份层对抗打开门

## 3. 本章剧情梳理

- 开场状态：17号被判`旧位不清`，准备转送旧名室
- 事件一：知微被加挂`位失`灰扣，瘦男人带她沿深走廊进入旧名室门口
- 事件二：沈辞从灰卡、门牌和旧木框看懂，旧名室先收的是“半个人”，而且会先核旧称
- 事件三：沈辞透过门缝看见`沈`下分列`旧生 / 外应`，并听到屋内直接报`沈姓两位，先交旧称`
- 结尾结果：学校已把知微和门外的沈辞一起拖进旧名室的旧称流程
