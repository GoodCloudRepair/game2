# 第034章：广播先叫学号

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：旧城追索
- 章节功能：正式揭开广播楼东口复听规则，并把17号推进到“门前待签”这一步
- POV人物：沈辞
- POV类型：主POV

- 本章结束时间：D1 22:33

- 目标字数：2000-2100字

## 2. 本章目标

- 本章要推进的主线层级：从“抵达东口”推进到“弄清17号已被排入二次复听并即将门前待签”
- 本章核心冲突：沈辞不能直接出声、不能正面入列，却必须在短时间里看懂第二声到底怎么认人
- 本章必须交代的信息：`广播先叫学号 / 复听期间不纠正称呼 / 第二声不应旧名`；复听不仅记回应，也记抬头等本能反应；17号状态为`灰列 / 二次复听 / 不归班列`；门前还差一道签字
- 本章结尾钩子：白牌立起`不要替十七号签字`
- 本章风格目标：把广播楼写成“用学号壳包着旧称诱导”的收音筛选口
- 本章对应卷目标：把学校线从分班口推进到真正开始“听”的地方

## 3. 本章剧情梳理

- 开场状态：沈辞与林照顺着灰列对象潜到广播楼东口长廊
- 事件一：两人看见东口规则板，确认广播楼先用学号固定对象，再用第二声试探旧名反应
- 事件二：沈辞现场看见七号撑住、五号仅因抬头就被记作二次，读懂这里不只记口头回应，还记本能动作
- 事件三：他从顺序页和副单上确认`17 / 灰列 / 二次复听 / 不归班列`，且`17到门前再签`
- 结尾结果：毛玻璃门后冒出与无编号旧录音带相似的旧磁带底噪，白牌立起`不要替十七号签字`
