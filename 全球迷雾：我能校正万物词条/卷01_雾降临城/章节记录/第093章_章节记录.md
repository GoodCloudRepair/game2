# 第093章：后校先看后身影

## 1. 本章定位

- 所属卷：卷01 雾降临城
- 所属阶段：阶段04 / 门域核心前段
- 章节功能：把`后校本人`正式落成“先认后身影”的核心流程
- POV人物：沈辞
- POV类型：近景限知

- 本章结束时间：D2 05:23

## 2. 本章目标

- 本章要推进的主线层级：保护知微 / 主角身份线 / 门域核心线
- 本章核心冲突：沈辞必须接受核心后校第一步根本不看正脸和名字，而是先看物留给他的后身影
- 这章结束后局面具体变化：沈辞被后校白道确认“正面不作本人凭”，并在白道尽头迎来`后身格不认单人影`
- 这章会让读者对谁产生新的认知：对核心“认人方式”的冷硬逻辑、对沈辞一路痕迹为何重要
- 本章必须交代的信息：核心后校先看后身影
- 本章新增物品/能力/规则/伏笔：后校白道、`后校先看后身影`、`正面不作本人凭`、`手常停处 / 久留处 / 被记处`
- 本章结尾钩子：后身格明确写出`不认单人影`
- 本章风格目标：写“被识别”的压迫，不写“被追逐”的刺激
- 本章对应卷目标：把沈辞从“活项前移”推进到真正开始被核心校人

## 3. 本章剧情梳理

- 开场状态：沈辞已经通过白托盘，被允许`活项前移 / 后校本人`
- 事件一：沈辞进入后校白道，看见`后校先看后身影 / 正面不作本人凭`
- 事件二：白道从鞋边落灰、旧胶边、修复台木沿等物痕中抽出后身影，证明核心先认物留下的人
- 事件三：沈辞意识到后身影里喉口那段“没说满”的空白也被一起记账
- 结尾结果：白道尽头翻出`后身格不认单人影`
