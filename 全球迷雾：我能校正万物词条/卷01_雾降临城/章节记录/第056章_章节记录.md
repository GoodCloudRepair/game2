# 第056章：回头的人算退件

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：正式让东回廊从“被点亮的新路径”变成“沈辞必须亲自走进去的单向线”
- POV人物：沈辞
- POV类型：主POV
- 时间：D2

- 本章结束时间：D2 01:24

## 2. 本章目标

- 本章要推进的主线层级：从“东回廊已经挂出沈辞的空夹”推进到“沈辞正式顺着东回廊跟进去”
- 本章核心冲突：东回廊要求沈辞一旦跟入就不能再按来路退回，而知微又还被留在旧名室里
- 本章必须交代的信息：`外应先走 / 陪行不过内线 / 回头的人算退件`；林照走外窗维修道平行跟线；`辞`字空黑夹先自行启动
- 本章结尾钩子：第一拐角翻出新的动态规则`空夹先过 / 人后贴线 / 不得并行`
- 本章风格目标：把“是否进去”写成一条已经开始自己运行的路，而不是普通的行动选择
- 本章对应卷目标：让学校线从旧名室静态拆人继续推进成真实移动流程

## 3. 本章剧情梳理

- 开场状态：`辞`字空黑夹已经挂到旧名室侧窗外沿，东回廊喇叭刚刚报出`沈外应，候并`
- 事件一：空黑夹自行沿轨滑动，入口翻出`外应先走 / 陪行不过内线 / 回头的人算退件`
- 事件二：林照判断自己只能走外窗维修道平行跟线，沈辞则必须独自走东回廊内线
- 事件三：旧名室里传出极像知微挪动的轻响，沈辞几乎本能回头，却从`退件`灰纸与踩烂鞋尖看出回头后果，强压住动作
- 结尾结果：沈辞正式踏入东回廊，第一拐角亮出`空夹先过 / 人后贴线 / 不得并行`
