# 第060章：并门前先交来处

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：把东回廊这一小段推进到`并门前`，并正式打开更深的`来源柜 / 文修线`
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 01:42
- 本章结束时间：D2 01:49

- 目标字数：2000-2100字

## 2. 本章目标

- 本章要推进的主线层级：从“文修前声候核”推进到“更深的来源柜正式开门”
- 本章核心冲突：沈辞不能把自己或完整现名送进并门，只能先把`来处`递进去，让学校自己把更深来源线翻出来
- 本章必须交代的信息：`并门前先交来处 / 样声不过门 / 无源称退挂`；`文修 / 家录`等来源柜分类；文修格式窄盒与档案封套
- 本章结尾钩子：右侧小门翻出`文修前声，请入来源柜`
- 本章风格目标：让这一章像摸到学校后台材料库的门缝，而不是简单过一个新关卡
- 本章对应卷目标：为下一段“来源柜 / 文修线”小层级打开入口，并把修复中心前史继续往主线硬接

## 3. 本章剧情梳理

- 开场状态：沈辞被`辞`字黑夹推进到并门前，喇叭报出`文修前声，候核`
- 事件一：并门前柜墙亮出`先交来处 / 样声不过门 / 无源称退挂`三条规则，沈辞看见`东口 / 右窗 / 文修 / 家录`等窄盒分类
- 事件二：他从柜墙与封套格式中进一步确认，这里使用的正是和文修中心异常转录档近乎同源的材料体系
- 事件三：沈辞抓住`先交来处，不交现人`这一句，只把`文修前声`窄条递入左格，而不把自己送进并门
- 结尾结果：最右侧小门因此开缝，并翻出`文修前声，请入来源柜`
