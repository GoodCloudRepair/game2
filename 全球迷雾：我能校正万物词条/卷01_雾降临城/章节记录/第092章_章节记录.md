# 第092章：先校物后校人

## 1. 本章定位

- 所属卷：卷01 雾降临城
- 所属阶段：阶段04 / 门域核心起步
- 章节功能：让沈辞第一次以“活项”身份被核心过渡口主动接半步
- POV人物：沈辞
- POV类型：近景限知
- 时间：D2 清晨前
- 本章开始时间：D2 05:09
- 本章结束时间：D2 05:16
- 时间跨度：约7分钟
- 与上一章间隔：紧接第091章
- 地点：入核过渡口
- 目标字数：2000-2100字
- 实际字数：2023

## 2. 本章目标

- 本章要推进的主线层级：保护知微 / 主角身份线 / 门域核心线
- 本章核心冲突：核心先认物后认人，沈辞必须先接受自己一路带来的门外物痕与残件被分层处理
- 这章结束后局面具体变化：托盘把`临时协查`、外写、样残边等逐项归类，最终允许`活项前移 / 后校本人`
- 这章会让读者对谁产生新的认知：对核心“物先人后”的秩序、对沈辞一路所携带痕迹的意义、对顾焰不再代答的克制
- 本章必须交代的信息：核心先校物后校人
- 本章新增物品/能力/规则/伏笔：白托盘、`挂识 / 外写 / 残物 / 后身`四格、`样残边 / 可续链`、`活项前移 / 后校本人`
- 本章结尾钩子：沈辞第一次被核心主动接入半步
- 本章风格目标：把“入核第一步”写成冷细节处理，而不是大动作穿门
- 本章对应卷目标：让沈辞真正从窗外站到核心过渡口内部

## 3. 本章剧情梳理

- 开场状态：入核口已经接纳沈辞的`活 / 在`状态
- 事件一：侧缝推出白托盘，先分`挂识 / 外写 / 残物 / 后身`
- 事件二：`临时协查`、旧权限压痕、前置口冷白、样残边等被逐项归类
- 事件三：`样残边`被判`可续链`，最终托盘给出`物先净 / 人可后校`
- 结尾结果：侧缝进一步开启，核心允许`活项前移 / 后校本人`
