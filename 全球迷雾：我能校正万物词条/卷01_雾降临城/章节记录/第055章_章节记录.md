# 第055章：外应先走东回廊

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：正式亮出东回廊这条外应转送线，并把沈辞从“挂在门外”推进到“已有一只空夹在等他”
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 01:12
- 本章结束时间：D2 01:18
- 时间跨度：6分钟
- 与上一章间隔：0分钟
- 地点：旧名室门口、侧窗外沿、东回廊播报线前
- 目标字数：2000-2100字
- 实际字数：2000字

## 2. 本章目标

- 本章要推进的主线层级：从“未并走东回廊”推进到“东回廊已经替沈辞把外应夹和路条准备好了”
- 本章核心冲突：沈辞必须面对一个比之前都更明确的现实，即学校已经不满足于在纸面挂他，而是在物理路径上给他备好了下一格位置
- 本章必须交代的信息：`东回廊 / 外应先走 / 内格暂存 / 并称不过门`；空黑夹先压`辞`字；`来处不在门内者，走东`；播报`沈外应，候并`；`过东廊者，不回旧名室门前`
- 本章结尾钩子：门里知微还在，门外属于沈辞的东回廊空夹已经挂出来了
- 本章风格目标：把这一章写成“路已经被你预备好”的压迫感，而不是单纯听见一个新地名
- 本章对应卷目标：让学校线从房间内的静态处理，推进到下一段动态追线

## 3. 本章剧情梳理

- 开场状态：旧名室已经决定`旧生未定，外应先送`
- 事件一：瘦男人翻出东回廊白单和一只更窄的空黑夹，夹顶先压上`辞`字
- 事件二：沈辞与林照看见`来处不在门内者，走东`，确认这条路专门接“门外已挂格”的外应对象
- 事件三：更深处喇叭播报`沈外应，候并`，并露出`过东廊者，不回旧名室门前`
- 结尾结果：知微还留在旧名室里，但沈辞的下一只流程夹已经挂到了窗外
