# 第100章：认回的人不从正环出

- 所属卷：卷01 雾降临城
- 所属阶段：阶段04 / 门域核心前段
- 章节功能：承接静名槽半称回认，写清“认回不等于带离”，并打开偏环路线与外层封门同步

## 2. 本章目标

- 本章要推进的主线层级：保护知微 / 门域核心线 / 封门线
- 本章核心冲突：沈辞刚让知微完成受限回认，却不能顺着最想打开的正环把她带离，否则会牵动定层二环
- 这章结束后局面具体变化：沈辞确认正环属于定层结构，回认只能换取合法靠近和偏环入口，外层封门装置第一次与核心内环同步
- 这章会让读者对谁产生新的认知：对知微的锚点作用、对沈辞必须从“救走她”转向“先稳住能救她的路”
- 本章必须交代的信息：`认回的人不从正环出`不是单纯阻挠，而是因为正环仍在承担定层稳定
- 本章新增物品/能力/规则/伏笔：`正环不出`、`回认非带离`、`源位不脱层`、`偏环待接`、`外锁接入 / 待内名`
- 本章结尾钩子：外层封门装置同步到内环，只能锁外名，等待后续内名读口
- 本章风格目标：用结构失稳、动作克制和冷信息落地推进，不放完整煽情重逢
- 本章对应卷目标：完成兄妹线受限回认的阶段性落点，并把门域核心线自然切向封门之夜

## 3. 本章剧情梳理

- 开场状态：静名槽落下偏缝，薄牌提示`认回的人不从正环出`
- 事件一：沈辞本能想从17格正前方牵回知微，正环与二环立刻失稳，提示正环不能出
- 事件二：知微主动把回应压向偏缝，不往外伸手，表示她也知道不能走正环
- 事件三：沈辞用月票夹常触边压住旧触白桥，不再强拉，只让白桥顺着偏缝贴稳
- 结尾结果：偏环入口被内层承认，外层封门装置同步接入，内环翻出`外锁已接 / 待内名`
