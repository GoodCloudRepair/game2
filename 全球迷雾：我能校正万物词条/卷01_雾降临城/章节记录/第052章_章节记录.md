# 第052章：门外称先挂外格

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：坐实旧名室的内外两层挂格结构，并把`沈外格`正式推进到靠窗核来处这一步
- POV人物：沈辞

- 本章结束时间：D2 01:00

## 2. 本章目标

- 本章要推进的主线层级：从“先挂旧称”推进到“门外称只能先挂外格，下一步还要靠窗核来处”
- 本章核心冲突：沈辞必须尽快弄清`小辞`这一格现在到底危险到什么程度，是否已经等同于自己被旧名室认定
- 本章必须交代的信息：`门外称先挂外格 / 外格只存，不替内认 / 外格靠窗，不应第二遍`；外格分主称层与旁格层；知微因`旧位未清`导致左边内格仍暂挂不上；`沈外格，靠窗核来处`
- 本章结尾钩子：旧名室的下一步，已经不是逼知微，而是隔着侧窗继续往门外摸沈辞
- 本章风格目标：把威胁从“白签挂出来”推进到“流程已经明确递向门外人”
- 本章对应卷目标：继续抬高学校线对沈辞本人的收编感

## 3. 本章剧情梳理

- 开场状态：`小辞`已经挂进`沈`的外格
- 事件一：顾姓女生因门外一声`丫丫`被挂外格，现场坐实“外格只存，不替内认”
- 事件二：沈辞看清右槽分前后两层，外格只是先留门外称，不等于已经认成门里对象
- 事件三：旧名室因知微`旧位未清`暂时挂不上内格，于是直接把`沈外格`推进到侧窗核来处
- 结尾结果：`沈外格，靠窗核来处`被正式记入流程
