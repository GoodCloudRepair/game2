# 第085章：旧借图不留回程口

## 1. 本章定位

- 所属卷：卷01 雾降临城
- 所属阶段：阶段03 / 财团与旧档起步（进入三域前沿）
- 章节功能：让旧借附页展开成“前窗视线图”，并把父亲线从“留痕”推进到“他想让沈辞看见什么”
- POV人物：沈辞
- POV类型：近景限知
- 时间：D2 凌晨
- 本章开始时间：D2 04:31
- 本章结束时间：D2 04:35
- 时间跨度：约4分钟
- 与上一章间隔：紧接第084章
- 地点：三域前置口白桥深段
- 目标字数：2000-2100字

## 2. 本章目标

- 本章要推进的主线层级：保护知微 / 父亲线 / 主角身份线
- 本章核心冲突：旧借图不像逃生图，而像一张逼中位活线继续往前看的视线图
- 这章结束后局面具体变化：`到前窗 / 不回程 / 中位留活`被同时坐实，父亲留线的意图从“让他避开”改成“让他看见”
- 这章会让读者对谁产生新的认知：对沈父、对顾焰、对沈辞此刻所在中位的历史预留性
- 本章必须交代的信息：旧借图不留回程口，它把前进目标钉成`前窗`
- 本章新增物品/能力/规则/伏笔：旧借图、`到前窗`、`不回程`、`中位留活`
- 本章结尾钩子：桥尾深门翻出`连带项校位 / 请清单独项`
- 本章风格目标：让父亲线通过旧纸、旧记法和空间响应自己浮出来，不靠人物自述讲透
- 本章对应卷目标：把“去核心救人”推进成“去核心前先看见该看的东西”

## 3. 本章剧情梳理

- 开场状态：右轨旧借附页已经弹出，可供中位活线阅读
- 事件一：沈辞读出`看左，不停 / 过白桥 / 到前窗 / 不回程`
- 事件二：他想起父亲留下旧记号的方式，意识到这不是逃生图而是视线图
- 事件三：压在折痕下的`中位留活`露出，图、柜、样带、中轨同时被钉成同一方向
- 结尾结果：前置口从“看图”切到“连带项校位”
