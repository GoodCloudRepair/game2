# 第038章：旧播控不放全句

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：旧城追索
- 章节功能：揭开旧播控右窗只补“后半声”的机制，并把修复中心残带正式接进学校广播楼后台
- POV人物：沈辞
- POV类型：主POV
- 时间：D1
- 本章开始时间：D1 23:08
- 本章结束时间：D1 23:19
- 时间跨度：11分钟
- 与上一章间隔：0分钟

- 目标字数：2000-2100字
- 实际字数：2040字

## 2. 本章目标

- 本章要推进的主线层级：从“旧播控右窗是什么”推进到“它专门用残带补后半句，并且样带来自修复中心”
- 本章核心冲突：沈辞必须在不暴露自己的前提下，弄清父亲似的声音为什么到了右窗只剩“后半句”
- 本章必须交代的信息：`旧播控不放全句 / 补听对象只认后半声 / 右窗不收自己认来的亲属`；门前第一笔会被一起送到右窗；样带分成`学号前声 / 旧称壳 / 家属后半 / 补名样带`；父亲似的声音在右窗里缺少同一口气；修复中心残带被调进来并写着`只取前半，不放全句`
- 本章结尾钩子：沈辞看见纸套上的编号格式与自己上午在修复室里使用的完全同源
- 本章风格目标：把“恐怖感”从直接诱导转成更冷、更后台的拆件感
- 本章对应卷目标：让父亲线和主角职业线真正从气氛相似，推进到材料同源

## 3. 本章剧情梳理

- 开场状态：沈辞与林照顺着后侧楼梯追到旧播控右窗
- 事件一：两人看见右窗白牌和`前声 / 后声 / 续认`三槽，确认门前第一笔也被送到了这里
- 事件二：沈辞听出右窗送出的父亲似声音只有“壳”，没有同一句里连续的气口，证明这不是完整原声
- 事件三：两人在左窗机房纸箱和调拨单上看见`海陵地方文书修复中心`、`只取前半，不放全句`等信息，确认样带来源异常
- 结尾结果：沈辞认出纸套右下角红码格式与自己今天上午在修复室里接触过的编号一致
