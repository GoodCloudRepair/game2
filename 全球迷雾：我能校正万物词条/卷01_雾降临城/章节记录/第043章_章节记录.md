# 第043章：先写姓不补后字

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：让17号正式把`沈`姓写进一号补名室流程，并把补名室第一铃阶段的规则从“照签抄”推进到“先收姓、后点尾”
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 00:06
- 本章结束时间：D2 00:11
- 时间跨度：5分钟
- 与上一章间隔：0分钟
- 地点：西教学楼一号补名室中右列课桌区、补名桌讲台前
- 目标字数：2000-2100字
- 实际字数：2000字

## 2. 本章目标

- 本章要推进的主线层级：从“17号桌角露出`沈`姓”推进到“知微必须先把`沈`写进流程，后字则被留到第二铃后再点”
- 本章核心冲突：沈辞必须在不能出手、不能喊停的前提下，看着知微把`沈`姓先写进补名室，同时快速拆出这一层对前字、后字和回描的限制
- 本章必须交代的信息：`先写姓，不补后字`；17号白夹变为`待尾`；`后字抢落`会记`迟到补列`；`前字入列，不得回描`；`铃前不护后字`；补名桌存在`前字簿`与后字灰纸套；黑板写`前字入列 / 后字待点 / 二铃后不改姓`；活动牌翻出`二铃后，待尾点字`
- 本章结尾钩子：第二铃后才会真正点尾，而17号已经被先压进`沈`姓之下
- 本章风格目标：把教学楼的恐怖从“旧座签认名”进一步压缩到“名字被拆开登记”的冷流程感
- 本章对应卷目标：继续把回澜中学写成比广播楼更安静、更规整、也更贴近兄妹核心的名字陷阱

## 3. 本章剧情梳理

- 开场状态：第042章章末，17号桌角露出`沈`姓，知微还没真正落笔
- 事件一：女老师命17号`先写姓，不补后字`，知微只写下`沈`，白夹翻到`待尾`
- 事件二：右二列九号抢落后字被记`迟到补列`，另一名学生试图回描前字、低马尾女生试图护住第二格，都被当场压回规则里
- 事件三：沈辞从`前字簿`、后字灰纸套和黑板细字里读出一号补名室的真实顺序是“先入姓、后点尾、二铃后不改姓”
- 结尾结果：活动牌翻出`二铃后，待尾点字`，说明17号真正更危险的还在第二铃后
