# 第063章：校兄称不收门外自认

- 所属卷：第一卷
- 所属阶段：回澜中学收束
- 章节功能：把`校兄称`这一层正式立住，并确认学校只认知微在校内留下过的兄称笔迹，不认沈辞门外自报

## 2. 本章目标

- 本章要推进的主线层级：从`平辈旧称 / 小辞 / 先核兄称`推进到“校兄称只认门内旧生在学校里留下过的兄称笔迹”
- 本章核心冲突：沈辞不能在柜前承认自己是“哥”，只能从三张被裁过的学校联络纸里找出真正属于知微旧日留笔的那一张
- 这章结束后局面具体变化：学校确认`哥`这层称呼不是门外现认，而是知微在校内流程里确实写过、留过、被存过的一层旧称
- 这章会让读者对谁产生新的认知：对知微在学校里的日常留痕、对学校收`兄称`的方式、对沈辞识别“家庭痕迹之外的校内笔迹”的能力
- 本章必须交代的信息：`校兄称不收门外自认 / 旧生留笔先交 / 无笔作旁称`；学校会把兄称先核成校内留笔，而不是现人口头关系
- 本章新增物品/能力/规则/伏笔：校兄称联络纸、`旧生留笔`核验规则、`转联络单`下层流程
- 本章结尾钩子：蓝皮`联络单`被推出，下一步开始核`来接人`
- 本章风格目标：让“哥”这一层先落在知微自己曾经留出的校内纸痕上，而不是直接走情绪爆发
- 本章对应卷目标：继续收束学校线，把兄妹关系从家庭称呼层推进到校内可存证的真实留痕层

## 3. 本章剧情梳理

- 开场状态：第062章尾牌只露出一个`哥`字，家录柜已要求`先核兄称`
- 事件一：喇叭亮出`校兄称不收门外自认 / 旧生留笔先交 / 无笔作旁称`
- 事件二：细槽吐出三张被裁过的学校联络纸，沈辞从折痕、透明胶、纸屑和残留笔势中认出真正属于知微的那张
- 事件三：那张联络纸被柜内核过，翻出`旧生留笔 / 校兄称存 / 转联络单`
- 结尾结果：下层推出蓝皮`联络单`，流程从“她写过没有”继续推进到“最后是谁来接”
