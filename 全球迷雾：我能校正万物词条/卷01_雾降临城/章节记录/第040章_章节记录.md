# 第040章：样声里少了一口气

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：让沈辞用自己的修复经验精准拆穿样带接缝，把17号从右窗续认硬转去`缺考补名`
- POV人物：沈辞
- POV类型：主POV
- 时间：D1
- 本章开始时间：D1 23:31
- 本章结束时间：D1 23:42

## 2. 本章目标

- 本章要推进的主线层级：从“发现样带是拼的”推进到“用接缝外露让17号右窗续认中止”
- 本章核心冲突：沈辞必须在不触发正面对抗的前提下，把右窗这次补后声做成一场流程内部认定的失败
- 本章必须交代的信息：父亲真声和样带之间差的不是字，而是同一口气；林照为沈辞掩掉探手动静；`旁听过门`小条可当临时拨片；右窗一旦判`样声不全`，会同步写下`关系样退库 / 本人续认中止 / 前笔封存 / 转补名`；左窗人明确提到`文修那边送来的东西`
- 本章结尾钩子：17号被旧教工雨衣接走，续送单去向改为`西教学楼一层 / 缺考补名处`
- 本章风格目标：把这一章的爽点写成“极专业、极克制的一下子”，而不是爆发式对抗
- 本章对应卷目标：让沈辞第一次靠职业能力直接撬动学校核心流程，而不只是旁观理解

## 3. 本章剧情梳理

- 开场状态：右窗即将用新拼好的样带逼知微补第二笔，沈辞已经听出问题在“气口”
- 事件一：沈辞用父亲真实说话习惯和旧带逻辑，确认右窗这盘带是“旧称壳 + 命令句”拼接
- 事件二：林照用旧铁桶细响掩护，沈辞用折硬的`旁听过门`条轻擦导带，让接缝在过轮时抖出破绽
- 事件三：右窗老头判定`样声不全`，撤掉第二笔续认白板，并把17改挂`前笔封存 / 样声退回 / 转补名`
- 结尾结果：左窗人低骂`文修那边送来的东西`，17号被正式送往西教学楼补名口
