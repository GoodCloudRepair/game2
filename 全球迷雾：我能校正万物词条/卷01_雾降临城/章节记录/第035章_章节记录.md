# 第035章：不要替十七号签字

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：旧城追索
- 章节功能：揭开17号签字口的真正逻辑，并把第二层男声与父亲旧录音声线硬接起来
- POV人物：沈辞

- 本章结束时间：D1 22:45

## 2. 本章目标

- 本章要推进的主线层级：从“门前再签”推进到“签字口其实在逼17自己承认第二层复听成立”
- 本章核心冲突：沈辞必须理解为什么17这一签绝不能被别人代落，同时还要忍住不在第二层男声响起前抢先乱动
- 本章必须交代的信息：`不要替十七号签字 / 复听对象只认本人落笔 / 旁听人不留原名`；灰皮册按`学号 / 笔认 / 签入人 / 层`分栏；17号行写着`本人笔认后进二听 / 代签作代认`；知微留下`别听男声第二层`
- 本章结尾钩子：第二层男声以与父亲旧录音极像的起声方式出现
- 本章风格目标：把签字写成比听筛更冷的“自我承认落笔”
- 本章对应卷目标：让兄妹线和父亲线在学校外沿第一次真正硬碰

## 3. 本章剧情梳理

- 开场状态：沈辞与林照经器材侧门贴近签字口，观察17号门前待签流程
- 事件一：沈辞看懂17号签字口规则，确认这里要的不是姓名，而是对象本人落下那一笔
- 事件二：他在灰皮册里看到17号备注`本人笔认后进二听 / 代签作代认`，彻底明白代签会把17直接写进第二层成立
- 事件三：知微在签字口再次留痕，提醒`别听男声第二层`；同时`旁听过门`小条暴露出他们可能借的边线壳
- 结尾结果：喇叭中传来与父亲旧录音极像的男声起势，第二层正式落下
