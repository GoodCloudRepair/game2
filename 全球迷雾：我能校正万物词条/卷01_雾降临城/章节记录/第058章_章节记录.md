# 第058章：候并不站本位

## 1. 本章定位

- 所属卷：第一卷
- 所属阶段：回澜中学
- 章节功能：正式揭开`候并格`的结构，并明确“候并”不是站进自己的位置，而是站在旁边等被扣进去
- POV人物：沈辞
- POV类型：主POV
- 时间：D2
- 本章开始时间：D2 01:30
- 本章结束时间：D2 01:36
- 时间跨度：6分钟
- 与上一章间隔：0分钟
- 地点：候并格第三排、外窗平行观察位
- 目标字数：2000-2100字
- 实际字数：2008字

## 2. 本章目标

- 本章要推进的主线层级：从“抵达候并格”推进到“看懂候并格如何让空夹占本位、让外应站代线”
- 本章核心冲突：候并格试探沈辞会不会自己站进`本位`，而那一步一旦踩错就可能直接省掉缓冲层
- 本章必须交代的信息：`候并不站本位 / 外应先立代线 / 并称不过门`；`本位 / 代线 / 退挂`三类站位；`借声挂存者，先核来源`
- 本章结尾钩子：抽屉弹出`来处单`，并写明`现名不过柜`
- 本章风格目标：把这章写成“你连站在哪里都在被流程定义”的静压式场景
- 本章对应卷目标：继续精密拆学校的名字处理链，让东回廊和旧名室形成完整衔接

## 3. 本章剧情梳理

- 开场状态：沈辞跟着`辞`字空黑夹到达候并格门前
- 事件一：他看见`本位 / 代线 / 退挂`三类站位，确认候并格是专门挂外应与空夹的中层承接区
- 事件二：深处竹杆试探性点向`本位`，沈辞依据`候并不站本位`稳住脚步，踩进`沈外应代线`
- 事件三：头顶空夹因此暂时稳定，来处单完整露出，并写明`借声挂存者，先核来源`
- 结尾结果：候并格下方抽屉弹出，要求`先交来处`，并明确`现名不过柜`
