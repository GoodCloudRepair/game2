# 第088章：外窗只收第一眼

## 1. 本章定位

- 所属卷：卷01 雾降临城
- 所属阶段：阶段03 / 三域前沿收束
- 章节功能：让`核心外窗`第一次真正给出“核心在做什么”的视觉答案
- POV人物：沈辞
- POV类型：近景限知
- 时间：D2 清晨前
- 本章开始时间：D2 04:45
- 本章结束时间：D2 04:50
- 时间跨度：约5分钟
- 与上一章间隔：紧接第087章
- 地点：前沿廊终点 / 核心外窗前
- 目标字数：2000-2100字
- 实际字数：2000

## 2. 本章目标

- 本章要推进的主线层级：保护知微 / 父亲线 / 世界观格局线
- 本章核心冲突：外窗只给第一眼，沈辞必须在极短视窗里看清核心本质
- 这章结束后局面具体变化：沈辞确认核心不是怪物巢，而是一台把人和物重新编排的整理场
- 这章会让读者对谁产生新的认知：对沈父留下旧借图的意图、对核心本体、对沈辞的判断力
- 本章必须交代的信息：核心在“整理 / 重排”，不是单纯“吞噬 / 发疯”
- 本章新增物品/能力/规则/伏笔：`外窗只收第一眼 / 重看不作真`、`外窗挂识 / 保持前视`、`门外观察号，仅作外窗挂识`
- 本章结尾钩子：黑窗底部明示门外观察号只够挂到窗前
- 本章风格目标：把核心第一次亮相写成冷秩序感，不写成普通邪物巢穴
- 本章对应卷目标：把“知微在核心边缘”推进成“核心是什么样子”

## 3. 本章剧情梳理

- 开场状态：17柜、样带系与沈辞已经抵达`核心外窗`
- 事件一：外窗亮出`外窗只收第一眼 / 重看不作真`
- 事件二：沈辞在第一眼里看见核心内部是一座重排人、物、白牌和旧场景的巨大整理场
- 事件三：外窗底部补出`门外观察号，仅作外窗挂识`
- 结尾结果：读者和主角都知道外层号码只能止于窗前，下一步要改写
