# 第062章：家录不收现在叫法

- 所属卷：第一卷
- 所属阶段：回澜中学收束
- 章节功能：把`家录`这一层正式立住，并确认`小辞`被归入`平辈旧称`而非父称线
- POV人物：沈辞

## 2. 本章目标

- 本章要推进的主线层级：从“文修来源件被核出”推进到“家录柜把`小辞`归入平辈旧称，并开始追门内的兄称”
- 本章核心冲突：沈辞不能在家录柜前自报“我是她哥”，只能靠旧袖套、旧痕迹和家庭物件习惯判断哪一件才是真正对应这条线的`家录`
- 这章结束后局面具体变化：学校后台把`小辞`从父亲/文修材料线分离出来，正式压进兄妹平辈旧称链，下一步开始追门内旧生的`兄称`
- 这章会让读者对谁产生新的认知：对沈辞与知微家庭日常的旧痕、对学校后台处理家庭称呼的方式、对父亲线和兄妹线当前优先级的区别
- 本章必须交代的信息：`家录不收现在叫法 / 先校旧喊顺位 / 父称后压，平辈先核`；`现称作旁代`；`小辞`被柜子归为`平辈旧称`
- 本章新增物品/能力/规则/伏笔：家录袖套、`父称 / 平辈 / 旁代`三分签、`门内旧生，校兄称`窄校签
- 本章结尾钩子：尾牌露出一个`哥`
- 本章风格目标：冷、近、细，让家庭称呼像被学校拆成一条更私密也更危险的流程
- 本章对应卷目标：继续收束学校线，把兄妹线从名字/来源层推进到家庭称呼层，同时暂时把父称线后压

## 3. 本章剧情梳理

- 开场状态：来源柜刚翻出`沈姓家录，另柜待校`，下柜露出一截旧家录纸套边
- 事件一：家录柜亮出`不收现在叫法 / 先校旧喊顺位 / 父称后压，平辈先核`，并弹出`父称 / 平辈 / 旁代`三只家录袖套
- 事件二：沈辞通过纸套旧痕、透明胶、黑发和知微小时候留下的小白点，锁定真正属于兄妹日常那一只`平辈`袖套
- 事件三：家录柜核定`平辈旧称 / 小辞 / 先核兄称`
- 结尾结果：下层窄校签写明`门内旧生，校兄称`，尾牌露出一个`哥`
