# 物品与能力台账

这份文档用于控制“设定膨胀”。  
凡是会影响剧情判断、战斗结果、资源流向的东西，都应该记下来。

## 一、核心规则台账

| 名称 | 当前定义 | 已确定规则 | 不能轻易改动的点 |
| --- | --- | --- | --- |
| 灰雾 | 具备筛选与收容属性的规则污染 | 会侵蚀现实，会让区域脱离正常命名体系 | 不能写成普通毒雾或单纯魔气 |
| 门域 | 灰雾深层结构与现实连接后的裂口 | 不同门域拥有不同规则、怪物与历史残片 | 进入门域必须有风险和代价 |
| 失名 | 存在定义被抹去后的异化状态 | 人、物、区域都可能失名 | 失名必须和“定义变化”有关 |
| 编目庭 | 灾难背后的古老管理系统 | 以稳定高于一切为原则清理、编号、归档文明 | 不能提前写成简单邪恶组织 |
| 反向识别 | 高危目标对识名者的回望机制 | 某些半失名目标或门域残留会借识名接触反向锁定、认出、呼叫识名者 | 不能写成随时随地都会触发的万能追踪 |
| 医院腕带登记规则 | 儿童医院门域内的身份筛选流程 | 病区只认腕带、病历、编号，不认自然姓名与个人叙述 | 不能写成拿到任何腕带就能全域通行 |
| 静置区转入规则 | B1静置区的深层流程规则 | 只收正式转入对象，不收家属、陪护或自行步行进入者 | 必须跟床走，且不能回应广播认人 |
| 广播认人规则 | B1交接通道与交接电梯的深层识别流程 | 广播会主动诱导病人或陪送者说出真实身份，并以此重新定义交接对象 | 不能写成单纯幻听或普通催眠 |
| 静置巡视闭目规则 | B1静置观察区内部巡视规则 | 巡视期间病人必须闭眼、静置、不主动应答 | 醒着的人会被优先加固，不能写成可轻易规避的小规则 |
| 复核移送规则 | B1通往观察二室的深层转运流程 | 只有被标记为`加固待转`或`异常反应复核`的对象，左后侧隐门才会在提送时开启 | 不能写成常开通道，也不能让主角无代价逆向闯入 |
| 提送签收规则 | 对象进入`复核待提送`状态时 | 提送前不得自述、不得认亲、不得离床，违规者会被改入更严的静默处置 | 不能写成只针对病人单向生效；执行者同样必须按规则去人化 |
| 观察单代言规则 | 对象进入观察二室后 | 观察单会先于对象本人定义其主诉、姿态和行为意义 | 不能让对象轻易靠自述翻案；这里是“记录先于人” |
| 回声采样规则 | 二室广播启动时 | 系统用熟悉声音与旧源样本测试对象会对哪一种联系产生回应 | 第三次采样风险最高，不能写成普通情感试探 |
| 输液大厅挂接规则 | 对象进入输液大厅下层后 | 大厅不收探视，未挂接对象不得靠门，滴尽前不得呼叫接领人 | 不能写成普通留观输液；这里本质上是挂接和分流区 |
| 瓶面回声镜规则 | 对象直视并认领瓶中倒影时 | 液体会把对象认出的那张脸与对应回声接进液路 | 不能写成单纯幻觉；一旦认领成立，流程会把回应收走 |
| 门前观察架退挂规则 | 对象进入靠门弧位后 | 门前不挂空瓶，滴尽先退，回声未满不得接门 | 不能靠主动挣扎脱身；动作过大只会被判`液路受扰` |
| 后勤清转规则 | 对象被标记为`退样清转`后 | 离院的人不走正门，清转对象不得回头，出院单先于对象离院 | 不能写成真正出院安全线；这里只是医院深层的后车物流线 |
| 殡仪街总站候车规则 | 进入废弃公交总站候车区后 | 别上写着`回程`的车，候车线外的人不报站，末班车未到不得离队 | 不能写成普通鬼打墙；这里本质上是被候车习惯固定住的场所秩序 |
| 文化馆暂泊点名规则 | 进入社区文化馆正门雨棚与暂泊座位区后 | 暂泊人员先落座，点名只认本人应答，前排不留空椅 | 不能写成普通避难点登记；一旦落座就容易被写进点名秩序 |
| 北库候领规则 | 进入文化馆后台北库签转区后 | 候领对象不得自报，回执作本人用，未盖签章不得离库；补签回执须本人复领，回执不认带队，空位先核后退 | 不能写成普通仓库领物；这里是把人和物一并按回执外分的后台流程 |
| 西坡闸过闸规则 | 改签车辆抵达西坡闸坡口后 | 西坡闸不认步行人，过闸不报去处，空车先靠右复核 | 不能写成普通卡口；这里认的是车、签与是否空位，不认自然身份 |
| 临设交接口规则 | 车辆抵达澜中临设侧签口后 | 临设只认交接单，改签件不走正门，交接口不收自己下来的人 | 不能写成普通避难点门岗；这里认的是交接关系，不认临时自述和自行下车 |
| 北棚待核规则 | 对象被送入`操场北棚`后 | 北棚今晚不见家属，不收补认，待核对象只能留在棚内流程里 | 不能让家属靠喊人直接带走对象；这里要先完成棚内核对 |
| 北棚分色待分规则 | 对象在北棚尚未落班时 | 蓝/白/红/灰不同颜色对应不同去向，未落班者不得入教室 | 不能写成普通临时病棚；这里本质上是学校外沿的待分流层 |
| 班牌分列规则 | 对象经过旧旗杆分牌桌时 | 班牌不写本名，只写颜色、口、列、次；借牌不进东口 | 不能靠偷一块牌就混进去；真正生效的是卡槽和顺序写入 |
| 东口复听规则 | 对象进入广播楼东口待听区后 | 广播先叫学号，复听期间不纠正称呼，第二声不应旧名 | 不能写成普通点名；这里是在用学号壳钓旧名本能 |
| 签字口笔认规则 | 对象进入门前待签位时 | 复听对象只认本人落笔，代签作代认，旁听人不留原名 | 不能替17号好心签字；这里是把“本人承认”写成流程证据的口子 |
| 第二层旧称规则 | 对象被推进第二层家族/旧称诱导时 | 第二层只听旧称，别答喇叭里的小名，未签者坐门前 | 不能把第二层写成单纯复读；这里会直接调取更私密、更旧的记忆壳 |
| 第二层近场反应规则 | 门内对象进入第二层且门外存在真正相关者时 | 广播楼会通过近场反应记录“是谁先被旧称钓动” | 不能只防门内对象；贴得太近的家属或熟人也会被一起记进流程 |
| 二听受扰复排规则 | 第二层完整放送被机器受扰、回授或流程问题中断时 | 对象会被改挂`缓签复排`并送往旧播控重排 | 打断只能换口子，不能直接解除整个对象流程 |
| 旧播控后半声规则 | 对象被转入旧播控右窗补听后 | 旧播控不放全句，只放后半声，逼对象自己补全前半记忆 | 不能写成普通复播；这里本质上是在用残壳诱导对象自我补完 |
| 第二笔续认规则 | 对象在门前已留下第一笔后进入右窗 | 本人笔认只收第二笔，前笔不断，不得退号，补听成立再续全号 | 第一笔哪怕没写完整也不算白留；这里只差最后那一下就会把本人补死 |
| 教楼补名规则 | 对象以缺考、待补或样声退回状态进入西教学楼时 | 缺考的人先补名字，未落座前不问来处，借名者不得入列；进入补名室后只照座签抄，写满者先入空卷 | 不能靠认亲、自证或喊名字破局；教学楼先认位置和列次，再认残名缺口 |
| 铃声入室规则 | 第一铃响起后 | 缺考对象会被推进指定补名室或对应列次 | 不能把铃声写成气氛音效；这里是直接推动对象入室生效的口令 |
| 座签补名规则 | 对象在一号补名室被点到具体列位后 | 对象只能照桌角旧座签残名补字，不得直接写本名 | 这里不认自我记忆，只认座位给出的缺口 |
| 空卷收纳规则 | 对象抢写完整本名、越过座签顺序时 | 会被判`写满`并先送入空卷秩序 | 不能把写本名当成自救；写得越快，反而越先被拖深 |
| 前后字拆分规则 | 对象在第一铃后的一号补名室被正式点到时 | 先写姓或前字，后字需等第二铃后再点，且二铃后不改姓 | 抢落后字、提前护尾、试图回描前字都会被记入异常流程 | 第043章明确坐实`待尾`、`迟到补列`与`前字入列，不得回描` |
| 同姓未齐并尾规则 | 第二铃后同姓下仍有灰夹、待补、迟到或空位对象未齐时 | 本层暂不并后字，可改送更高层继续并尾 | 不能替别人代点后字，代点作代认；必须让流程真正“看见未齐” | 第045章已被沈辞反向利用，把17号拖成`上楼并尾` |
| 送层遮尾规则 | 对象从一号补名室被改挂`上楼并尾`并进入楼层交接时 | 楼上只认姓牌，后字、班项和原班级信息都要遮掉 | 不能露第二格，也不能把本层班项带上楼；送层只承认交接批次 | 第046章坐实`上楼只带姓牌`，说明教学楼会按层继续拆名 |
| 待尾区隔位规则 | 对象进入二层待尾区后 | 同姓需隔位、不并肩、不照面，未齐前不得过册 | 任何并肩、越线、照面都可能被算成提前并位 | 第047章明确二层是按姓等齐的候件层 |
| 旧位核对规则 | 同姓对象进入二层过册前 | 需先在旧位册、座次图和木牌中核定旧位，位不定不翻尾卡 | 轻碰木牌、沿今序站位、被人代扶都可能直接认位 | 第048章坐实`知`为旧生位、`辞`为外应位的分流依据 |
| 旧位代指规则 | 核位时旁人出声、代指、靠线或提示位置 | 会被视作替对象排旧位，甚至形成旁声作证 | 线外也不能帮；一句话、一步站位都可能变成证据 | 第049章顾姓女生脱口而出即被挂条 |
| 旧位不清转送规则 | 旧位信息彼此矛盾、执行层也无法合法代排时 | 对象暂不翻尾，改挂`旧位不清 / 位失`继续送往旧名室 | 只是继续送深，不是脱身；来源会被一并带入下一层 | 第049-050章17号被正式送进旧名室 |
| 旧名室半人规则 | 对象旧位不清、位失、来源未并时 | 旧名室先按“缺口 / 半个人”接收，再补更深层身份 | 未核清旧位、来源与旧称前，不翻全名 | 第050章第一次打开学校更深一层规则 |
| 旧称交付规则 | 尾未定、来源未并的对象进入旧名室时 | 先核乳名、小字、旧喊法和门外称，再决定是否并尾 | 错称、不交、或被旧称钩动都可能反被流程利用 | 第050章点出`沈姓两位，先交旧称` |
| 旧称自报规则 | 对象在旧名室主动纠正、解释或说出旧称时 | 会被记成`自报旧称`并压入后核流程 | 开口不算证明，只会把自己往更深一层压 | 第051章周姓男生现场坐实 |
| 门外称外格规则 | 旧称来自门外熟人、旧广播流程或门外关系线时 | 称呼只先挂到外格，不直接替内格认人 | 外格只是留存与待核，不代表已经并成门里对象 | 第052章`小辞`、`丫丫`两例坐实 |
| 主称借叫限制 | 门外人替对象再叫旧称、代认或借声时 | 称呼只能落在旁格，不作主称 | 越亲近越危险；借叫会把主称位继续空着 | 第053章顾姓流程完整坐实 |
| 借声挂存规则 | 旧称源于广播楼旧声、转录残响或流程带入时 | 旧称可挂外格，但只能记作挂存来源 | 不能直接收人，却会完整带进下一层并来源流程 | 第053章`小辞`被压成这一状态 |
| 来源并合规则 | 对象内外两格都被扣到同一姓下且可相互作证时 | 黑夹会翻起更深一层尾角，并立刻触发转送 | 一旦并成就不再只是挂着看，而是实质推进收名 | 第054章通过周姓对象完整展示 |
| 来源未并转送规则 | 对象内外两格仍有空缺、旧位卡住或主称未定时 | 不在旧名室继续耗，改送东回廊等下一层处理 | 只是换线，不是脱身；会优先推进更容易送的那一边 | 第054章正式落到`沈`这一姓上 |
| 东回廊外应规则 | 对象被记为门外外应、且旧名室决定先送时 | 外应会被独立挂夹、夹单，转入东回廊候并 | 一旦过东廊，就不再回旧名室门前 | 第055章正式打开下一段路径 |
| 东回廊贴线规则 | 外应正式跟入东回廊时 | 空夹先行，活人必须贴线落后半步 | 与夹走平会被算到位，甚至触发并件口 | 第056-057章完整坐实 |
| 候并格代线规则 | 外应到达候并格后 | 空夹占本位，活人只能站代线等候后续核验 | 踩错本位就可能自我交件，来处不清还会退挂为无源件 | 第058章正式落地 |
| 外应半姓核验规则 | 候并格开始核来处时 | 此层只认半个姓与来源，不认完整现名 | 全称作自认，会把借声残片直接往本人层推进 | 第059章正式亮相 |
| 并门前来处规则 | 外应被送到并门前、且涉及样声来源时 | 先交来处，不交现人；样声不过门 | 来处核不清则`无源称退挂`，核清后才会打开来源柜 | 第060章正式打开下一层 |
| 来源柜封套核验规则 | 对象被推进来源柜、且来处已被锁到文修/家录等旧材料类时 | 柜内只认实物封套、封口号与旧借调痕迹，不收活人口述来处 | 认错封套或先报人口，整件会被退挂，并可能继续牵出更深来源线 | 第061章正式执行 |
| 家录柜旧喊顺位规则 | 来源柜已把对象推进家录层，且涉及家庭旧称材料时 | 只认旧喊法和关系顺位，不认活人口头解释现在关系 | 自报现称会被记成旁代；父称、平辈、旁代一旦走错线就会把家庭关系拆成危险错件 | 第062章正式执行 |
| 校兄称留笔规则 | 家录层已核到`兄称`，且需要确认这一称是否在学校内部真实留存时 | 只认门内旧生写过、交过、被学校存过的留笔 | 门外自认会被压成旁称，不能直接进入后续联络核验 | 第063章正式执行 |
| 联络单来接规则 | 校兄称已被核成留笔，且学校继续确认现实外接位时 | 先认谁真实到校、谁真实来接，不先认完整家属谱 | 家属栏可能被后压；空来、未到或代填都不会被算作当前生效位 | 第064章正式执行 |
| 回听格规则 | 联络单已核到现实外接位，且学校需要确认门内对象对何种称呼有稳定回听时 | 只保留真实有应的称呼片段，不留完整句 | 外口误应会被记成错向；无应片段会被压成废标 | 第065章正式执行 |
| 观察册规则 | 学校已确认对象存在稳定异常回听、停步或牵引模式时 | 把对象从普通旧生/缺席处理里剥出，转入长期观察册 | 班栏、学号、缺席原因会被压成封皮；真实观察项只在另页流转 | 第066章正式执行 |
| 对象分类卡规则 | 观察册已确认对象需要深层分类时 | 用独立硬卡把对象真实类别、牵引项与暂留地单独挂出 | 普通旧生册只作外壳，深层去向不会在表层流程直接露满 | 第067章正式揭出 |

## 二、沈辞能力台账

| 能力名 | 阶段 | 效果 | 代价/限制 | 当前状态 | 预计解锁阶段 | 备注 |
| --- | --- | --- | --- | --- | --- | --- |
| 识名 | 初始能力 | 看见怪物、物品、区域的真实词条与隐藏定义 | 需要接触、观察或精神聚焦；信息过载可能造成头痛、耳鸣、视野发灰等负担 | 已确定 | 第一卷 | 第004章首次显现，第008章开始稳定作用于物品，第010章首次触发被高危目标反向锁定 |
| 改名 | 第一阶段进阶 | 短时间改写一个目标的局部规则 | 不能长期维持；改写越强，反噬越重 | 已确定 | 第二卷 | 第一卷只埋征兆 |
