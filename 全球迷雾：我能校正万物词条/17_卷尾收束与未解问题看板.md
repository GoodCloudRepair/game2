# 卷尾收束与未解问题看板

这份文档用于控制卷尾阶段“该收什么、该留什么、不能再开什么”。  
当前重点服务卷01后段，避免第101章以后继续横向扩展导致卷尾失焦。

## 使用规则

1. 每写完一章，检查本看板中是否有问题被回收、推进或延期。
2. 卷尾前20章原则上不新增大型支线，只新增完成卷尾必需的小规则。
3. 未解问题分三类：本卷必须回收、本卷部分回收、留到下卷。
4. 如果某章新增问题没有进入本表或伏笔台账，需要补记。

## 当前禁止新增

- 新大型势力。
- 新城市地图支线。
- 新完整能力阶位。
- 新亲缘角色。
- 与封门无关的新门域副本。
- 大段解释编目庭终极真相。

## 每章写后检查

- 本章是否推进了“静名槽 / 定层 / 封门 / 官方吸纳”之一？
- 本章是否新增了一个卷尾前不该新增的问题？
- 本章是否回收或推进了本表中的至少一项？
- 本章结尾是否把读者推向卷尾高潮，而不是推向新支线？
- 如果本章只是在解释规则，是否有可见代价或关系变化支撑？
