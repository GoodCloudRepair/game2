# 关系台账

关系不是静止设定，而是会随着章节推进不断变化。  
这份台账只记“已经发生变化的关系”和“未来准备推进的方向”。

| 关系双方 | 当前关系 | 核心纽带/矛盾 | 最近变化 | 后续方向 |
| --- | --- | --- | --- | --- |
| 沈辞 / 沈知微 | 兄妹，强保护关系，已绑定封门结构 | 沈辞想救她，知微却正在压住城市门缝 | 第108-112章知微继续稳住17格，帮沈辞压住外线回执和真门名倒写反噬；沈辞在撤城与替换方案前明确拒绝把她当作一次性锚点 | 第113章必须用规则证明`共鸣项不可替换`，不能只靠情绪反驳顾焰 |
| 沈辞 / 林照 | 高危搭档，内外双线互保 | 理性、克制、秩序感上的互相识别 | 第108-112章林照按沈辞读出的断点处理学校线、救援线和东门换线，不用姓名与撤离口令，关系从“她断外线、他读内名”推进到“她用现实救援托住他的读口” | 卷尾继续承担外圈救援和现实伤亡压力 |

| 沈辞 / 门域核心 | 已进入真门名倒写与封门夜战段 | 核心只认未写圆活项，并不断切断门外关系 | 第108-112章核心以外线回执稳内名，排除文昌回头线后开始倒写真门名，同时反读沈辞自身词条；第一栏失守后撤城和替换方案都暴露定层被接管风险 | 后续继续读倒写笔画，但必须防止活项被门反读和共鸣项被工具化 |

## 卷01收束关系补记

- 沈辞 / 沈知微：第113-120章从“不可替换”推进到“互相托命后幸存”。知微以半称前痕主动轻应，沈辞用她争来的三息读门；封门后她仍有源位灰痕，完整姓名与完整亲称暂不可碰，兄妹线完成阶段性回收但未完全恢复日常。
- 沈辞 / 林照：第116章林照用只报物不报人名的方式帮沈辞抵住门反读，第119章明确支持留缝保未归项，第120章继续替知微挡系统编号；关系从内外协作推进到现实层互相托底。
- 沈辞 / 杜衡：第118-120章杜衡接受沈辞最终旧边名并执行留缝封门，随后递出临时证；关系从封门互用转为官方吸纳/监管借用。
- 沈辞 / 顾焰：第113章替换方案被规则否决，第117章黑箱记录近似改名不可复现，第119-120章复核标记未撤；关系从工具化冲突转为财团持续观察与后续解释权争夺。
- 沈辞 / 城防局：卷尾正式从临时协查升级为`监管借用`，沈辞被纳入体系但不是自由入职，也不是完全信任。

## 卷02开篇关系补记

- 沈辞 / 城防局：第121章临时证正面入档，背面自补`不得离开监测半径`；关系从“监管借用”进一步制度化为“可调度但受监测半径限制”。
- 沈辞 / 城防局：第122章反读评估判出`调度污染：三格`与`不稳定变量`，关系从单纯监管推进为“可用但不能直连系统”的隔离使用状态。
- 沈辞 / 杜衡：杜衡明确不解释更深背注，只执行记录与评估传票；两人关系进入更冷的制度化互用。
- 沈辞 / 杜衡：第122章杜衡确认“初评通过不代表安全”，继续只给执行边界，不替沈辞争取信任背书。
- 沈辞 / 林照：林照继续替知微挡系统编号，并把`紧急联系人`栏压空；关系保持现实边线互保。
- 沈辞 / 沈知微：知微仍只对两下旧触产生轻应，没有恢复完整称名；兄妹线继续克制，不进入团圆状态。
- 沈辞 / 顾焰：顾焰的联席蓝点只能外缘可见，无法写入`编目师候选`来源；解释权争夺仍在，但权限边界再次显露。

## 使用规则

1. 关系变化必须写清楚“因为什么变”。
2. 如果关系变了，但没有事件支撑，那就是写飘了。
3. 感情线推进必须和共同经历绑定，不能凭空升温。
