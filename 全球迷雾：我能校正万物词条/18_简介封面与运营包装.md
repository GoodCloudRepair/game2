## 简介草案

### 短简介

全球灰雾降临，听见名字的人会被门里的东西拖走。  
旧档修复员沈辞发现，自己能看见万物背后被篡改的词条。  
当妹妹沈知微被编成`17号共鸣项`，他必须在不能直呼姓名、不能承认亲属关系的规则里，把她一点点认回来。

### 长简介

灰雾出现后，城市开始失去正确的名字。  
车站把活人播成乘客，医院把妹妹登记成`17号`，学校把旧称拆成无法回家的碎片，官方白栏只认观察号，不认人。

沈辞原本只是地方文书修复中心的旧档修复员。  
直到一卷没有编号的磁带，在十几年灰尘里喊出了他妹妹沈知微的名字。

他看见了悬在万物背后的词条。  
错误的名字会杀人，错误的档案会吞城。  
而他能做的，是在规则缝隙里读出真名，把被改坏的一切校正回来。

可越往深处走，他越发现：  
灰雾不是灾难的全部。  
有人早就进入过门里。  
他的父亲留下了残缺方法。  
而他自己，也许从一开始就不是普通的幸存者。

## 封面方向

| 项目 | 要求 |
| --- | --- |
| 必须出现 | 书名`全球迷雾：我能校正万物词条`；作者名`春秋烽火` |
| 主视觉 | 灰白雾海中的城市轮廓，前景有被纠错的半透明词条、档案页、红色校正线或旧磁带元素 |
| 气质 | 悬疑、冷感、规则感、城市级灾变；不要做成纯玄幻打斗或纯言情 |
| 人物 | 可有一个背影或剪影，不必画具体场景；重点是“人站在被错误命名的城市前” |
| 色彩 | 灰白、冷青、旧纸黄为主，少量红色校正线点睛 |
| 避免 | 过多具体章节场景、怪物堆脸、赛博霓虹过强、血腥恐怖封面、看不清书名 |

## 包装自检

- 简介前三行是否在300字内给出冲突？
- 简介是否同时说清“灾变规则、主角能力、妹妹目标”？
- 封面是否一眼能看清书名和作者名？
- 封面是否准确传达规则悬疑，而不是误导成纯言情或纯玄幻？
- 当前正文是否兑现了简介里承诺的能力和主线？
- 如果读者反馈“看不懂/太冷/太慢”，是简介误导，还是正文节奏问题？
