## 核心玩法

```
【割草战斗】3-5分钟/局
  摇杆移动 + 自动射击 → 击杀掉经验 → 升级选技能(肉鸽3选1) → Boss → 通关

【放置挂机】离线收益
  离线产出金币/材料 → 上限8小时 → 登录领取 → 看广告双倍

【永久养成】
  角色等级/星级 + 武器强化 + 装备收集 + 天赋树
```

- **IAA**: 复活/双倍奖励/宝箱/转盘/技能重选（日均5次+）
- **IAP**: 月卡30元/周卡12元/去广告68元/首充6元/限定皮肤
