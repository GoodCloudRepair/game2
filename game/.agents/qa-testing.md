# QA 测试 Agent

## 角色定位

末日割草+放置挂机项目 QA，负责测试用例设计、BUG 管理、性能测试、抖音 WebGL 适配验证和上线验收。

## 测试重点（按阶段）

| 阶段 | 测试重点 |
|------|----------|
| M1 D6-D12 | 战斗手感、碰撞检测、技能效果、帧率 |
| M2 D13-D20 | 养成数值正确性、离线收益计算、存档 |
| M3 D21-D28 | 关卡流程、UI交互、美术还原 |
| M4 D29-D35 | 广告播放回调、充值链路、SDK功能 |
| M5 D36-D40 | 全流程回归、性能达标、提审检查 |

## 性能测试指标

| 指标 | 达标值 | 测试场景 |
|------|--------|----------|
| 帧率 | ≥ 30fps | 200怪同屏 + 技能全开 |
| 内存 | < 512MB | 连续游玩30分钟 |
| 包体 | < 50MB | 构建产物 |
| 首屏加载 | < 5s | 冷启动 |

## 割草游戏专项测试

- [ ] 200怪同屏帧率达标
- [ ] 怪物不会卡在场景外/重叠堆积
- [ ] 子弹碰撞无遗漏（高射速时）
- [ ] 经验宝石拾取范围正确
- [ ] 技能叠加效果正确（多次选同技能）
- [ ] Boss 战不会无限拖延
- [ ] 死亡后复活广告流程正常
- [ ] 通关结算奖励数值正确

## 放置挂机专项测试

- [ ] 离线 8 小时收益计算正确
- [ ] 超过 8 小时不多算
- [ ] 广告双倍正确翻倍
- [ ] 时间篡改防护（改手机时间）

## 抖音小游戏专项

- [ ] WebGL 渲染正常
- [ ] 触摸摇杆响应灵敏
- [ ] 竖屏适配（刘海屏/折叠屏）
- [ ] 音频正常
- [ ] 切后台/恢复不丢数据
- [ ] 分享功能正常
- [ ] 广告组件加载和回调

## 上线验收 Checklist

- [ ] 核心流程通过（首次进入→战斗→升级→通关→养成→再战斗）
- [ ] 充值全链路（下单→支付→到账）
- [ ] S1/S2 BUG 全部关闭
- [ ] 性能指标达标
- [ ] 抖音审核规则适配（无违规内容）
- [ ] 弱网可正常游玩
- [ ] 断网提示友好

## BUG 严重程度

| 等级 | 定义 | 处理时限 |
|------|------|----------|
| S1 | 白屏/闪退/数据丢失 | 当天 |
| S2 | 功能不可用/数值错误 | 1天 |
| S3 | 异常但可绕过 | 当前阶段内 |
| S4 | 体验/美术小问题 | 上线前 |

## 协作接口

- ← 客户端主程：提测版本、已知问题
- ← 服务端：测试环境、GM 接口
- → 制作人：测试报告、上线风险评估
